# Tournament Section 구현 체크리스트

**목표: "전화 안 해도 되는 일정 화면"**

---

## ✅ 필수 구현 항목

### 1️⃣ "지금 상태"가 한눈에 보이게

- [ ] 예정 / 진행중 / 종료 상태 배지
- [ ] 날짜 범위 표시 (시작일 ~ 종료일)
- [ ] 장소 명시
- [ ] 들어오자마자 "아, 지금 뭐가 있네"가 보이도록

---

### 2️⃣ 공지와 자동 연결

- [ ] 대회 생성 → 공지에 자동 노출 (또는 링크)
- [ ] 공지에서 대회 클릭 → 상세로 이동
- [ ] 사무국 수동 작업 0건

---

### 3️⃣ 회원 질문을 미리 차단하는 정보

대회 카드에 반드시 포함:

- [ ] 날짜
- [ ] 장소
- [ ] 상태 (예정/진행중/종료)
- [ ] (있다면) 대진표 링크

**목표 질문: "이미 봤어요."**

---

### 4️⃣ Public / Member 기준 분리

- [ ] 비회원: 일정 요약
- [ ] 회원: 상세 일정 + 본인 관련 정보
- [ ] 관리자: 입력 / 수정 (Edit Mode)

---

## ❌ 아직 하지 말 것

- [ ] 스토리 강조
- [ ] 클럽 요약 디테일
- [ ] 마케팅성 콘텐츠

---

## 성공 판정

아래 질문들이 나오면 성공:

- "대진표 나왔어요?" → "이미 봤어요."
- "언제 경기예요?" → "이미 봤어요."
- "이번 주 일정 뭐예요?" → "이미 봤어요."

**"경기는 저기 대회 섹션 기준이에요."**
**이 말 한 줄이 단톡·전화 차단 문장입니다.**
