## 1️⃣ 플랫폼 전체 구조

### YAGO SPORTS = 4개 메인 도메인

```
YAGO SPORTS PLATFORM
│
├─ HOME (/home)
│   ├─ Quick Start (빠른 시작)
│   │   ├─ 거래 (Trading)
│   │   ├─ 경기 활동 (Sports Activity)
│   │   └─ 이벤트 (Events)
│   ├─ Activity Feed (활동 피드)
│   └─ Personal Stats (개인 통계)
│
├─ MARKETPLACE (/sports/:sport/market)
│   ├─ 상품 목록
│   ├─ 장비 거래
│   └─ 팀 용품
│
├─ SPORTS HUB (/sports) ⭐ 핵심
│   ├─ 내 팀 (Teams)
│   ├─ 경기 (Matches)
│   ├─ 팀 이벤트 (Team Events)
│   ├─ 선수 (Players)
│   ├─ 통계 (Statistics)
│   ├─ 대회 (Tournaments)
│   └─ 유소년 아카데미 (Academy)
│
├─ COMMUNITY
│   ├─ Activity Feed
│   ├─ Social Features
│   └─ Media Gallery
│
└─ PROFILE (/mypage)
    ├─ 내 정보
    ├─ 내 팀
    ├─ 내 경기
    └─ 설정
```

## 4️⃣ 화면 간 네비게이션

### 4-1. 주요 네비게이션 흐름

```
HOME (/home)
  ↓
Quick Start: "경기 활동" 클릭
  ↓
SPORTS HUB (/sports)
  ↓
모듈 선택 (내 팀/경기/이벤트/선수/통계/대회/아카데미)
  ↓
도메인 페이지
  ↓
상세 페이지
```

### 4-2. Teams 네비게이션

```
SPORTS HUB
  ↓
"내 팀" 클릭
  ↓
TEAMS ENTRY (/teams)
  ↓
팀 선택
  ↓
TEAM WORKSPACE (/sports/:type/team/*)
  ↓
탭 선택 (일정/멤버/기록/공지/이벤트)
```

### 4-3. Matches 네비게이션

```
SPORTS HUB
  ↓
"경기" 클릭
  ↓
MATCHES (/matches)
  ↓
경기 선택 또는 "경기 생성"
  ↓
MATCH DETAIL (/matches/:matchId)
```

### 4-4. Tournament 네비게이션

```
SPORTS HUB
  ↓
"대회" 클릭
  ↓
TOURNAMENTS (/tournaments)
  ↓
대회 선택
  ↓
TOURNAMENT DETAIL (/tournaments/:tournamentId)
  ↓
탭 선택 (개요/참가팀/경기일정/순위/통계)
```
