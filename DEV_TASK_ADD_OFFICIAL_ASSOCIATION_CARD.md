# [개발 지시서] 축구 허브 – 노원구 축구협회 공식 카드 추가

**설명용 ❌ / 실행용 ⭕ (그대로 복붙해서 사용 가능)**

---

## ✅ 현재 상태 요약 (팩트 체크)

### 📍 위치
- 축구 허브 (축구 카테고리 메인)
- 경로: `/sports/football` 또는 `/hub/soccer`

### 📦 현재 노출 카드
- 우리 팀 관리
- 팀 / FC 찾기
- 일정
- 근처 시설
- 거래 장터

### ❌ 노원구 축구협회 공식 카드
- 아직 코드에 추가되지 않음
- 그래서 UI에 "안 보이는 게 정상" 상태

**👉 지금은 "버그"가 아니라 "미구현" 상태다.**

---

## 🎯 왜 아직 노출이 안 되느냐 (아키텍처 기준)

현재 축구 허브는 구조적으로:

```
[카테고리: 축구]
 ├─ 개인/팀 기능 카드 (우리 팀 관리, 팀 찾기)
 ├─ 활동 카드 (일정, 근처 시설)
 └─ 상업/보조 카드 (거래 장터)
```

**❗ 공식 주체(협회)를 나타내는 카드가 IA 상 존재해야 하는데, 아직 JSX에 없음**

즉,
- 디자인 문제 ❌
- 권한 문제 ❌
- 데이터 문제 ❌
- **➡️ 카드 컴포넌트 자체가 아직 추가되지 않음**

---

## 🎯 목표

**축구 허브 화면 최상단에 공식 주체 카드: "노원구 축구협회"를 추가한다.**

이게 되면:
- IA 위반 상태 해소
- "공식 주체 → 팀 → 개인" 구조 완성
- 이후 Hero Section, Tournament 연동이 논리적으로 이어짐

## 2. 카드 UI 사양 (기존 카드와 동일한 컴포넌트 사용)

### 아이콘
- ⚽ 또는 🏛️ (협회/공식 느낌)

### 제목 (Title)
```
노원구 축구협회
```

### 설명 (Subtext, 작은 글씨)
```
노원구 공식 축구 행정 · 대회 · 대관 정보
```

### 배지 (optional)
```
OFFICIAL 또는 공식
```

## 3. 클릭 동작 (중요)

```typescript
onClick → /association/assoc-nowon-football
```

- 이미 배포된 협회 공식 페이지로 연결
- 로그인 여부와 관계없이 접근 가능 (Public)

---

## 4. 권한/노출 조건

- 모든 사용자에게 노출
- 관리자 여부와 무관
- 단순 링크 카드 (Edit Mode ❌)

---

## 5. 주의 사항 (절대 금지)

### ❌ 하지 않는 것
- ❌ 팀 카드 영역 안에 섞지 말 것
- ❌ "팀 찾기" 하위로 넣지 말 것
- ❌ 숨김/실험 플래그 적용 금지
- ❌ 팀 생성 로직과 연결하지 말 것
- ❌ 관리자 기능 노출 금지

**👉 이 카드는 '공식 주체의 입구'다**

## 6. 완료 기준 (Definition of Done)

1. ✅ 축구 허브 접속 시 카드 맨 앞에 노원구 축구협회 노출
2. ✅ 카드 클릭 시 협회 공식 페이지 정상 이동
3. ✅ 모바일/데스크탑 모두 정상 노출
4. ✅ 기존 카드 동작 영향 없음

## 📋 구현 가이드 (기술 힌트)

    route="/association/assoc-nowon-football"

<ActionGrid

  context={{ sport: type }}
