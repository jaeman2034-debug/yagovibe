# 🎯 Sports Hub 제안 분석

## 📋 제안 내용

### 플로우 변경
```
현재: /start → /login → /home (AI 리포트)
제안: /start → /login → /sports-hub (카테고리 허브) → /home (AI 리포트)
```

### 페이지 역할
```
/sports-hub (신규)
역할: 콘텐츠 중심 메인 허브
내용: 4x4 카테고리 그리드 (축구, 농구, 골프 등)
```

## 🔍 현재 상태

### 기존 플로우
```
/start (StartScreen)
├─ 로그인 → /login
├─ 회원가입 → /signup
└─ 게스트 로그인 → /home (AI 리포트)

/home (Home.tsx)
├─ AI 리포트 대시보드
├─ 3열 위젯
└─ PDF 생성 기능

CategoryGrid.tsx
├─ 마켓, 시설, 팀, 관리, 이벤트 아이콘
└─ 사용하지 않음
```

### 문제점
1. StartScreen에서 바로 `/home`으로 이동
2. CategoryGrid가 사용되지 않음
3. 커뮤니티 진입점이 애매함
