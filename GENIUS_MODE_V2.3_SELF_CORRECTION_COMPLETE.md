# 🔥 천재 모드 2.3 - 자기 교정 레이어 완성

## ✅ 구현 완료

### 1. 신뢰 점수 계산

**기능**:
- 기억 데이터 기반 신뢰도 계산
- 코스 길이 기반 신뢰도 조정
- 피드백 및 행동 점수 데이터 기반 신뢰도 증가

**신뢰도 요소**:
- favoriteTags > 3개 → +0.2
- favoriteTags > 0개 → +0.1
- favoriteCategories > 0개 → +0.1
- bestTime > 0개 → +0.1
- liked/hated 리스트 존재 → +0.1씩
- 코스 길이 >= 3 → +0.1
- 코스 길이 >= 2 → +0.05
- 최근 피드백 존재 → +0.1
- 행동 점수 데이터 > 10개 → +0.1
- 행동 점수 데이터 > 5개 → +0.05

**최대 신뢰도**: 0.95 (완벽하지 않다는 인정)

**파일**: `src/utils/selfCorrection.ts` (신규)

---

### 2. 의심 로직

**기능**:
- 신뢰도가 낮을 때 의심 표현
- 신뢰도 레벨별 다른 메시지

**의심 메시지**:
- 신뢰도 < 0.4: "아직 학습 중이에요. 더 알려주시면 더 잘 맞춰드릴게요 🙏"
- 신뢰도 < 0.6: "확신은 약하지만 시도해볼 만해요"
- 신뢰도 < 0.8: "괜찮은 선택일 거예요"
- 신뢰도 >= 0.8: 의심 표현 없음

**파일**: `src/utils/selfCorrection.ts`

---

### 3. 문장 통합

**기능**:
- 기본 문장 + 의심 메시지 통합
- 신뢰도가 높으면 의심 메시지 없음

**파일**: `src/utils/selfCorrection.ts`, `src/utils/courseBuilder.ts`
