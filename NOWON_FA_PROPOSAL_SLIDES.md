# 노원구 축구협회 도입 제안서 (PPT 슬라이드 텍스트)

**실전용 · 5장 · 그대로 사용 가능**

---

## 🟦 1장. 지금 노원구 축구협회의 현실

### 제목
```
팀 관리와 운동장 배정, 아직도 수작업이신가요?
```

### 본문 (3줄 핵심)
```
팀 등록은 여전히 전화, 카카오톡, 엑셀로 관리되고 있습니다

운동장 대관은 선착순·개별 요청으로 민원이 반복됩니다

누가 회원 팀인지 매번 확인해야 하는 구조입니다
```

### 하단 메시지 (강조 문구)
```
지금의 협회 운영은 '사람이 버티고 있는 구조'입니다
```

---

## 🟦 2장. 우리가 제안하는 방식

### 제목
```
협회는 승인만, 시스템이 나머지를 처리합니다
```

### 한 줄 정의 (중앙 배치)
```
팀은 먼저 만들고
협회는 신청된 팀만 승인합니다
```

### 구조 요약 (불릿 또는 도식 옆 텍스트)
```
팀 생성 (비회원)

협회 회원 신청

관리자 승인

혜택 자동 적용
```

### 하단 메시지
```
전화, 엑셀, 개인 판단을
시스템과 기준으로 대체합니다
```

---

## 🟦 3장. 실제 화면으로 보는 운영 흐름

### 소제목
```
실제 운영은 이렇게 단순해집니다
```

### 사용자 화면 (좌측 설명 텍스트)
```
팀이 직접 팀을 생성합니다

활동 지역으로 노원구를 선택합니다

노원구 축구협회 회원으로 신청합니다
```

### 관리자 화면 (우측 설명 텍스트)
```
회원 신청 팀 목록만 확인합니다

승인 / 반려 버튼으로 결정합니다

승인 기록은 자동으로 저장됩니다
```

### 말로 설명할 때 쓰는 문장 (슬라이드 하단)
```
협회는 '관리'가 아니라 '결정'만 합니다
```

---

## 🟦 4장. 공정한 운동장 배정 방식

### 제목
```
차별이 아닌, 질서입니다
```

### 규칙 (명확하게)
```
회원 팀: 이용일 7일 전부터 예약 가능

비회원 팀: 이용일 3일 전부터 예약 가능
```

### 효과 (3줄)
```
기준이 명확해 민원이 줄어듭니다

선착순·감정노동이 사라집니다

설명 없이도 화면으로 이해됩니다
```

### 하단 메시지
```
공정함은 '같음'이 아니라
'예측 가능함'에서 나옵니다
```

---

## 🟦 5장. 도입 효과와 다음 단계

### 제목
```
노원구 축구협회가 얻는 변화
```

### 도입 효과
```
팀 관리 자동화

대관 분쟁 및 민원 감소

협회 운영 신뢰도 상승
```

### 다음 단계 제안
```
노원구 축구협회 시범 운영

1~2개월 운영 데이터 확인

타 구 협회 확장 가능
```

### 마지막 한 줄 (엔딩 문장)
```
노원구가 기준이 됩니다
```

---

## 📋 사용 가이드

### 슬라이드 구성 예시

**1장: 현실 (문제 제시)**
- 제목: 큰 글씨, 중앙 배치
- 본문 3줄: 왼쪽 정렬, 불릿 포인트
- 하단 메시지: 기울임 또는 강조 색상

**2장: 제안 (해결책)**
- 제목: 큰 글씨
- 한 줄 정의: 중앙, 큰 글씨 (두 줄 나눔)
- 구조 요약: 도식화 또는 불릿
- 하단 메시지: 강조

**3장: 운영 흐름 (시연)**
- 좌측/우측 분할 또는 상하 분할
- 스크린샷 또는 간단한 다이어그램

**4장: 배정 방식 (정책)**
- 규칙: 표 또는 큰 글씨
- 효과: 불릿 포인트
- 하단 메시지: 명언 스타일

**5장: 다음 단계 (CTA)**
- 도입 효과: 불릿 또는 아이콘
- 다음 단계: 타임라인 또는 불릿
- 마지막 문장: 큰 글씨, 강조

## 💡 발표 팁

### 1장에서
- "지금의 협회 운영은 '사람이 버티고 있는 구조'입니다" 
- 이 문장을 읽으면서 잠시 멈추기
- 공감을 이끌어내는 순간

### 2장에서
- "팀은 먼저 만들고, 협회는 신청된 팀만 승인합니다"
- 이 문장을 천천히, 명확하게 읽기
- 핵심 메시지 강조

### 3장에서
- 실제 화면을 보여주면서 설명
- "협회는 '관리'가 아니라 '결정'만 합니다"
- 이 차이를 강조

### 4장에서
- "공정함은 '같음'이 아니라 '예측 가능함'에서 나옵니다"
- 협회장들이 가장 좋아하는 문장
- 잠시 멈추고 의미 전달

### 5장에서
- "노원구가 기준이 됩니다"
- 마지막 문장을 확신 있게 읽기
- 질문 받을 준비
