# 🧪 2️⃣ 실제 사용자 테스트 스크립트 (1번 MVP 기준)

## 🎯 테스트 목표 (이거 하나만 본다)

> **"키보드 없이 말로
> 주변 장소를 찾는 게
> 진짜로 더 편한가?"**

❌ 기능 평가
❌ UI 취향 조사
⭕ **편리함 체감 여부**

---

## 👥 테스트 대상 (정확히)

* 20~40대
* 운동/외출을 주 1회 이상 하는 사람
* 지도 앱을 **자주 쓰지만 좋아하진 않는 사람**

📌 **3~5명만 해도 충분**

---

## ⏱️ 총 소요 시간: 15분 / 1인

---

# 🧭 테스트 진행 순서 (스크립트 그대로 읽어도 됨)

---

## STEP 0. 사전 설명 (30초)

❌ 이렇게 말하지 말 것

> "이 앱은 음성 검색 지도입니다"
> "이 기능을 테스트해주세요"

✅ 이렇게만 말해라

> "지도 앱 하나 테스트하려고 해요.
> 그냥 평소처럼 써주세요."

**중요:**
* 기능 설명 ❌
* 사용법 안내 ❌
* 힌트 ❌

---

## STEP 1. 첫 인상 테스트 (1분)

### 행동

* 앱 실행
* **아무 설명 없이** 지도 화면만 보여줌
* 10초 정도 기다림

### 질문 1

> "이 앱, 뭐 하는 앱 같아요?"

### 성공 신호

* ✅ "말로 찾는 지도?"
* ✅ "주변 장소 찾는 앱?"
* ✅ "음성 검색 지도?"

### 실패 신호

* ❌ "잘 모르겠어요"
* ❌ "일반 지도 앱?"
* ❌ "뭘 하는 앱이에요?"

### 기록

```
[ ] 첫 인상: 명확함 / 애매함
[ ] 언급한 키워드: _______________
```

---

## STEP 2. 핵심 태스크 ① (3분)

### 과제

> "지금 여기서
> **근처 축구장을 찾아보세요**"

❗ 힌트 ❌
❗ 설명 ❌
❗ 도움 ❌

### 관찰 포인트

* 키보드 찾는가? (YES / NO)
* 말부터 하는가? (YES / NO)
* 망설임이 있는가? (몇 초?)
* 버튼을 찾는가? (YES / NO)

### 성공 신호

* ✅ 버튼 클릭 → 바로 말함
* ✅ "근처 축구장" 자연스럽게 말함
* ✅ 결과 보자마자 이해

### 실패 신호

* ❌ 키보드 찾음
* ❌ "어떻게 하면 되죠?" 질문
* ❌ 10초 이상 망설임

---

### 질문 2

> "방금 찾는 과정, 어땠어요?"

### 추가 질문 (필요시)

> "어려웠어요? 편했어요?"

### 성공 신호

* ✅ "편했어요"
* ✅ "말로 하니까 빠르네요"
* ✅ "간단하네요"

### 실패 신호

* ❌ "어려웠어요"
* ❌ "뭘 해야 할지 모르겠어요"
* ❌ "기존 지도가 더 편해요"

### 기록

```
[ ] 키보드 찾음: YES / NO
[ ] 말부터 함: YES / NO
[ ] 망설임 시간: _____초
[ ] "편하다" 언급: YES / NO
[ ] 인상적인 말: _______________
```

---

## STEP 3. 핵심 태스크 ② (3분)

### 과제

> "이번엔
> **지금 여는 헬스장** 찾아보세요"

### 관찰 포인트

* 다시 말하려는가?
* UI 헷갈림 있는가?
* 조건 검색 이해했는가?

### 성공 신호

* ✅ "지금 여는 헬스장" 자연스럽게 말함
* ✅ 결과 보자마자 이해

### 실패 신호

* ❌ "어떻게 말해야 하죠?" 질문
* ❌ 여러 번 시도

---

### 질문 3

> "이번에도 어렵진 않았어요?"

### 추가 질문 (필요시)

> "조건(지금 여는)을 넣는 게 자연스러웠어요?"

### 기록

```
[ ] 조건 검색 이해: YES / NO
[ ] 자연스러움: 높음 / 보통 / 낮음
[ ] 어려웠던 점: _______________
```

---

## STEP 4. 비교 질문 (2분)

### 질문 4

> "이거랑
> 네이버지도 / 카카오맵에서
> 같은 거 찾는 거랑 비교하면 어때요?"

### 추가 질문 (필요시)

> "어떤 게 더 편해요?"

### 성공 신호

* ✅ "이게 더 간단해요"
* ✅ "키보드 안 쳐도 돼서 좋아요"
* ✅ "이게 더 빠른 것 같아요"

### 위험 신호

* ⚠️ "기존 지도랑 비슷해요"
* ⚠️ "기존 지도가 더 편해요"

### 기록

```
[ ] 비교 결과: 이게 더 편함 / 비슷함 / 기존이 더 편함
[ ] 언급한 이유: _______________
```

---

## STEP 5. 자유 발화 테스트 (2분)

### 질문 5

> "그럼
> 지금 떠오르는 거 하나
> 그냥 말해보세요"

### 예시 (말하지 말 것)

* "근처 카페"
* "배드민턴장"
* "주차 되는 헬스장"

👉 **이 단계에서 자연스럽게 말 나오면 거의 성공**

### 관찰 포인트

* 자연스럽게 말하는가?
* 망설임이 있는가?
* 다양한 표현을 쓰는가?

### 성공 신호

* ✅ 즉시 말함
* ✅ 다양한 표현 사용
* ✅ "이렇게 말하면 되나요?" 질문 없음

### 실패 신호

* ❌ 10초 이상 망설임
* ❌ "뭘 말해야 하죠?" 질문

### 기록

```
[ ] 자연스러움: 높음 / 보통 / 낮음
[ ] 말한 내용: _______________
[ ] 망설임: YES / NO
```

---

### 기록

```

```

# ✅ 최종 성공 판정 기준 (이거 3개면 OK)

1. **말부터 했다** (STEP 2에서 키보드 찾지 않음)
2. **'편하다'라는 단어가 나왔다** (STEP 2, 4, 6 중 하나)
3. **기존 지도보다 빠르다고 느꼈다** (STEP 4)

👉 3명 중 2명만 만족해도

# 📊 종합 결과 분석

## 성공률 계산

```
말부터 한 비율 = 말부터 한 사람 / 전체 참가자
"편하다" 언급 비율 = "편하다" 언급한 사람 / 전체 참가자
기존보다 빠르다고 느낀 비율 = 빠르다고 느낀 사람 / 전체 참가자
```

## 최종 판정

### 성공 조건
- 말부터 한 비율 ≥ 60% (3명 중 2명)
- "편하다" 언급 비율 ≥ 60%
- 기존보다 빠르다고 느낀 비율 ≥ 40%

### 실패 조건
- 말부터 한 비율 < 40%
- "편하다" 언급 비율 < 40%
- 기존보다 빠르다고 느낀 비율 < 20%
