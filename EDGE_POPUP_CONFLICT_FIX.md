# 🔥 Edge 브라우저 팝업 설정 충돌 해결

## 🚨 발견된 문제

Edge 브라우저 설정에서 확인한 내용:
1. ❌ **충돌 발견**: `http://localhost:5173`이 **두 곳에 모두** 등록되어 있음
   - "팝업 또는 리디렉션을 사용할 수 없음" (차단 목록) ← 여기에 있으면 차단됨!
   - "팝업 전송 및 리디렉션 사용 허용" (허용 목록) ← 여기에 있어도 무의미
2. ❌ 주소창에 빨간색 X 아이콘 표시 (팝업 차단됨)
3. ⚠️ HTTP 연결 경고 ("이 사이트에 대한 연결이 안전하지 않습니다")

## 🎯 문제 원인

**차단 목록이 허용 목록보다 우선순위가 높습니다!**

같은 URL이 차단 목록과 허용 목록에 모두 있으면, 차단 목록의 설정이 적용됩니다.

## ✅ 해결 방법

### Step 1: 차단 목록에서 제거

1. **Edge 설정 페이지에서**
   - "팝업 또는 리디렉션을 사용할 수 없음" 섹션 찾기
   - `http://localhost:5173` 옆의 **X 버튼** 클릭하여 제거
   - 또는 `https://localhost:5173`도 제거 (있다면)

2. **확인**
   - "팝업 전송 및 리디렉션 사용 허용" 섹션에만 `http://localhost:5173`이 남아있는지 확인

### Step 2: 브라우저 재시작

1. **Edge 완전히 종료**
   - 모든 Edge 창 닫기
   - 작업 관리자에서 Edge 프로세스 확인 (있다면 종료)

2. **Edge 재시작**

### Step 3: 다시 테스트

1. **`http://localhost:5173/login` 접속**
2. **주소창 확인**
   - 빨간색 X 아이콘이 사라졌는지 확인
   - 또는 회색 자물쇠 아이콘으로 변경되었는지 확인

3. **"G 구글로 로그인" 버튼 클릭**
   - 팝업이 정상적으로 열리는지 확인

## 🔍 추가 확인 사항

### 차단 목록에 있는 다른 항목들도 확인:

이미지에서 확인된 차단 목록:
- `https://localhost:5173` ← 이것도 제거 필요!
- `http://localhost` ← 이것도 제거 필요!

**모두 제거하세요!**

### 허용 목록 확인:

허용 목록에 다음이 있어야 합니다:
- ✅ `http://localhost:5173` (현재 있음)
