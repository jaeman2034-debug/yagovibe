# Hero Section 최종 UI/UX 구조 확정

**목적**: 구청·체육회·회원·비회원 모두가 "아, 여기가 기준이구나"를 3초 안에 이해하게 만든다

---

## 1️⃣ Hero Section 전체 구조 (아키텍처)

### 레이아웃 구조
```
┌───────────────────────────────────┐
│ 노원구 축구협회                    │  ← Title (고정)
│                                   │
│ 노원구 내 축구 대회·대관·행정을    │  ← 행정/공식 문구
│ 공식적으로 운영합니다.            │
│                                   │
│ 노원구 축구의 모든 일정과 운영     │  ← 회원/현장 문구
│ 기준이 이곳에서 관리됩니다.       │
│                                   │
│ Association ID: assoc-nowon-football│ ← 보조 정보 (작게)
└───────────────────────────────────┘
```

---

## 2️⃣ 구성 요소 (순서 고정)

### 1. Title (제목)
```
노원구 축구협회
```
- **위치**: 최상단
- **스타일**: 큰 제목 (H1)
- **특징**: 단독, 명확, 간결

### 2. 행정/공식 문구 (구청·체육회용)
```
노원구 내 축구 대회·대관·행정을
공식적으로 운영합니다.
```
- **위치**: Title 아래
- **스타일**: 중간 크기 (H2 또는 P)
- **특징**: "공식" 명시, 행정 범위 명확

### 3. 회원/현장 문구
```
노원구 축구의 모든 일정과 운영
기준이 이곳에서 관리됩니다.
```
- **위치**: 행정/공식 문구 아래
- **스타일**: 중간 크기 (H2 또는 P)
- **특징**: "기준" 강조, 회원/비회원 모두 이해 가능

### 4. 보조 정보 (선택)
```
Association ID: assoc-nowon-football
```
- **위치**: 최하단
- **스타일**: 작은 텍스트
- **특징**: 기술적 식별자, 중요도 낮음

---

## 3️⃣ UX 원칙 (절대 고정)

### 금지 사항
- ❌ 버튼 없음
- ❌ CTA 없음
- ❌ 클릭 유도 없음
- ❌ 애니메이션 없음
- ❌ Quick Links 없음
- ❌ 스크롤 앵커 없음

### 원칙
**Hero는 행동을 요구하지 않는다**
→ 정의·권위·기준만 전달

---

## 4️⃣ 왜 이렇게 단순해야 하나?

### 구청 담당자
→ "아, 공식 페이지구나"
- 공식성 즉시 인지
- 행정 기준으로 인식

### 회원
→ "여기서 기준이 내려오는구나"
- 협회가 관리 주체임을 인지
- 기준점으로 인식

### 비회원
→ "문의하기 전에 여기부터 보는구나"
- 공식 정보 확인 장소로 인지
- 전화 전 확인 필수로 인식

### 효과
**분쟁·전화·설명 비용을 줄이는 구조**

---

## 5️⃣ 스타일 가이드 (참고용)

### 제목 (Title)
- 크기: `text-4xl` 또는 `text-5xl` (모바일: `text-3xl`)
- 굵기: `font-bold`
- 색상: `text-gray-900`
- 정렬: 중앙

### 문구 (공식/회원)
- 크기: `text-lg` 또는 `text-xl` (모바일: `text-base`)
- 색상: `text-gray-700`
- 정렬: 중앙
- 줄간격: `leading-relaxed`

### 보조 정보
- 크기: `text-xs` 또는 `text-sm`
- 색상: `text-gray-500`
- 정렬: 중앙
