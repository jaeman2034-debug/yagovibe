# 🔥 천재 모드 2.6 - 윤리/안전 레이어 완성

## ✅ 구현 완료

### 1. 과신 금지 (겸손 표현)

**기능**:
- 신뢰도 기반 겸손 표현
- 과하지 않은 추천
- 책임 있는 추천

**겸손 표현 규칙**:
- 신뢰도 < 0.4: "아직 학습 중이라 참고만 해주세요 🙏"
- 신뢰도 < 0.6: "아직 확신은 약하지만 시도해볼 만해요"
- 신뢰도 < 0.8: "꽤 괜찮은 선택일 거예요"
- 신뢰도 < 0.95: "잘 맞을 거예요 ✨"
- 신뢰도 >= 0.95: 겸손 메시지 없음 (완벽하지 않다는 인정)

**파일**: `src/utils/ethicsSafety.ts` (신규)

---

### 2. 위험 필터

**기능**:
- 안전한 장소만 추천
- 부적절한 콘텐츠 필터링
- 폐업한 장소 필터링

**필터링 규칙**:
- 성인 콘텐츠 태그 → 필터링
- 폐업한 장소 → 필터링
- 위험한 태그 → 필터링
- 평점 < 1.0 → 필터링
- 부적절한 키워드 → 필터링

**파일**: `src/utils/ethicsSafety.ts`

---

### 3. 최종 문장 통합

**기능**:
- 기본 문장 + 겸손 표현 통합
- 윤리적 문장 생성
- 책임 있는 추천 확인

**파일**: `src/utils/ethicsSafety.ts`, `src/utils/courseBuilder.ts`
