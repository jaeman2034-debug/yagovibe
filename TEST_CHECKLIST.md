# ✅ 실전 테스트 체크리스트

## 1️⃣ 기본 채팅 송수신
- [ ] 다른 계정으로 접속
- [ ] 메시지 보내기 / 받기
- [ ] 즉시 반영 확인
- [ ] 위치 공유 메시지 전송/수신

## 2️⃣ 읽음 처리 & 뱃지
- [ ] 상대가 메시지 보냄
- [ ] 내가 채팅방 안 열었을 때 → `/app/chat` 리스트에서 🔴 표시 확인
- [ ] 채팅방 들어가면 → 🔴 사라지는지 확인
- [ ] 하단 네비게이션 채팅 아이콘에 🔴 표시 확인

## 3️⃣ 상품 카드 표시
- [ ] 상품 이미지 보임
- [ ] 상품명 / 가격 보임
- [ ] [상품 보기] 링크 동작 확인
- [ ] 상품 삭제 시 "판매 종료" 메시지 표시

## 4️⃣ 모바일 STT 테스트 🎤
- [ ] 🎤 버튼 탭
- [ ] 마이크 권한 요청 확인
- [ ] 말하기 → 텍스트 입력창에 들어오는지
- [ ] **자동 전송 안 되는지** (중요!)
- [ ] 수정 후 전송 가능 여부
- [ ] 인식 중 빨간색 배경 표시

## 5️⃣ TTS 요약 테스트 🔊
- [ ] 채팅 몇 줄 만든 뒤
- [ ] [🔊 요약 듣기] 버튼 클릭
- [ ] **한 문장만 읽히는지**
- [ ] 중복 재생 안 되는지
- [ ] 재생 중 "중지" 버튼 동작

## 6️⃣ 추천 문장 테스트 🤖
- [ ] 입력창에 텍스트 없을 때 추천 문장 표시
- [ ] STT 입력 시 추천 문장 자동 숨김
- [ ] 추천 문장 탭 → 즉시 전송
- [ ] 추천 문장 롱프레스 → TTS 읽고 전송
- [ ] 상황 바뀌면 추천 문장 자동 교체

## 7️⃣ 푸시 알림 테스트 (제일 중요)

### 🤖 Android
- [ ] 앱 닫은 상태
- [ ] 상대가 메시지 보냄
- [ ] **푸시 도착 여부**
- [ ] 푸시 클릭 → 채팅방 이동
- [ ] FCM 토큰 Firestore 저장 확인

### 🍎 iOS
- [ ] Safari로 접속
- [ ] **홈 화면에 추가** (PWA)
- [ ] 다시 실행
- [ ] 메시지 보냄 → 푸시 도착?
- [ ] 푸시 클릭 → 채팅방 이동

## 8️⃣ 채팅 고정/숨김
- [ ] 채팅 고정 → 리스트 상단 이동
- [ ] 숨김 → 리스트에서 사라짐
- [ ] 새 메시지 와도 푸시는 오는지 확인
- [ ] "숨긴 채팅 보기" 버튼 동작

## 9️⃣ 에러 케이스
- [ ] 네트워크 끊김 시 메시지 전송 실패 처리
- [ ] 권한 거부 시 적절한 안내
- [ ] 상품 삭제 후 채팅 유지 확인

## 🐛 발견된 잠재적 이슈

### ⚠️ 확인 필요 사항

1. **롱프레스 타이머 스코프 문제**
   - 현재 `longPressTimer`가 컴포넌트 내부에 있어서 메모리 누수 가능성
   - `useRef`로 관리하는 것이 더 안전

2. **추천 문장 로딩 중 상태**
   - `isLoadingSuggestions`가 있지만 UI에 표시되지 않을 수 있음

3. **FCM 토큰 중복 저장**
   - 여러 파일에서 FCM 토큰 저장 로직이 있어서 중복 가능성
   - `registerFcmToken` 함수 통일 필요
