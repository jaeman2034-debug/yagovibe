## 🔥 한 문장 피치 (5가지 버전)

### 버전 A (정석)

```
전화번호 인증이 번거로운 서비스에서
회원가입 이탈 없이 바로 시작하게 해주는 서비스
```

### 버전 B (공격적)

```
가입 때문에 유저를 잃고 싶지 않은 팀을 위한
전화번호 기반 초간편 온보딩
```

### 버전 C (유저 관점)

```
앱 설치하고 번호만 입력하면 바로 쓰는 서비스
```

### 버전 D (문제 해결)

```
운동 팀 찾기 어려워서
전화번호로 바로 가입하는 서비스
```

### 버전 E (시간 강조)

```
이메일 인증 기다리지 말고
1분 안에 시작하는 서비스
```

## 🔥 문제 설명 글 템플릿 (브런치/노션)

```
# [문제] 때문에 직접 만들어봤습니다

## 1. 내가 겪은 문제

[개인적 경험을 솔직하게]

예를 들어, 운동 팀을 찾기 위해 여러 앱을 써봤는데
회원가입이 너무 복잡했어요.
이메일 인증하고, 비밀번호 만들고, 프로필 입력하고...
이미 지쳐서 포기한 적이 많았습니다.

## 2. 기존 해결법의 한계

[불편함을 구체적으로]

기존 서비스들은:
- 회원가입 절차가 너무 복잡함
- 이메일 인증을 기다려야 함
- 비밀번호를 기억해야 함

이런 번거로움 때문에
실제로 서비스를 사용하기 전에 포기하는 경우가 많았어요.

## 3. 그래서 만든 방법

[해결책을 간단하게]

그래서 [한 문장 피치]를 만들었어요.

전화번호만 입력하면 바로 시작할 수 있어서,
회원가입 때문에 포기하는 일이 없어요.

## 4. 아직 부족한 점

[솔직한 고백]

아직 초기 단계라:
- 기능이 부족할 수 있어요
- UX가 완벽하지 않을 수 있어요
- 버그가 있을 수 있어요

하지만 핵심 기능은 동작하고,
지속적으로 개선하고 있습니다.

## 5. 써보고 말해달라

[피드백 요청]

비슷한 문제를 겪고 계신 분들,
혹시 써보시고 뭐가 불편한지 알려주시면
정말 감사하겠습니다!

[링크]

피드백 주시면 정말 감사하겠습니다!
```
