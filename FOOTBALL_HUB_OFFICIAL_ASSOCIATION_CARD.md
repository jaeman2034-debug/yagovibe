# 축구 허브 「노원구 축구협회 공식 카드」 UI 문구 + 구조 최종 확정

**목적**: 축구 허브 최상단에 공식 주체 카드를 추가하여 IA(정보 구조) 기준선 확립

---

## 1️⃣ 카드의 정체성 (한 줄 정의)

**노원구 축구 행정의 공식 기준점**

- 이 카드는 "기능 카드"가 아님
- ⭕ **공식 주체가 여기 있다는 선언용 카드**

---

## 2️⃣ 카드 위치 (IA 고정)

### 위치
- **축구 허브 최상단**
- **모든 개인/팀 기능 카드 위**

### IA 구조
```
[축구]
────────────────
[🏛 노원구 축구협회]  ← 공식 주체 (NEW)
────────────────
[우리 팀 관리]
[팀 / FC 찾기]
────────────────
[일정]
[근처 시설]
[거래 장터]
```

---

## 3️⃣ 카드 구조 (UI 구성)

### 1. 아이콘
- **🏛** 또는 **⚽🏛** (행정/공식 느낌)
- 팀/사람 아이콘 사용 금지 ❌

### 2. 메인 타이틀 (고정)
```
노원구 축구협회
```
- 부제 없음
- "공식성"은 짧을수록 강하다

### 3. 서브 문구 (1줄 · 최종안)
```
노원구 축구 행정 · 대회 · 대관 공식 운영
```
- 구청/회원/비회원 모두 이해 가능
- "플랫폼" "서비스" 같은 IT 용어 ❌
- 행정 주체임을 명확히

### 4. 상태 배지 (작게)
- **"공식"** 또는 **"Verified"**
- 신뢰 시각화용, 클릭 유도 아님

### 5. 클릭 동작 (단일)
```
/association/assoc-nowon-football
```
- 로그인 ❌
- 권한 ❌
- 읽기 전용 공식 페이지 진입

---

## 4️⃣ 카드 레이아웃 예시

### 데스크탑/모바일 공통 구조
```
┌─────────────────────────────────────┐
│ 🏛  노원구 축구협회                  │
│                                     │
│ 노원구 축구 행정 · 대회 · 대관      │
│ 공식 운영                            │
│                                     │
│ [공식]                              │
└─────────────────────────────────────┘
```

### 시각적 특징
- **배경**: 밝은 회색 또는 흰색 (`bg-gray-50` 또는 `bg-white`)
- **테두리**: 얇은 테두리 (`border border-gray-200`)
- **아이콘 크기**: 중간 크기 (팀 카드보다 약간 크게)
- **배지 위치**: 우측 상단 또는 타이틀 옆

---

## 5️⃣ 이 카드에 넣지 않는 것 (중요)

### 금지 사항
- ❌ 신청 버튼
- ❌ 팀 생성 버튼
- ❌ 버튼 2개 이상
- ❌ CTA 문구 ("바로가기", "더 보기" 등)
- ❌ "참여하기" / "가입하기" 문구
- ❌ "회원 혜택" 강조

### 이유
이 카드는 **"들어오라"가 아니라 "여기가 기준이다"**를 말하는 카드다.

---

## 6️⃣ 클릭 동작 상세

### 클릭 시
- **동작**: `/association/assoc-nowon-football`로 이동
- **권한 체크**: ❌ 없음 (Public 접근)
- **로그인 요구**: ❌ 없음
- **새 창**: ❌ 현재 창에서 이동

### 클릭 불가 영역
- 배지 클릭: 동일 동작 (협회 페이지로 이동)
- 카드 전체: 클릭 가능

---

## 7️⃣ 반응형 UI

### 모바일 화면
- 카드 너비: 전체 너비
- 아이콘: 중간 크기 유지
- 텍스트: 적절한 크기 (`text-base` 또는 `text-lg`)
- 패딩: 적절한 여백 (`p-4`)

### 데스크탑 화면
- 카드 너비: 다른 카드와 동일 (그리드 시스템)
- 레이아웃: 아이콘 + 텍스트 가로 배치 가능
- 패딩: 넓은 여백 (`p-6`)

---

## 8️⃣ 시각적 위계 (다른 카드와의 차별화)

### 공식 카드 특징
- **배경색**: 다른 카드보다 약간 진하게 또는 구분되는 색상
- **아이콘**: 행정/공식 아이콘 (🏛)
- **배지**: "공식" 또는 "Verified" 표시
- **크기**: 다른 카드와 동일하되, 위치가 최상단

### 시각적 우선순위
1. **공식 카드** (최상단, 행정 아이콘)
2. **팀 카드** (중간, 참여 주체)
3. **유틸리티 카드** (하단, 개인 활동)

---

## 9️⃣ 이 카드가 만들어내는 효과

### 효과 1: 무의식적 인식
- 사용자는 무의식적으로 협회를 최상위로 인식
- 팀/개인 활동이 "산하 활동"으로 정렬됨

### 효과 2: 구조적 설득력
- 나중에 구청 담당자에게 보여줘도 구조적으로 설득력 있음
- "공식 주체 → 참여 주체 → 개인 활동" 흐름이 명확

### 효과 3: 분쟁 차단
- "노원구 축구의 공식은 어디?" 질문 즉시 해소
- 전화 문의 감소
