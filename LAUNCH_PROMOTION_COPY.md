# 📢 초기 홍보 글 문구 모음

## 🎯 핵심 원칙

1. **거창하게 말하지 말기**
2. **문제 하나만 해결한다고 말하기**
3. **3초 안에 이해되게**
4. **커뮤니티 중심으로 접근**

---

## 📱 카카오톡방/밴드용

### 버전 1: 간단한 소개
```
🔥 동네에서 축구화 거래할 사람 여기 모여요

새로 만든 스포츠 장비 거래 플랫폼입니다.
지금 가입하면 초기 유저 혜택도 있어요!

👉 [링크]
```

### 버전 2: 문제 해결 중심
```
동네에서 스포츠 장비 거래 어려우셨죠?
당근마켓은 너무 많고, 전문 사이트는 복잡하고...

그래서 만든 게 있어요 👇
축구·농구·러닝 장비만 모은 거래 플랫폼

👉 [링크]
```

### 버전 3: 혜택 중심
```
초기 유저 모집 중! 🎁

동네 스포츠 장비 거래 플랫폼 만들어봤어요.
지금 가입하면:
- 등록 수수료 무료
- 초기 유저 뱃지
- 거래 완료 시 리뷰 작성 시 추가 혜택

👉 [링크]
```

## 💡 활용 팁

1. **커뮤니티 특성에 맞게 수정**
   - 카톡방: 간단하고 친근하게
   - 당근마켓: 문제 해결 중심
   - 인스타: 해시태그 활용

2. **반복하지 말기**
   - 같은 문구를 여러 곳에 올리지 말기
   - 각 커뮤니티에 맞게 수정하기

3. **피드백 수집**
   - 어떤 문구가 반응이 좋은지 체크
   - 좋은 반응 받은 문구는 재활용

4. **타이밍 중요**
   - 저녁 8시~10시 (활발한 시간대)
   - 주말 오후 (여유로운 시간대)
