# 〈사무국 업무 범위〉

**Phase 5: 사무국 무손(無手) 시뮬레이션 완료 후 재정의**

---

## ✅ 승인

사무국이 **승인만** 하는 업무:

- 예외 상황 승인
- 특별 요청 승인
- 정책 변경 승인
- 긴급 조치 승인

**특징:**
- 시스템이 제안 → 사무국이 승인/거절만
- 입력/계산/전달은 시스템이 자동 처리

---

## ✅ 예외 처리

사무국이 **예외만** 처리하는 업무:

- 시스템 오류 대응
- 분쟁 조정
- 긴급 상황 대응
- 특수 케이스 처리

**특징:**
- 정상 흐름은 시스템이 자동 처리
- 예외만 사무국이 개입

### ❌ 회비 입력
- **이전**: 사무국이 직접 입력
- **현재**: 시스템 자동화
- **사무국 역할**: 예외 처리만

### ❌ 정산 입력
- **이전**: 사무국이 직접 계산/입력
- **현재**: 시스템 자동화
- **사무국 역할**: 승인만
