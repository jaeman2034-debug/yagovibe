# 노원구 축구협회 도입 제안 발표 멘트 (그대로 읽어도 되는 버전)

## 🎯 발표 정보

**대상:** 협회장 / 사무국 / 실무자

**목표:** "한 번 써보자"를 끌어내는 것

**톤:** 설명 ❌ / 공감 ⭕ / 안심 ⭕

**시간:** 10분

---

## ⏱️ 0–1분 | 오프닝 (관심 고정)

```
오늘은 새로운 시스템을 소개하려고 온 게 아닙니다.

지금 협회에서 가장 힘든 일을 덜어드리려고 왔습니다.
```

**[2초 멈춤]**

```
팀 관리, 운동장 배정…

사실 시스템보다 사람이 버티고 있는 구조잖아요.
```

---

## ⏱️ 1–3분 | 문제 공감 (1장)

```
지금은 팀 등록이 전화, 카톡, 엑셀로 나뉘어 있고

운동장 배정은 선착순이나 개별 요청으로 민원이 반복됩니다.
```

**[1초 멈춤]**

```
그리고 제일 어려운 건

'이 팀이 회원 팀인지 아닌지'를 매번 사람이 확인해야 한다는 점입니다.
```

### 👉 핵심 멘트 (강조, 천천히)

```
이건 누군가가 열심히 해서 버티는 구조지,

오래 갈 수 있는 구조는 아닙니다.
```

**[3초 멈춤 - 공감 시간]**

---

## ⏱️ 3–4분 | 해결 방식 제시 (2장)

```
그래서 저희가 제안하는 방식은 아주 단순합니다.
```

**[1초 멈춤]**

### (천천히, 명확하게)

```
'협회는 승인만 하고, 나머지는 시스템이 합니다.'
```

**[2초 멈춤]**

```
팀은 먼저 스스로 팀을 만들고,

협회에는 가입 신청만 합니다.
```

```
협회는 신청된 팀만 보고,

받을지 말지만 결정하시면 됩니다.
```

### 👉 강조 (명확하게)

```
전화도, 엑셀도, 개인 판단도 필요 없습니다.
```

---

## ⏱️ 4–6분 | 실제 운영 흐름 (3장)

```
실제 화면으로 보면 더 단순합니다.
```

**[슬라이드 또는 데모 화면 표시]**

### 사용자 쪽 설명

```
팀은 스스로 팀을 만들고

활동 지역으로 노원구를 선택합니다.
```

```
그리고 노원구 축구협회 회원으로 신청합니다.
```

**[화면 전환 - 관리자 화면]**

### 관리자 쪽 설명

```
관리자 화면에는

'회원 신청 팀 목록'만 뜹니다.
```

```
팀 정보 보고

승인 또는 반려만 누르시면 됩니다.
```

### 👉 핵심 한 줄 (천천히, 강조)

```
협회는 '관리'가 아니라 '결정'만 합니다.
```

**[3초 멈춤 - 반응 관찰]**

```
이 차이가 중요합니다.

관리는 계속 해야 하는 일이고,

결정은 한 번만 하면 되는 일이거든요.
```

---

## ⏱️ 6–8분 | 운동장 배정 규칙 (4장 – 가장 중요)

```
협회에서 가장 민감한 부분이

운동장 배정이라는 걸 알고 있습니다.
```

**[1초 멈춤]**

```
그래서 기준을 아주 단순하게 만들었습니다.
```

### (명확하게, 숫자 강조)

```
회원 팀은 7일 전부터,

비회원 팀은 3일 전부터 예약합니다.
```

**[1초 멈춤]**

```
이건 차별이 아니라 질서입니다.
```

### 👉 설득 멘트

```
누가 먼저 전화했는지가 아니라

상태에 따라 자동으로 결정됩니다.
```

```
그래서 설명할 필요가 없습니다.

화면이 대신 설명합니다.
```

**[데모 화면 보여주기 - 가능하면]**

```
달력에서 보시면,

회원 팀은 D-7일부터 모든 시간대가 열려 있고,

비회원 팀은 D-3일부터만 열립니다.

이렇게 시각적으로 보이니까

'왜 안 되냐'는 질문 자체가 줄어들어요.
```

---

## ⏱️ 8–9분 | 도입 효과 (5장)

```
이 구조가 적용되면

협회에서 바뀌는 건 이 세 가지입니다.
```

### (각각 1초씩 멈춤)

```
첫째, 팀 관리 자동화입니다.

전화 문의가 80% 이상 줄어들 겁니다.
```

```
둘째, 대관 민원 감소입니다.

기준이 명확하니까 분쟁이 없어요.
```

```
셋째, 협회 운영 신뢰도 상승입니다.

투명하고 공정한 시스템이니까요.
```

**[1초 멈춤]**

```
무엇보다

'왜 우리는 안 되나요?'라는 질문이 줄어듭니다.
```

---

## ⏱️ 9–10분 | 마무리 & 다음 단계

```
저희 제안은

한 번에 다 바꾸자는 게 아닙니다.
```

```
노원구 축구협회에서

시범 운영부터 해보자는 제안입니다.
```

```
1~2개월만 운영해 보시고

판단하셔도 늦지 않습니다.
```

**[2초 멈춤]**

### 마지막 한 문장 (천천히, 확신 있게)

```
노원구가 기준이 됩니다.
```

**[3초 멈춤 - 질문 받을 준비]**

```
노원구 축구협회가 1호 고객이 되어,

전국 축구협회의 표준 모델이 될 수 있습니다.

감사합니다. 질문 있으시면 언제든 말씀해 주세요.
```

---

## 📌 발표 팁 (중요)

### ⚠️ 절대 빨리 말하지 마라

- 각 문장 후 1~2초 멈춤
- 중요한 문장은 3초 멈춤
- 공감을 이끌어내는 순간에는 더 길게 멈춤

### 💬 "시스템"보다 "사람"을 계속 언급해라

❌ "시스템이 자동으로 처리합니다"
⭕ "사람이 할 일을 시스템이 대신 합니다"

❌ "기술적으로 구현했습니다"
⭕ "협회에서 힘들어하시는 부분을 자동화했습니다"

### 🔧 기술 질문 나오면 이렇게 말해라

```
기술은 저희가 책임지고,

협회는 운영만 보시면 됩니다.
```

또는

```
복잡한 건 제가 다 처리하고,

협회에서는 버튼만 누르시면 됩니다.
```

## 💡 응급 대응

### 시간이 부족할 때
- 3장 (실제 화면) 간략히 설명
- 4장 (운동장 배정)은 반드시 포함
- 5장 (마무리)는 빠르게

### 시간이 남을 때
- 3장 데모 화면 자세히 보여주기
- 4장 규칙 설명 보강
- 질문 시간 확대

### 반응이 없을 때
- "질문 있으시면 언제든 말씀해 주세요"로 유도
- 또는 "지금까지 설명한 내용 중 궁금한 점이 있으신가요?"

3. **"관리가 아니라 결정"** (3장)
