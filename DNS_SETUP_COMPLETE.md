## 🔍 다음 단계

### **1단계: DNS 전파 확인**

DNS 변경 사항이 전 세계로 전파되는 데 **5-15분** 소요될 수 있습니다.

### **2단계: Firebase 콘솔에서 검증**

1. **Firebase 콘솔** → Hosting → 커스텀 도메인
2. `yagovibe.com` 클릭
3. **"검증" 또는 "다시 확인"** 버튼 클릭
4. 상태가 "연결됨"으로 변경될 때까지 대기

### **3단계: SSL 인증서 발급 대기**

- DNS 인증 완료 후 Firebase가 자동으로 SSL 인증서 발급
- 발급 시간: **최대 15분~1시간**
