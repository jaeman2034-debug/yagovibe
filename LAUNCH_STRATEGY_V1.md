# 🚀 마켓 런칭 전략 v1

## 📋 런칭 전 최소 조건 체크리스트

### ✅ 1단계: 게시글 시드 확보 (최우선)

**목표**: 최소 30~50개 글, 종목별 최소 5개씩

- [ ] **지인/커뮤니티에서 물건 사진 받아서 대신 등록**
  - 동네 카톡방/밴드에서 "중고 거래 도와주세요" 공지
  - 사진만 받아서 관리자가 대신 등록
  - 초기엔 무료로 등록 서비스 제공

- [ ] **중고 거래글 허락 받고 옮겨오기**
  - 당근마켓/중고나라에서 허락 받고 옮기기
  - "새 플랫폼에 올려도 될까요?" 직접 연락
  - 초기엔 수동으로 옮겨도 OK

- [ ] **관리자 계정으로 등록**
  - 초기엔 관리자가 직접 등록해도 OK
  - 실제 물건이면 더 좋지만, 테스트용도 허용
  - **핵심**: 유저가 처음 들어왔을 때 "거래되는 느낌"만 주면 됨

**체크 포인트**:
- [ ] 축구 관련 게시글 5개 이상
- [ ] 농구 관련 게시글 5개 이상
- [ ] 러닝 관련 게시글 5개 이상
- [ ] 기타 종목 게시글 5개 이상
- [ ] 총 게시글 수 30개 이상

---

### ✅ 2단계: 첫 거래 5건 만들기

**목표**: 리뷰 5개 확보 (신뢰도 확보)

- [ ] **지인끼리 거래 유도**
  - 지인 A가 글 올리고, 지인 B가 구매
  - 실제 거래라면 더 좋지만, 테스트 거래도 OK
  - 거래 완료 후 리뷰 작성 유도

- [ ] **테스트 거래 진행**
  - 관리자가 판매자/구매자 역할 분담
  - 실제 거래 흐름 테스트
  - 리뷰 작성까지 완료

- [ ] **리뷰 5개 확보**
  - 리뷰 5개 생기면 신뢰도가 확 달라짐
  - 평점 4.0 이상 유지
  - 리뷰 내용도 어느 정도 작성

**체크 포인트**:
- [ ] 거래 완료 5건 이상
- [ ] 리뷰 5개 이상
- [ ] 평균 평점 4.0 이상

---

### ✅ 3단계: 홈 첫 화면 메시지

**목표**: 3초 안에 이해되게

- [ ] **한 줄 설명 작성**
  - 예시: "축구·농구·러닝 장비를 사고파는 마켓"
  - 예시: "동네 스포츠 장비 거래 플랫폼"
  - 예시: "스포츠 장비 중고거래, 여기서 시작하세요"

- [ ] **홈 화면에 표시**
  - 상단 헤더 또는 히어로 섹션
  - 간단하고 명확하게
  - 3초 안에 이해 가능

**체크 포인트**:
- [ ] 한 줄 설명 작성 완료
- [ ] 홈 화면에 표시 완료
- [ ] 테스트 사용자에게 이해도 확인

## 🎯 초기 유저 확보 전략

### 1️⃣ 타겟 좁히기 (중요)

**원칙**: 모든 스포츠 X → 1~2개 종목만 집중

**추천 종목**:
- 축구 (가장 활발한 커뮤니티)
- 러닝 (개인 장비 거래 많음)

**전략**:
- 이 종목이 커지면 다른 종목 붙이기
- 플랫폼은 처음에 좁을수록 성공률 높음

---

### 2️⃣ 유입 채널 (현실적인 순서)

#### 1순위: 동네 카톡/밴드/오픈채팅
- 동네 카톡방에 공지
- 스포츠 동호회 밴드에 공지
- 오픈채팅방에 링크 공유

#### 2순위: 당근 동네 게시판
- "동네 스포츠 장비 거래 플랫폼 만들어봤어요" 글 작성
- 실제 거래 가능한 물건 올리기

#### 3순위: 인스타 지역 해시태그
- 지역 해시태그 활용 (#서울축구 #강남러닝 등)
- 스포츠 관련 해시태그 활용

#### 4순위: 학교/동호회 단톡방
- 학교 축구부/농구부 단톡방
- 동호회 단톡방

**원칙**: 광고 말고 "커뮤니티"부터

---

### 3️⃣ 유입 문구 예시

#### 카톡방/밴드용
```
🔥 동네에서 축구화 거래할 사람 여기 모여요
새로 만든 스포츠 장비 거래 플랫폼입니다.
지금 가입하면 초기 유저 혜택도 있어요!

👉 [링크]
```

#### 당근마켓용
```
동네 스포츠 장비 거래 플랫폼 만들어봤어요
축구·농구·러닝 장비 사고팔기 좋은 곳입니다.
지금 가입하면 초기 유저 혜택도 있어요!

👉 [링크]
```

#### 인스타용
```
#서울축구 #중고거래
동네에서 축구화 거래할 사람 여기 모여요
새로 만든 스포츠 장비 거래 플랫폼입니다.

👉 [링크]
```

**핵심**: 거창하게 말하지 말고 **문제 하나만 해결한다고 말하기**

## 📊 첫 2주 운영 전략

### 매일 체크할 것

**3가지 지표만 본다**:
1. 오늘 게시글 수
2. 오늘 채팅 수
3. 오늘 거래 완료 수

**체크 시간**: 매일 저녁 8시

---

### 해야 할 행동

#### 게시글이 없으면
- 직접 등록 (관리자 계정으로)
- 지인에게 부탁해서 등록
- 커뮤니티에 "글 올려주세요" 공지

#### 거래가 안 되면
- 가격 조정 유도 (판매자에게 DM)
- "가격 좀 내려보시겠어요?" 직접 제안
- 거래 성사되면 리뷰 작성 유도

#### 채팅이 느리면
- 판매자에게 DM으로 "채팅 확인해주세요" 알림
- 구매자에게 "판매자에게 직접 연락해보세요" 제안

**원칙**: 초반엔 **플랫폼이 아니라 수동 운영**
