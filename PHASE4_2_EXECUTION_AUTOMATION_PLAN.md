## Phase 4-2에서 바로 진행되는 것 (설명 X, 실행 O)

### 1️⃣ 공지 입력 자동화 (Admin 전용)

**위치**: `/association/:id/admin/notices`

**기능**:
- 작성 / 수정 / 예약 게시
- Public 페이지에 즉시 반영

**결과**:
- ❌ 단톡 공지
- ❌ 전화 공지
- ❌ 엑셀/워드 초안

---

### 2️⃣ 대회/일정 = 입력 1번 → 자동 노출

**프로세스**:
- Admin이 대회 1건 입력
- 자동 생성:
  - 협회 공식 페이지 TournamentSection
  - 일정 카드

**결과**:
- "대진표 나왔어요?" 전화 0건 목표

---

### 3️⃣ 대관 상태 = 토글 → 기준 화면 고정

**권한**:
- **Admin만**: available / blocked / event 토글
- **Public**: 읽기 전용

**결과**:
- 회원 vs 비회원 분쟁 사전 차단

---

### 4️⃣ 엑셀 제거 기준 (성공 조건)

아래 중 **2개 이상**이면 Phase 4-2 성공:

1. ✅ 사무국이 "엑셀 안 켰다"
2. ✅ 구청 보고 자료가 화면에서 바로 나온다
3. ✅ 공지/대회 관련 문의가 "이미 봤어요"로 끝난다
