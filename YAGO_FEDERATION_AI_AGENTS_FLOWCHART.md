## 1️⃣ AI 에이전트 전체 구조

### 에이전트 계층

```
대표 AI 비서 (홈페이지 우하단)
  ↓
역할별 에이전트 라우팅
  ├─ 대회 안내 에이전트
  ├─ 경기 운영 에이전트
  ├─ 팀/선수 등록 에이전트
  ├─ 규정/문서 에이전트
  ├─ 협회 행정 에이전트
  ├─ 후원/제휴 에이전트
  └─ 신규 협회 생성 에이전트
```

### 에이전트 배치 위치

```
퍼블릭 홈페이지
  └─ 대표 AI 비서 (우하단 챗봇)

관리자 대시보드
  └─ AI 운영실 (/admin/ai)
      ├─ AI 대시보드
      └─ 에이전트별 패널
```

## 2️⃣ 대표 AI 비서

### 역할

**홈페이지 전체 검색 및 안내**

### 질문 예시

```
"이번 대회 일정 알려줘"
"노원구청장기 참가팀이 몇 팀이야?"
"유소년 팀 등록은 어디서 해?"
"대진표 보여줘"
"협회 연락처 알려줘"
```

### 기능

```
1. 홈페이지 전체 검색
2. 대회/공지/일정/규정 빠른 응답
3. 해당 페이지로 즉시 이동
4. 민원/문의 접수
5. 역할별 에이전트로 라우팅
```

### 플로우차트

```
사용자 질문
  ↓
의도 분석
  ↓
┌─────────────────┬─────────────────┬─────────────────┐
│ 대회 관련       │ 경기 관련       │ 규정 관련       │
│ → 대회 안내     │ → 경기 운영     │ → 규정/문서     │
│   에이전트      │   에이전트      │   에이전트      │
└─────────────────┴─────────────────┴─────────────────┘
  ↓
응답 생성
  ↓
페이지 이동 (필요시)
```

## 3️⃣ 대회 안내 에이전트

### 역할

**브로슈어 기반 대회 정보 제공**

### 질문 범위

```
"2025 노원구청장기 대회 일정이 뭐야?"
"참가 자격 알려줘"
"부별 구성이 어떻게 돼?"
"참가비는 얼마야?"
"신청 마감일이 언제야?"
"대진표 보여줘"
```

### 자동 기능

```
1. 대회 페이지 생성
2. 요강 요약
3. 참가 안내 문구 생성
4. FAQ 자동 생성
5. 공지 초안 생성
```

### 플로우차트

```
대회 관련 질문
  ↓
대회 정보 조회 (Firestore)
  ↓
브로슈어 데이터 참조
  ↓
응답 생성
  ├─ 대회 개요
  ├─ 일정표
  ├─ 참가 자격
  ├─ 규정 요약
  └─ 관련 페이지 링크
```

## 4️⃣ 경기 운영 에이전트

### 역할

**운영진용 경기 관리 보조**

### 질문 범위

```
"이번주 경기 일정 생성해줘"
"경기장 배정 추천해줘"
"심판 배정 도와줘"
"결과 입력 검증해줘"
"순위 자동 계산 확인해줘"
```

### 자동 기능

```
1. 경기 일정 생성 보조
2. 대진표 자동 생성
3. 경기장 배정 추천
4. 심판 배정 보조
5. 결과 입력 검증
6. 순위 자동 계산 확인
7. 연기/취소 공지 초안 작성
```

### 플로우차트

```
운영 업무 요청
  ↓
┌─────────────────┬─────────────────┬─────────────────┐
│ 일정 생성       │ 결과 검증        │ 공지 작성       │
│ → Round Robin   │ → 데이터 검증   │ → 템플릿 생성   │
│   알고리즘      │ → 순위 계산      │ → 초안 작성     │
└─────────────────┴─────────────────┴─────────────────┘
  ↓
결과 확인
  ↓
운영진 승인
```

## 5️⃣ 팀/선수 등록 에이전트

### 역할

**팀 관리자와 사무국용 등록 관리**

### 질문 범위

```
"이 선수는 해당 부 출전 가능?"
"서류 누락된 팀이 어디야?"
"명단 등록 안 끝난 팀 알려줘"
"중복 등록 체크해줘"
```

### 자동 기능

```
1. 참가 신청 안내
2. 선수등록 서류 체크
3. 중복 등록 탐지
4. 출전 가능 여부 확인
5. 연령 규정 검토
6. 명단 누락 점검
```

### 플로우차트

```
등록 관련 질문/요청
  ↓
데이터 조회 (팀/선수 정보)
  ↓
┌─────────────────┬─────────────────┬─────────────────┐
│ 서류 검증       │ 중복 체크        │ 규정 검토       │
│ → 필수 서류     │ → 선수 중복     │ → 연령 규정     │
│   확인          │   확인          │   확인          │
└─────────────────┴─────────────────┴─────────────────┘
  ↓
검수 결과 리포트
  ↓
승인/반려 제안
```

**조직/임원/행사 운영**

```
"회의 일정 정리해줘"
"행사 안내문 작성해줘"
"임원 명단 업데이트해줘"
"조직도 페이지 생성해줘"
```

### 자동 기능

```
1. 회의 일정 정리
2. 행사 안내문 작성
3. 임원 명단 관리
4. 조직도 페이지 업데이트
5. 행정 문서 초안 생성
6. 대외 공문 작성 보조
```

### 플로우차트

```
행정 업무 요청
  ↓
┌─────────────────┬─────────────────┬─────────────────┐
│ 문서 작성       │ 일정 관리       │ 조직 관리       │
```

## 9️⃣ 신규 협회 생성 에이전트

### 역할

**신규 협회 SaaS 인스턴스 자동 생성**

### 입력값

```
협회명
지역
종목
로고
대표 색상
협회 소개
운영자 정보
기본 대회 유형
연령/부 구성
기본 문서 양식
```

### 자동 생성 항목

```
1. 협회 홈페이지
2. 관리자 대시보드
3. 공지 게시판
4. 대회/리그 모듈
5. 팀/선수 등록 모듈
6. 경기 일정/결과 모듈
7. 순위/기록 모듈
8. 규정/자료실 모듈
9. 후원사/광고 모듈
10. AI 에이전트 세트
```

### 플로우차트

```
신규 협회 생성 요청
  ↓
기본 정보 입력
  ↓
템플릿 선택 (종목별)
  ↓
┌─────────────────────────────────────────┐
│ 자동 생성 프로세스                       │
│                                         │
│ 1. Federation Document 생성            │
│ 2. Homepage Template 생성               │
│ 3. 기본 메뉴 구조 생성                   │
│ 4. 관리자 계정 설정                      │
│ 5. 기본 규정/문서 생성                   │
│ 6. AI 에이전트 초기화                    │
│ 7. 테마/디자인 적용                      │
└─────────────────────────────────────────┘
  ↓
협회 홈페이지 활성화
  ↓
운영 시작 가능
```

## 🔟 AI 에이전트 통합 플로우

### 전체 플로우차트

```
사용자 질문/요청
  ↓
의도 분석 (NLP)
  ↓
┌─────────────────────────────────────────┐
│ 에이전트 라우팅                          │
│                                         │
│ 대회 관련 → 대회 안내 에이전트          │
│ 경기 관련 → 경기 운영 에이전트          │
│ 등록 관련 → 팀/선수 등록 에이전트       │
│ 규정 관련 → 규정/문서 에이전트          │
│ 행정 관련 → 협회 행정 에이전트          │
│ 후원 관련 → 후원/제휴 에이전트          │
│ 생성 요청 → 신규 협회 생성 에이전트     │
└─────────────────────────────────────────┘
  ↓
에이전트별 처리
  ↓
응답 생성
  ↓
사용자에게 전달
  ↓
필요시 페이지 이동/액션 실행
```
