# 🧭 음성 검색 → 지도 결과 정확한 플로우

> 한 줄도 안 흔들리게 고정된 화면 단계

---

## STEP 0. 초기 지도 화면

**사용자 상태:**
- 지도 열림
- 내 위치 보임 (파란 점)
- 하단에 음성 힌트 ("지금 말해도 돼요")

**화면 구성:**
```
[StatusHeader: Idle]
"지금은 쉬는 중이에요"

[MapContainer]
- 지도
- 내 위치 점

[StateBar: Idle]
"지금 말해도 돼요"
```

**사용자 액션:**
- 🎙️ **말한다**

---

## STEP 1. 음성 입력

**사용자 말:**
> "근처 축구장"

**시스템 반응:**
- 🔄 지도 **그대로** (변화 없음)
- 🎤 StateBar → "듣고 있어요" (Listening 상태)
- ⏳ 로딩 느낌만 (약간의 펄스 애니메이션)

**화면 구성:**
```
[StatusHeader: Idle] (변화 없음)
"지금은 쉬는 중이에요"

[MapContainer] (변화 없음)
- 지도
- 내 위치 점

[StateBar: Listening]
"지금 말해도 돼요"
"듣고 있어요" (작은 텍스트, 펄스)
```

**❌ 하지 않는 것:**
- 화면 전환 없음
- 팝업 없음
- 키보드 없음
- 입력창 없음

---

## STEP 2. 결과 표시 (핵심)

**지도 반응:**
- 📍 **축구장 핀 여러 개 표시** (3-5개)
- 📌 가장 가까운 1개 **강조** (더 큰 핀 or 색상 다름)
- 🗺️ 지도 자동 줌 (모든 핀이 보이도록)

**하단에 결과 카드 1개만:**
```
┌─────────────────────────┐
│ 의정부 ○○축구장          │
│ 도보 12분 · 무료         │
│ [길 안내] [상세 보기]    │
└─────────────────────────┘
```

**화면 구성:**
```
[StatusHeader: Idle] (변화 없음)
"지금은 쉬는 중이에요"

[MapContainer]
- 지도 (자동 줌)
- 내 위치 점
- 축구장 핀 여러 개
- 가장 가까운 핀 강조

[ResultCard] (하단 고정)
- 장소명
- 거리/시간
- 버튼 2개

[StateBar: Idle] (복귀)
"지금 말해도 돼요"
```

**❌ 하지 않는 것:**
- 리스트 ❌
- 스크롤 ❌
- 카드 여러 개 ❌
- 카드 1개 ⭕

---

## STEP 3. 사용자 선택

**사용자가 할 수 있는 액션은 딱 3개:**

### 3-1. 카드 클릭 → 상세 정보
- "상세 보기" 버튼 클릭
- 또는 카드 전체 클릭

### 3-2. "여기로 가자" 말하기 → 길찾기
- 음성으로 "여기로 가자" / "길 안내" / "가고 싶어"
- 또는 "길 안내" 버튼 클릭

### 3-3. 다른 말 하기 → 재검색
- "더 가까운 곳" / "다른 곳" 등
- 새로운 검색 시작

---

## STEP 4. 상세 정보 (선택)

**카드 클릭 시:**

**화면 구성:**
```
[StatusHeader: Idle] (변화 없음)

[MapContainer] (약간 어둡게)
- 지도
- 선택된 핀 강조

[DetailCard] (하단 확장)
- 장소명
- 주소
- 운영 시간
- 후기 요약 (AI 생성)
- [길 안내] [닫기]
```

**❌ 하지 않는 것:**
- 전체 화면 전환 ❌
- 지도 사라짐 ❌
- 모달 팝업 ❌

---

## STEP 5. 길찾기 시작 (3-2 선택 시)

**"여기로 가자" 말하거나 버튼 클릭:**

**화면 구성:**
```
[StatusHeader: Navigating]
"🏃 의정부 ○○축구장 가는 중"
"🚶 도보 · 12분 남음"

[MapContainer]
- 지도
- 경로 표시 (파란 선)
- 내 위치 점
- 목적지 핀

[ActionCue] (하단)
"⬆️ 직진 · 120m"

[StateBar: Navigating]
"계속 안내 중이에요"
```

**이후는 네비게이션 플로우로 이어짐**

---

## 🔄 플로우 다이어그램

```
[STEP 0: 초기]
  ↓ (말하기)
[STEP 1: 음성 입력]
  ↓ (결과 도착)
[STEP 2: 결과 표시]
  ↓
  ├─ (카드 클릭) → [STEP 4: 상세 정보] → (닫기) → [STEP 2]
  ├─ ("여기로 가자") → [STEP 5: 길찾기]
  └─ (다른 말) → [STEP 1: 재검색]
```

---

## ✅ 성공 기준

이 질문에 **YES** 나오면 성공:

> ❓ "키보드 없이
> 말만 해도
> 주변 운동장 / 시설 찾기 편한가?"

---

## ⚠️ 예외 처리

### 결과가 없을 때

**화면 구성:**
```
[StatusHeader: Idle] (변화 없음)

[MapContainer] (변화 없음)

[EmptyCard] (하단)
"근처 축구장을 찾지 못했어요"
"다시 말해볼까요?"

[StateBar: Idle]
"지금 말해도 돼요"
```

### 음성 인식 실패

**화면 구성:**
```
[StatusHeader: Idle] (변화 없음)

[MapContainer] (변화 없음)

[StateBar: Idle] (복귀)
"지금 말해도 돼요"
```

**❌ 에러 메시지 표시 안 함**
**✅ 자연스럽게 복귀**

---

## 🎯 핵심 원칙

1. **화면 전환 없음** - 컴포넌트만 교체
2. **카드 1개만** - 리스트 금지
3. **지도는 항상 보임** - 사라지지 않음
4. **에러 노출 안 함** - 자연스럽게 복귀
