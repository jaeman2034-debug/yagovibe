# ✅ Firebase 테스트 전화번호 삭제 가이드

## 현재 화면 확인

**Firebase Console → Authentication → 로그인 방법 → 전화번호**

화면에 보이는 것:
- "테스트용 전화번호(선택사항)" 섹션
- 전화번호 입력 필드: "+1 650-555-1234"
- 인증 코드 입력 필드: (비어있음)

---

## 🔍 확인해야 할 것

### 1. 등록된 테스트 번호 목록 확인

화면에서 **등록된 테스트 번호 목록**이 보이는지 확인하세요.

**보이는 경우:**
- 테스트 번호가 목록으로 표시됨
- 각 번호 옆에 삭제 버튼(휴지통 아이콘)이 있음

**보이지 않는 경우:**
- 입력 필드에 값이 있지만 아직 등록되지 않았을 수 있음
- 또는 이미 등록되어 있지만 목록이 보이지 않을 수 있음

---

## 🚨 즉시 해야 할 것

### 옵션 1: 등록된 테스트 번호가 목록에 보이는 경우

1. **각 테스트 번호 옆의 삭제 버튼(휴지통 아이콘) 클릭**
2. **모든 테스트 번호 삭제**
3. **"저장" 버튼 클릭**

### 옵션 2: 입력 필드에만 값이 있고 목록이 없는 경우

1. **전화번호 입력 필드의 "+1 650-555-1234" 삭제**
2. **인증 코드 입력 필드도 비우기**
3. **"저장" 버튼 클릭**

### 옵션 3: "제공업체 삭제" 버튼이 보이는 경우

⚠️ **주의:** 이 버튼은 전화번호 인증 전체를 삭제하는 것이므로 사용하지 마세요!

---

## ✅ 확인 체크리스트

- [ ] 등록된 테스트 번호 목록 확인
- [ ] 테스트 번호가 있으면 모두 삭제
- [ ] 입력 필드의 값 삭제 (등록되지 않은 경우)
- [ ] "저장" 버튼 클릭
- [ ] 다시 SMS 테스트

**반드시 모든 테스트 번호를 삭제하고 저장해야 합니다.**
