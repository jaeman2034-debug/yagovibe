# 🧪 대회 운영 자동화 QA 체크리스트 (현장 테스트용)

**목적**: 실제 대회 운영 시나리오 기준 테스트  
**환경**: 모바일 + 데스크톱 혼용

---

## 📋 Phase 1: 대회 준비 단계 (D-7 ~ D-1)

### ✅ 1. 홈 화면 진입
- [ ] 홈 > 대회 일정 섹션 표시
- [ ] 빈 상태일 때 "진행 중인 대회가 없습니다" 메시지
- [ ] [+ 대회 생성] 버튼 표시
- [ ] [기존 대회 불러오기] 버튼 표시

### ✅ 2. 대회 생성
- [ ] [+ 대회 생성] 클릭 → Drawer 열림
- [ ] 대회명 입력
- [ ] 날짜 선택 (시작일/종료일)
- [ ] 주관 입력
- [ ] 저장 버튼 클릭
- [ ] 저장 후 자동으로 운영 모드로 이동
- [ ] 생성된 대회가 홈 화면에 표시

### ✅ 3. 대회 카드 확인
- [ ] 대회명 표시
- [ ] 기간 표시
- [ ] 진행 상태 배지 표시
- [ ] 오늘 경기 수 표시 (0일 때 회색)
- [ ] 미배정 심판 수 표시 (0일 때 미표시, 1 이상일 때 빨간 점)
- [ ] 미검인 선수 수 표시 (0일 때 미표시, 1 이상일 때 빨간 점)

### ✅ 4. 경기장 등록
- [ ] 대회 생성 후 경기장 등록 단계 표시
- [ ] 경기장 선택 또는 신규 등록
- [ ] 코트 수 입력
- [ ] 저장 후 경기장별 일정표 뼈대 생성 확인

### ✅ 5. 경기 편성
- [ ] 경기 입력 UI 표시
- [ ] 시간/장소 중복 경고 표시
- [ ] 엑셀 업로드 기능 (선택)

### ✅ 6. 심판 배정
- [ ] 심판 풀 불러오기
- [ ] 심판 선택 UI
- [ ] 중복 배정 방지 (같은 시간대 같은 심판)
- [ ] 미배정 심판 수 실시간 업데이트

## 📋 Phase 2: 현장 운영 (D-day)

### ✅ 7. 운영자 현장 도착
- [ ] 홈 화면에서 대회 카드 확인
- [ ] 오늘 경기 수 정확히 표시
- [ ] 미배정 심판 수 정확히 표시
- [ ] 미검인 선수 수 정확히 표시
- [ ] 문제 있는 경기 미리 체크 가능

### ✅ 8. 전창 운영 모드 진입
- [ ] 대회 카드 클릭 → 운영 모드 페이지로 이동
- [ ] 경기장별 탭 표시
- [ ] 경기 목록 표시
- [ ] 경기 상태 표시 (대기/진행 중/종료)

### ✅ 9. 경기 상세 화면
- [ ] 경기 카드 클릭 → 상세 모달 열림
- [ ] 홈팀/원정팀 출전 명단 표시
- [ ] 검인 상태 표시
- [ ] 경고/퇴장 기록 표시
- [ ] 심판 메모 표시

### ✅ 10. 모바일 선수 검인
- [ ] [QR 스캔] 버튼 클릭 → 카메라 열림
- [ ] QR 코드 스캔
- [ ] 즉시 검인 결과 표시:
  - ✅ 검인 완료
  - ❌ 미등록
  - ❌ 자격 없음
- [ ] 검인 이력 저장 확인
- [ ] 출전 명단에 검인 상태 반영

### ✅ 11. 수동 검인
- [ ] [수동 검인] 버튼 클릭
- [ ] 선수 선택
- [ ] 검인 완료 처리
- [ ] 검인 이력 저장

### ✅ 12. 경기 시작
- [ ] [경기 시작] 버튼 클릭
- [ ] 경기 상태 "진행 중"으로 변경
- [ ] 시작 시간 자동 기록
- [ ] 심판 자동 기록

### ✅ 13. 경고/퇴장 기록
- [ ] 선수 선택
- [ ] 경고/퇴장 버튼 클릭
- [ ] 시간 입력 (선택)
- [ ] 사유 입력 (선택)
- [ ] 기록 저장
- [ ] 경기 상세에 반영

### ✅ 14. 경기 종료
- [ ] [경기 종료] 버튼 클릭
- [ ] 경기 상태 "종료"로 변경
- [ ] 종료 시간 자동 기록
- [ ] 점수 입력 (선택)
- [ ] 모든 정보 한 화면에 표시

### ✅ 15. 예외 상황 대응
- [ ] 일정표에서 다음 경기 즉시 확인
- [ ] 대기 팀/심판 안내 가능
- [ ] 경기 지연 시 일정 조정 가능

## 📊 테스트 시나리오

### 시나리오 1: 대회 생성부터 운영까지
1. 빈 상태에서 대회 생성
2. 경기장 등록
3. 경기 편성
4. 심판 배정
5. 운영 모드 진입
6. 경기 시작
7. 선수 검인
8. 경고 기록
9. 경기 종료
10. 리포트 생성

**예상 시간**: 30분  
**체크 포인트**: 모든 단계 오류 없이 진행

### 시나리오 2: 예외 상황
1. 시간 중복 경기 생성 시도 → 경고 표시
2. 같은 심판 중복 배정 시도 → 차단
3. 미등록 선수 QR 스캔 → "미등록" 표시
4. 자격 없음 선수 검인 → "자격 없음" 표시

**예상 시간**: 10분  
**체크 포인트**: 모든 예외 상황 정확히 처리

### 시나리오 3: 모바일 검인
1. 모바일에서 운영 모드 진입
2. 경기 선택
3. QR 스캔
4. 검인 완료 확인
5. 데스크톱에서 검인 상태 확인

**예상 시간**: 5분  
**체크 포인트**: 모바일-데스크톱 동기화
