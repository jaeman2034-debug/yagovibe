### 가이드라인 목표

- 공공성과 민간 서비스의 충돌 방지
- 신뢰 유지
- 장기 협업 가능
- 윤리적 운영

---

### 톤 가이드

- 명확함
- 절대적
- 신뢰감
- 공공성

---

## 서문 (고정)

```
기록장은 보여주기 위한 서비스가 아니라, 남기기 위한 서비스다.

아래 원칙은 기능보다 신뢰를 우선하기 위한 최소 규칙이다.
```

---

## 0️⃣ 가이드라인 핵심

**공공 제안과 민간 팀 서비스가 충돌하지 않도록 하는 안전장치**

**신뢰가 깨지지 않게 미리 막는 규칙**

---

## 1조. 기록 우선의 원칙

기록장은 홍보·모집·성과 강조를 목적으로 하지 않는다.

모든 콘텐츠는 사실 기반의 기록이어야 한다.

### 적용

- 홍보·모집·성과 강조 금지
- 사실 기반 기록만 허용

### 위반 시

- 해당 콘텐츠 수정 또는 삭제
- 재발 방지 조치

---

## 2조. 과장 금지의 원칙

"최고", "성공", "폭발", "급증" 등의 표현을 사용하지 않는다.

수치는 비교가 아닌 현황 표시로만 사용한다.

### 적용

- 과장 표현 사용 금지
- 수치는 현황 표시로만 사용

### 위반 시

- 해당 표현 즉시 수정
- 재발 방지 조치

---

## 3조. 자발성 보장의 원칙

팀의 사용 여부는 완전히 자율이다.

사용 중단에 따른 불이익을 두지 않는다.

### 적용

- 사용 여부 완전 자율
- 사용 중단 불이익 없음

### 위반 시

- 불이익 요소 즉시 제거
- 자발성 보장 조치

---

## 4조. 프라이버시 최소 노출 원칙

실명, 얼굴 사진, 개인 식별 정보는 기본적으로 사용하지 않는다.

공개 범위는 날짜·지역·활동 사실로 제한한다.

### 적용

- 실명·얼굴 사진·개인 식별 정보 사용 금지
- 날짜·지역·활동 사실만 공개

### 위반 시

- 개인 정보 즉시 삭제
- 재발 방지 조치

---

## 5조. 공공·민간 분리 원칙

공공(지역·커뮤니티) 페이지는 읽기 전용이다.

공공 페이지에 상업적 CTA를 삽입하지 않는다.

### 적용

- 공공 페이지 읽기 전용
- 상업적 CTA 삽입 금지

### 위반 시

- 상업적 요소 즉시 제거
- 공공성 회복 조치

---

## 6조. 비경쟁 원칙

팀 간 랭킹, 비교 지표, 우열 표현을 만들지 않는다.

기록의 빈도는 보여줄 수 있으나 순위화하지 않는다.

### 적용

- 랭킹·비교 지표·우열 표현 금지
- 빈도는 보여주되 순위화 금지

### 위반 시

- 경쟁 요소 즉시 제거
- 사실 기반 집계만 유지

---

## 7조. 침묵 존중의 원칙

알림은 안심·확인 목적으로만 사용한다.

사용자의 침묵을 문제로 규정하지 않는다.

### 적용

- 알림은 안심·확인 목적만
- 침묵을 문제로 규정하지 않음

### 위반 시

- 과도한 알림 제거
- 침묵 존중 조치

---

## 8조. 자동화 투명성 원칙

자동 생성 콘텐츠임을 속이지 않는다.

다만, AI를 전면에 내세우지 않는다.

### 적용

- 자동 생성 콘텐츠 투명성 유지
- AI 전면 노출 금지

### 위반 시

- 투명성 회복 조치
- 재발 방지 조치

---

## 9조. 확장 절제의 원칙

기능 추가는 신뢰를 높일 때만 허용한다.

편의성보다 톤과 공공성을 우선한다.

### 적용

- 기능 추가는 신뢰 향상 시만
- 편의성보다 톤·공공성 우선

### 위반 시

- 기능 추가 보류
- 신뢰 우선 조치

---

## 10조. 중단 가능성 보장의 원칙

언제든지 서비스를 중단할 수 있어야 한다.

중단 이후에도 기존 기록은 존중한다.

### 적용

- 서비스 중단 자유
- 중단 후 기록 존중

### 위반 시

- 중단 프로세스 개선
- 기록 존중 조치

---

## 운영자 행동 강령 (요약)

- 설명하지 말 것
- 설득하지 말 것
- 비교하지 말 것
- 조용히 유지할 것

**기록이 스스로 말하게 둔다.**
