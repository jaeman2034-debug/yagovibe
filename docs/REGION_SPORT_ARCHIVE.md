# 🗺️ 지역·종목 아카이브 설계 문서

## 목적

개별 팀 기록을 모아도 톤은 그대로, 의미만 커지게

- ❌ 커뮤니티 기능
- ❌ 참여 유도
- ⭕ 읽기 전용 기록 집계

---

## 1️⃣ URL 구조 (SEO + 확장성)

```
/archive/region/{region}
/archive/sport/{sport}
/archive/region/{region}/sport/{sport}
```

**예시**:
- `/archive/region/포천`
- `/archive/sport/축구`
- `/archive/region/포천/sport/축구`

---

## 2️⃣ 페이지 성격

**공개**
- 읽기 전용
- CTA 없음
- 가입/문의 버튼 없음

**핵심**: "구경만 해도 되는 페이지"

---

## 3️⃣ 상단 헤더 (톤 고정)

**제목**:
```
{region} 지역 팀 활동 기록
또는
{sport} 팀 활동 기록
```

**서브**:
```
최근 일정 기간 동안의 팀 활동을 기록으로 모았습니다.
```

**제외 사항**:
- ❌ "인기 팀"
- ❌ "활발한 지역"

---

## 4️⃣ 핵심 콘텐츠 구성

### ① 최근 기록 리스트 (시간순)

**카드 단위**:
```
[팀명]
소흘 60대 FC

[기록 요약]
7월 셋째 주 정기 모임 기록

[메타]
2025년 7월 · 축구 · 포천
```

**규칙**:
- 최신순
- 팀 구분 명확
- 클릭 시 개별 포스트 페이지로 이동

### ② 활동 팀 요약 (숫자 최소)

**페이지 하단에만 작게**:
```
지난 30일간
- 활동 기록 18건
- 참여 팀 7팀
```

**제외 사항**:
- ❌ 그래프
- ❌ 비교

---

## 5️⃣ 쿼리 설계 (Firestore 기준)

### 지역 아카이브
```typescript
where("region", "==", region)
orderBy("createdAt", "desc")
limit(30)
```

### 종목 아카이브
```typescript
where("sport", "==", sport)
orderBy("createdAt", "desc")
limit(30)
```

### 지역 + 종목
```typescript
where("region", "==", region)
where("sport", "==", sport)
orderBy("createdAt", "desc")
limit(30)
```

**📌 인덱스만 추가, 로직 변경 없음**
