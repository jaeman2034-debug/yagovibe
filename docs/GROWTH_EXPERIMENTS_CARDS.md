# 🚀 그로스 실험 카드 20개 (실행판)

**목표**: 전환율 34% 달성, 노쇼율 4.5% 이하  
**실행 기간**: 14일 스프린트  
**형식**: 실험 카드 (측정 방법 + 중단 조건 포함)

---

## 📋 실험 카드 구조

각 실험 카드는 다음 정보를 포함:
- **실험 ID**: 고유 식별자
- **가설**: 명확한 가설
- **변수**: Control vs Variant
- **측정 방법**: 지표 및 데이터 수집
- **중단 조건**: 실패 시 즉시 중단 기준
- **예상 결과**: 승리 Variant 예측

### 실험 카드 #1: 사진 우선 온보딩

**실험 ID**: `EXP-ONBOARD-001  
**가설**: 첫 화면에서 사진 업로드를 강제하면 등록 완료율이 15% 증가한다.

**변수**:
- **Control**: 기존 플로우 (제목 → 가격 → 사진)
- **Variant A**: 사진 → AI 제목 추천 → 가격 가이드
- **Variant B**: 사진 → 가격 가이드 → AI 제목 추천

**측정 방법**:
- **주 지표**: 등록 완료율 (목표: +15%)
- **부 지표**: 이미지 포함률, 등록 소요 시간
- **샘플 크기**: 최소 1,000명/변수 (총 3,000명)
- **실행 기간**: 3일
- **데이터 수집**: Firestore `userEvents` 컬렉션

**중단 조건**:
- 등록 완료율이 Control 대비 -10% 이상 감소
- 이미지 포함률이 90% 미만
- 실험 시작 24시간 내 명확한 부정적 신호
