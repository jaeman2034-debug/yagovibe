# ✍️ AI 블로그 생성 프롬프트 실전 튜닝 가이드

## 목적

홍보·과장·AI 티 완전 제거 → "사람이 담담하게 쓴 팀 기록"처럼 보이게

---

## 0️⃣ 이 프롬프트의 철학

**설명하지 말고, 있었던 일을 정리한다**

- ❌ 잘 보이게
- ❌ 매력적으로
- ❌ 끌어들이기
- ⭕ 사실·맥락·날짜

---

## 1️⃣ 시스템 프롬프트 (고정 · 절대 수정 금지)

```
당신은 홍보 글을 쓰지 않습니다.
당신은 팀의 활동을 '기록'하는 역할입니다.

과장, 감탄사, 마케팅 표현을 사용하지 마세요.
사실과 맥락 위주로 담담하게 작성하세요.

이 글의 목적은 설득이 아니라 보관입니다.
```

---

## 2️⃣ 입력 데이터 포맷

```json
{
  "teamName": "소흘 60대 FC",
  "region": "포천",
  "ageGroup": "60대",
  "sport": "축구",
  "frequency": "주 1회",
  "vibe": "즐겁게",
  "postType": "intro"
}
```

**📌 주의**:
- 없는 정보는 절대 만들어내지 말 것
- 입력 안 준 필드는 언급하지 말 것

---

## 3️⃣ 글 생성 프롬프트 — 팀 소개(Intro)

```
아래 팀 정보를 바탕으로 팀 소개 글을 작성하세요.

규칙:
- 4~6문단
- 각 문단 2~3문장
- 감탄사 사용 금지
- 홍보 문장 금지
- "함께하세요", "많은 관심" 같은 표현 금지

구성:
1문단: 팀이 언제, 어떤 맥락으로 모이는지
2문단: 활동 방식과 분위기
3문단: 팀의 연령대와 운동 목적
4문단: 꾸준히 이어지고 있다는 사실

마무리는 조용하게 끝내세요.
```

---

## 4️⃣ 예상 결과 예시

```
소흘 60대 FC는 포천 지역에서 주 1회 모여 축구를 하는 팀입니다.
무리하지 않고 각자의 페이스에 맞춰 운동을 이어가고 있습니다.

팀은 즐겁게 운동하는 분위기를 중요하게 생각합니다.
경기보다는 함께 움직이고 몸을 푸는 시간에 더 집중합니다.

주 구성원은 60대이며, 건강을 유지하고 일상에 활력을 더하는 것을 목표로 합니다.
운동 경험의 많고 적음보다는 꾸준함을 우선합니다.

소흘 60대 FC의 활동은 특별한 이벤트보다 정기적인 모임으로 이어지고 있습니다.
이런 방식으로 팀의 기록도 차분히 쌓이고 있습니다.
```

**👉 이 느낌 아니면 프롬프트 실패**

---

## 5️⃣ 생성 후 자동 필터 (필수)

### ❌ 있으면 즉시 재생성

- "열정", "최고", "활기찬"
- "많은 분들의 관심"
- 느낌표 `!`

### ⭕ 있으면 통과

- 날짜
- 빈도
- "꾸준히", "무리하지 않고"

---

## 6️⃣ 프론트 저장 전 마지막 체크

```typescript
if (
  content.includes("함께") ||
  content.includes("관심") ||
  content.includes("환영")
) {
  throw new Error("PROMO_TONE_DETECTED");
}
```

**👉 한 번 걸러주면 끝까지 안전**

---

## 7️⃣ 이 프롬프트의 강점

- ✅ AI 냄새 거의 0
- ✅ 홍보 오해 없음
- ✅ 지자체/외부 공개 가능
- ✅ SEO에도 자연스럽게 먹힘

**핵심 인식**: "이 팀은 계속 활동하는구나"

## 8️⃣ 구현 위치
