**생성일**: 2025-01-27

**핵심 원칙**: 설명 최소화, 화면 중심, 결과 증명

## 🎯 추천 대상: 사무국장

### 왜 사무국장인가?

1. **실제 운영 담당자**
   - 일상 업무를 직접 수행
   - 현재 시스템의 문제점을 가장 잘 알고 있음
   - 이 화면의 가치를 즉시 이해할 수 있음

2. **회장 보고 담당자**
   - 회장에게 보고할 때 이 화면을 사용 가능
   - "이 화면 하나로 보고 끝"이라는 가치 제공
   - 사무국장이 먼저 이해하면 회장 설득도 쉬움

3. **숫자에 민감함**
   - 회원 수, 회비 납부율 등 숫자 관리 담당
   - 이 화면이 얼마나 시간을 절약해주는지 즉시 체감

## 💬 사무국장 미팅 시나리오

### 미팅 전 준비

**준비물:**
1. 와이어프레임 화면 (데이터 시뮬레이션 포함)
2. 노트북 또는 태블릿 (화면 보여주기용)
3. 인쇄본 1부 (선택사항)

**시간:**
- 15분 이내 (짧을수록 좋음)

**장소:**
- 협회 사무실 (사무국장이 편안한 곳)

---

### 미팅 진행 (15분)

#### 1단계: 인사 및 간단한 소개 (2분)

**말할 내용:**
```
"안녕하세요. 노원구축구협회 사무국장님, 
오늘 시간 내주셔서 감사합니다.

저희는 축구협회 행정 업무를 도와드리는 
서비스를 만들고 있습니다.

오늘은 15분만 시간 내주시면 됩니다."
```

**하지 말 것:**
- ❌ "플랫폼", "시스템" 같은 단어 사용
- ❌ 긴 설명
- ❌ 기술 설명

---

#### 2단계: 화면 보여주기 (5분)

**액션:**
1. 노트북/태블릿 화면을 사무국장 앞에 둠
2. 와이어프레임 화면을 보여줌
3. **아무 말도 하지 않음** (5초 기다림)

**기대 반응:**
- 사무국장이 화면을 보며 숫자를 확인
- "이게 뭐예요?" 또는 "이건 우리 협회 얘기인가요?" 질문

---

#### 3단계: 핵심 한 문장 (30초)

**사무국장이 질문하면:**

**질문 1: "이게 뭐예요?"**
```
"이 화면이 지금까지 협회에서 
사람이 하던 일을 대신 정리해주고 있습니다."
```

**질문 2: "이건 우리 협회 얘기인가요?"**
```
"네, 노원구축구협회 데이터를 
이렇게 정리해서 보여주는 화면입니다."
```

**질문 3: "이게 자동으로 만들어지는 건가요?"**
```
"네, 매일 자동으로 업데이트됩니다. 
별도로 입력하실 필요 없습니다."
```

---

#### 4단계: 핵심 지표 설명 (3분)

**사무국장이 관심 보이면:**

**카드 1: 산하 클럽**
```
"산하 클럽 20개 중 활동 중인 게 16개, 
비활동이 4개인 걸 한눈에 볼 수 있습니다."
```

**카드 2: 등록 회원**
```
"회원 수 2,134명을 성인, 실버, 유소년으로 
나눠서 볼 수 있고, 작년 대비 5% 증가한 것도 
바로 확인할 수 있습니다."
```

**카드 3: 회비 납부**
```
"회비 납부율 78%를 목표 90%와 비교해서 
보고, 미납 22%가 어디서 나오는지도 
바로 확인할 수 있습니다."
```

**카드 4: 올해 대회**
```
"올해 대회 6개 중 완료된 게 5개, 
참가율 88%를 한눈에 볼 수 있습니다."
```

**AI 요약:**
```
"이 요약 문장은 매일 자동으로 만들어집니다. 
작년 대비, 목표 대비 비교도 자동으로 들어갑니다."
```

---

#### 5단계: 시간 절약 강조 (2분)

**말할 내용:**
```
"이전에는 회원 수를 확인하려면 
여러 곳에서 모아야 했고, 
회비 납부율을 계산하려면 
엑셀에서 수작업으로 해야 했습니다.

이제는 이 화면 하나로 
모든 정보를 바로 볼 수 있습니다."
```

**하지 말 것:**
- ❌ "기능이 많습니다" 같은 설명
- ❌ "다른 협회도 사용합니다" 같은 홍보

---

#### 6단계: 다음 단계 제안 (2분)

**사무국장이 관심 보이면:**

**제안 1: 회장 보고용**
```
"이 화면을 회장님께 보고하실 때 
사용하시면 됩니다. 
15분 설명 대신 화면 하나로 끝납니다."
```

**제안 2: 시범 운영**
```
"한 달 정도 시범으로 운영해보시고, 
마음에 드시면 계속 사용하시면 됩니다."
```

**제안 3: 추가 기능**
```
"필요하시면 클럽별 상세 정보나 
회비 납부 내역도 볼 수 있습니다."
```

---

#### 7단계: 마무리 (30초)

**말할 내용:**
```
"오늘 시간 내주셔서 감사합니다. 
추가로 궁금한 점 있으시면 
언제든 연락 주세요."
```

**하지 말 것:**
- ❌ "계약하시겠습니까?" 같은 압박
- ❌ "다른 협회는 이미 사용 중입니다" 같은 비교

예를 들어, 회비 납부율이 78%인데 
목표가 90%라면, 12%p 부족한 걸 
바로 알 수 있습니다.

바로 확인할 수 있습니다."

이제는 이 화면 하나로 보고가 끝납니다.

회장님께서도 보고서를 읽는 시간 대신

## 🎯 최종 추천: 사무국장 → 회장 순서

1. **사무국장 먼저**
   - 실제 운영 담당자
   - 화면의 가치를 즉시 이해
   - 회장 보고 시 이 화면 사용 가능

2. **회장 다음**
   - 사무국장이 이미 이해한 상태
   - 회장 보고 시 사무국장이 화면 사용
