## 2️⃣ 컴포넌트별 상세 스펙

### ① HeroActivityCard (히어로 영역)

**위치:** Above the Fold (스크롤 없이 보임)

**구조:**
```
┌─────────────────────────────────┐
│  [팀 아이콘]  상태 배지          │
│              팀명                │
│              캐치프레이즈         │
│              최근 활동 요약       │
│              신뢰 배지            │
│              CTA 버튼            │
└─────────────────────────────────┘
```

**Props:**
```typescript
interface HeroActivityCardProps {
  teamName: string;
  tagline: string;
  status: "active" | "recruiting" | "inactive";
  recentActivity: {
    type: "schedule" | "attendance" | "blog" | "none";
    date?: string;
    event?: string;
    count?: number;
    title?: string;
  };
  badges: string[];
  onJoinClick: () => void;
  onViewIntroClick: () => void;
  joinButtonState: "default" | "requested" | "member" | "admin";
}
```

**반응형:**
- 모바일: 세로 스택, 아이콘 크기 축소
- 태블릿: 가로 배치, 아이콘 중간 크기
- 데스크톱: 가로 배치, 아이콘 큰 크기

## 6️⃣ 스타일 가이드

### 색상

- Primary CTA: `bg-blue-600` / `hover:bg-blue-700`
- Secondary CTA: `bg-gray-100` / `hover:bg-gray-200`
- 상태 배지:
  - 활발한 팀: `bg-green-100` / `text-green-700`
  - 모집 중: `bg-yellow-100` / `text-yellow-700`
  - 준비 중: `bg-gray-100` / `text-gray-700`

### 타이포그래피

- 히어로 제목: `text-2xl sm:text-3xl font-bold`
- 섹션 제목: `text-xl font-semibold`
- 본문: `text-base text-gray-700`
- 작은 텍스트: `text-sm text-gray-500`

### 간격

- 섹션 간격: `mb-6 lg:mb-8`
- 카드 내부 패딩: `p-6 lg:p-8`
- 버튼 간격: `gap-3`
