# 🔥 YAGO VIBE 베타 테스트 가이드

## 📋 개요

YAGO VIBE는 선수 퍼포먼스 관리 플랫폼입니다.
이 가이드는 베타 테스트 참여자를 위한 사용 설명서입니다.

---

## 🎯 베타 테스트 목표

**3일 동안 실제 사용 데이터 확보**

- 실제 선수 5~20명이 앱을 사용
- 일일 컨디션 기록
- 루틴 체크
- 활동 기록
- 코치 리포트 생성

---

## 📱 앱 설치 및 로그인

### 1. 앱 접속
- 웹 브라우저에서 제공된 링크로 접속
- 모바일 최적화되어 있으므로 모바일에서 사용 권장

### 2. 로그인
- 이메일 또는 구글 계정으로 로그인
- 첫 로그인 시 프로필 설정 필요

### 3. 프로필 설정
- 종목 선택 (예: 축구, 농구)
- 활동 지역 선택
- 간단 소개 입력 (선택)

---

## 🏃‍♂️ 선수용 가이드

### Day 1: 첫 사용

#### 1. 성장 탭 접속
- 하단 메뉴에서 "성장" 탭 선택
- 첫 사용 시 온보딩 가이드 표시

#### 2. 컨디션 입력
- 오늘 컨디션 카드에서 "입력하기" 클릭
- 수면 시간 입력 (슬라이더)
- 피로도 선택 (1~5)
- 통증 수준 선택 (0~10)
- 기분 선택 (1~5)
- "저장" 클릭

#### 3. 루틴 체크
- 오늘 루틴 카드에서 체크박스 선택
- 스트레칭, 보강운동, 영상 분석, 멘탈 트레이닝
- 체크 후 자동 저장

#### 4. 목표 설정
- 목표 카드에서 "목표 설정" 클릭
- 이번 달 목표 입력 (예: 훈련 20회)
- 저장

#### 5. 활동 시작
- 허브에서 스포츠 선택
- "응, 시작할게요" 클릭
- 활동 종료 시 "종료" 클릭

---

### Day 2: 일일 기록

#### 1. 컨디션 입력
- 매일 아침 또는 저녁에 컨디션 입력
- 리듬 점수 자동 계산 확인

#### 2. 루틴 체크
- 오늘 완료한 루틴 체크
- 연속 기록(streak) 확인

#### 3. 활동 기록
- 오늘 활동 시작 및 종료
- 오늘 활동 카드에서 기록 확인

---

### Day 3: 데이터 확인

#### 1. 리듬 그래프 확인
- 성장 탭에서 7일 리듬 그래프 확인
- 추세 분석 확인

#### 2. 통계 확인
- 오늘 활동 시간
- 이번 주 운동 횟수
- 이번 주 총 시간

#### 3. AI 추천 확인
- 코치 액션 카드에서 오늘 추천 확인
- 훈련 적기/회복 권장 등

---

## 👨‍🏫 코치용 가이드

### 1. 코치 대시보드 접속
- 코치 대시보드 메뉴 선택
- 팀 선수 목록 확인

### 2. 선수 상태 확인
- 각 선수 카드에서 리듬 점수 확인
- 위험 선수 필터 사용
- 컨디션 우수 선수 확인

### 3. 경기 전 리포트 생성
- "경기 전 리포트 생성" 버튼 클릭
- 리포트 모달에서 내용 확인
- 다운로드 또는 인쇄

## 📊 베타 성공 기준

다음 3개를 충족하면 성공:

1. **하루 1회 이상 앱 열림**
   - 3일 중 최소 2일 이상 접속

2. **루틴 체크 50% 이상**
   - 3일 중 최소 2일 이상 루틴 체크

3. **코치 리포트 사용됨** (코치만)
   - 경기 전 리포트 최소 1회 생성

---

## 💡 피드백 제공

베타 테스트 중 발견한 문제나 개선 사항:

1. **버그 리포트**
   - 어떤 화면에서 발생했는지
   - 어떤 동작을 했을 때 발생했는지
   - 스크린샷 첨부 (가능하면)

2. **UX 개선 제안**
   - 사용하기 어려웠던 부분
   - 개선하면 좋을 부분

3. **기능 제안**
   - 추가되면 좋을 기능
   - 더 유용하게 만들 수 있는 방법
