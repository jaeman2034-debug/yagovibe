# 🎯 첫 팀 맞춤 프롬프트 생성기

## 📌 사용법

**첫 팀 정보를 알려주면, 아래 3가지를 자동으로 생성합니다:**

1. **첫 팀 소개 글 프롬프트** (AI 생성용)
2. **활동 후기 예시** (실제 생성될 문구)
3. **주간 요약 예시** (실제 생성될 문구)

---

## 📋 필요한 정보

다음 정보만 알려주세요:

1. **팀 이름**: (예: 소흘 60대 FC)
2. **종목**: (예: 축구, 풋살, 당구, 배드민턴 등)
3. **지역**: (예: 포천시, 서울 강남구 등)
4. **연령대**: (예: 60대, 50대, 40대 등)
5. **멤버 수**: (예: 12명, 20명 등)
6. **분위기 키워드**: (예: 건강, 즐거움, 친목, 열정 등)

---

## 🎯 생성될 내용

### 1. 첫 팀 소개 글 프롬프트

**입력 정보를 바탕으로:**
- SEO 친화적 제목 생성
- 3~5문단 소개글 생성
- "이런 분께 추천" 불릿 생성
- 하단 참여 CTA 생성

**톤:**
- 과장 금지
- 진정성
- 지역/사람 중심

---

### 2. 활동 후기 예시

**실제 생성될 문구:**
- 활동 날짜/장소
- 참여 인원
- 활동 내용 요약
- 팀 분위기 묘사
- 하단 참여 CTA

---

### 3. 주간 요약 예시

**실제 생성될 문구:**
- 이번 주 모임 횟수
- 출석 상황
- 주요 활동 하이라이트
- 다음 주 예고

---

## 📝 예시 (소흘 60대 FC 기준)

### 첫 팀 소개 글

**제목:**
```
소흘 60대 FC는 어떤 팀인가요?
```

**본문:**
```html
<h2>안녕하세요, 소흘 60대 FC입니다!</h2>

<p>저희 소흘 60대 FC는 포천 지역에서 활동하는 60대 축구 동호회입니다. 
2023년에 창단하여 현재 12명의 열정적인 멤버들이 함께 땀 흘리고 있습니다. 
저희는 단순히 축구를 잘하는 것을 넘어, 함께 오래 즐길 수 있는 건강한 운동 문화를 만들어가는 것을 목표로 합니다.</p>

<p><strong>📍 활동 지역:</strong> 포천시 생활체육공원 및 인근 구장</p>
<p><strong>⚽ 종목:</strong> 축구</p>
<p><strong>👥 멤버 구성:</strong> 60대 중심 (50대 후반도 환영합니다!)</p>
<p><strong>🏃 운동 강도:</strong> 초급–중급 (무리하지 않는 선에서 즐겁게!)</p>

<p>저희 팀은 정기적인 운동 일정과 높은 출석률을 자랑하며, 
새로 오시는 분들도 부담 없이 어울릴 수 있도록 항상 따뜻한 분위기를 유지하고 있습니다. 
경쟁보다는 팀원 간의 유대감과 즐거움을 최우선으로 생각합니다.</p>

<h2>이런 분께 추천합니다!</h2>
<ul>
  <li>60대 이후에도 건강하게 축구를 즐기고 싶은 분</li>
  <li>경쟁보다는 즐거움과 친목을 중요하게 생각하는 팀을 찾는 분</li>
  <li>포천 지역에서 꾸준하고 안정적인 운동 모임을 원하는 분</li>
  <li>새로운 사람들과 어울리며 활기찬 노년을 보내고 싶은 분</li>
</ul>

<p><strong>📣</strong> 현재 소흘 60대 FC는 새로운 멤버를 모집 중입니다. 
관심 있으시다면 아래 버튼을 통해 언제든지 참여를 신청해 주세요!</p>
```

### 활동 후기 예시

**제목:**
```
오늘도 함께 땀 흘린 소흘 60대 FC ⚽
```

**본문:**
```html
<h2>2024년 1월 15일 소흘 60대 FC의 즐거운 활동 후기!</h2>

<p>오늘은 포천시 생활체육공원에서 소흘 60대 FC의 정기 축구 모임이 있었습니다. 
맑은 날씨 속에서 10명의 멤버들이 함께 땀 흘리며 즐거운 시간을 보냈습니다.</p>

<p><strong>👥 참여 인원:</strong> 10명 / 전체 12명</p>
<p><strong>☀️ 날씨:</strong> 맑음</p>
<p><strong>⚽ 활동 내용:</strong> 가벼운 워밍업 후 미니 게임 진행, 전술 훈련 등</p>

<p>무리하지 않으면서도 충분히 몸을 풀 수 있는 시간이었고, 
오랜만에 참여한 멤버도 자연스럽게 어울릴 수 있었습니다. 
팀원들의 열정과 웃음이 가득한 하루였습니다.</p>

<p>저희 소흘 60대 FC는 꾸준함과 즐거움을 가장 중요하게 생각합니다. 
함께 운동하며 건강한 일상을 이어가고 싶으신 분들은 언제든 환영합니다!</p>
```

### 주간 요약 예시

**제목:**
```
이번 주 소흘 60대 FC는 이렇게 활동했습니다
```

**본문:**
```html
<h2>2024년 1월 둘째 주 활동 요약</h2>

<p>이번 주 소흘 60대 FC는 총 <strong>2회의 정기 모임</strong>을 가졌습니다.</p>

<p><strong>📊 이번 주 통계:</strong></p>
<ul>
  <li>모임 횟수: 2회</li>
  <li>평균 출석률: 83% (10명/12명)</li>
  <li>주요 활동: 정기 축구 모임, 전술 훈련</li>
</ul>

<p><strong>🌟 이번 주 하이라이트:</strong></p>
<p>1월 15일 모임에서는 오랜만에 참여한 멤버도 자연스럽게 어울릴 수 있었고, 
팀원들의 열정과 웃음이 가득한 시간이었습니다.</p>

<p><strong>📅 다음 주 예고:</strong></p>
<p>다음 주에도 정기 모임이 예정되어 있습니다. 
함께 운동하며 건강한 일상을 이어가요!</p>
```
