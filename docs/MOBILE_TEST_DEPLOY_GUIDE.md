# 🚀 모바일 테스트 → 배포 실행 가이드 (FINAL)

## 목표

QR 기능은 DONE 상태. 망설일 이유 없이 바로 모바일 테스트 → 배포 진행.

---

## 1️⃣ 모바일 실전 테스트 (배포 전 10분 컷)

### ✅ 테스트 환경

- **A폰**: QR 보여주는 사람
- **B폰**: QR 스캔하는 사람
- (아이폰 + 안드로이드면 베스트, 아니어도 OK)

---

### 🔹 Step 1. A폰에서 QR 열기

**절차:**
1. 모바일 브라우저로 서비스 접속
2. ⭐ YAGO SPORTS 우측 QR 아이콘 터치
3. 풀스크린 QR 모달 표시 확인

**체크:**
- [ ] QR 크기 충분한가 (240px~300px)
- [ ] 배경 흰색인가
- [ ] 닫기(X) 잘 눌리는가 (터치 영역 44px)
- [ ] 안내 문구 표시되는가

---

### 🔹 Step 2. B폰으로 스캔

**절차:**
1. 카메라 / QR 스캐너로 QR 인식
2. URL 열림
3. 화면 확인

**체크 (중요):**
- [ ] ❌ 로그인 화면 안 뜸
- [ ] ❌ 회원가입 요구 안 뜸
- [ ] ⭕ 바로 YAGO SPORTS 허브/마켓 화면
- [ ] ⭕ 게스트로 둘러보기 가능

👉 **이거 되면 QR 기능 합격**

---

### 🔹 Step 3. 캡쳐 공유 테스트 (현실 시나리오)

**절차:**
1. A폰에서 QR 화면 캡쳐
2. 카톡/메신저로 B폰에 전송
3. B폰에서 이미지 스캔

**체크:**
- [ ] 결과 동일하게 게스트 둘러보기 진입
- [ ] 로그인/가입 요구 없음

---

## 2️⃣ 배포 전 필수 체크 (5분)

아래 4개만 확인하면 된다.

### 체크리스트

- [ ] `/qr` 경로에 auth guard 없음
- [ ] QR URL이 `/qr?market=home`
- [ ] 모바일에서 QR 모달 잘 열림
- [ ] 스캔 후 첫 화면이 "보기 가능한 화면"

**✔ 전부 YES → 바로 배포**

---

## 3️⃣ 배포 전략 (가볍게 가자)

### 🔥 권장

**바로 프로덕션 배포**

- "QR 기능 추가" 같은 공지 ❌
- → **조용히 배포 (자연 확산용)**

### 이 기능은

- 설명하면 약해지고
- 써보면 바로 이해된다

---

## 4️⃣ 배포 후 바로 할 것 (관찰 포인트)

처음 24시간 동안 이것만 본다:

### 관찰 항목

1. **QR로 들어온 사용자**
   - 수치 없어도 OK
   - 행동만 봐도 성공 여부 바로 느껴짐

2. **체류 시간**
   - 얼마나 머무는가
   - 바로 나가는가

3. **스크롤 깊이**
   - 허브 화면을 얼마나 탐색하는가
   - 카테고리까지 내려가는가

4. **카테고리 클릭 여부**
   - 실제로 종목을 선택하는가
   - 관심 있는 스포츠를 찾는가

## 5️⃣ 배포 전 코드 체크 (자동화)

### `/qr` 경로 확인

```bash
# App.tsx 또는 라우팅 파일에서 확인
grep -r "/qr" src/App.tsx
```

**확인 사항:**
- [ ] `/qr` 경로가 `PublicRoute`로 감싸져 있음
- [ ] `auth` 체크 없음
- [ ] 게스트 접근 허용

### QR URL 확인

```typescript
// QROverlay.tsx 또는 HubQRButton.tsx
const qrUrl = `${window.location.origin}/qr?market=home`;
```

**확인 사항:**
- [ ] URL이 `/qr?market=home` 형식
- [ ] `market=home` 파라미터 포함

## 6️⃣ 모바일 테스트 체크리스트 (실행용)

### A폰 (QR 보여주기)

- [ ] 모바일 브라우저 접속
- [ ] 허브 페이지 로드
- [ ] QR 아이콘 표시 확인
- [ ] QR 아이콘 터치
- [ ] 풀스크린 QR 모달 표시
- [ ] QR 크기 확인 (280px)
- [ ] 닫기 버튼 작동 확인

### B폰 (QR 스캔)

- [ ] 카메라 앱으로 QR 스캔
- [ ] URL 자동 열림
- [ ] 로그인 화면 안 뜸
- [ ] 회원가입 요구 안 뜸
- [ ] 허브/마켓 화면 바로 표시
- [ ] 게스트로 둘러보기 가능

### 캡쳐 공유 테스트

- [ ] A폰에서 QR 화면 캡쳐
- [ ] 메신저로 B폰에 전송
- [ ] B폰에서 이미지 스캔
- [ ] 동일하게 게스트 진입

## 7️⃣ 배포 후 24시간 관찰 포인트

### 수치 없어도 OK

**행동만 봐도 성공 여부 바로 느껴짐:**

1. **QR 진입 후 바로 나가는가?**
   - YES → 첫 화면이 막힘
   - NO → 계속 탐색 중

2. **카테고리까지 스크롤하는가?**
   - YES → 관심 있음
   - NO → 첫 화면에서 막힘

3. **종목을 클릭하는가?**
   - YES → 실제 사용 의도
   - NO → 탐색만 하고 나감
