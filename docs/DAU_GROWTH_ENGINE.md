## 3대 DAU 장치

### 1️⃣ 동네 핫딜 피드 (메인 홈 핵심)

**개념:**
"지금 우리 동네에서 사람들이 많이 보고 있는 물건"

**왜 터지냐:**
- 검색 ❌
- 목적 ❌
- 구경 본능 ✔

**Firestore 구조:**
```typescript
marketProducts {
  viewCount: number,      // 조회수
  favoriteCount: number,  // 찜하기 수
  createdAt: Timestamp
}
```

**핫딜 점수 공식:**
```typescript
hotScore = 
  (viewCount * 1) +
  (favoriteCount * 3) +
  recencyWeight

// 최신성 가중치
if (hours < 6) recencyWeight = 30;
else if (hours < 24) recencyWeight = 15;
else recencyWeight = 5;
```

**UX 위치:**
```
[홈]
🔥 지금 동네 핫딜
- 헬스 벤치 (1.5km)
- 러닝화 270 (급처)
```

**효과:**
👉 들어오자마자 볼 이유 생김

---

### 2️⃣ 오늘의 거래 (실시간 신뢰 + 중독성)

**개념:**
"오늘 실제로 거래된 것"

**이거 하나로 앱 신뢰도가 달라진다**

**데이터 구조:**
```typescript
marketProducts {
  status: "available" | "sold",
  soldAt?: Timestamp
}
```

**홈 노출:**
```
✅ 오늘 거래 완료
- 러닝화 260 (동네 직거래)
- 자전거 MTB
```

**사용자 뇌 반응:**
"여기서 진짜 거래되네?"

**DAU 효과:**
- 판매자: 노출 기대
- 구매자: 기회 놓칠까 봄
- 구경꾼: 하루 1회 접속

---

### 3️⃣ 카테고리 급상승 (체육인 감성 폭발)

**개념:**
"요즘 이 운동 장비 뜬다"

**계산 방식 (서버 1시간마다):**
```typescript
categoryTrendScore =
  (최근 24시간 등록 수 * 2) +
  (조회 증가율)
```

**UI:**
```
📈 요즘 뜨는 운동
1. 러닝
2. 헬스
3. 자전거
```

**효과:**
- 눌러보게 됨
- 검색 안 해도 탐색
