# 💰 수익화 설계 (UX 안 깨고, 구조 안 흔들고)

## 철학

**"검색·알림의 '결과'를 팔지 말고, '우선권'을 판다."**

### 금지 원칙
- ❌ 검색 결과 숨기기
- ❌ 유료 아니면 못 보게 하기
- ❌ 광고처럼 보이게 하기

### 허용 원칙
- ✅ 같은 결과, 다른 순서
- ✅ 같은 알림, 다른 타이밍
- ✅ 같은 시스템, 다른 가중치

## 수익화 포인트

### 1️⃣ 프리미엄 알림 우선권 (최강)

**구분** | **무료** | **프리미엄**
--- | --- | ---
알림 도착 | 3~10분 지연 | 즉시
하루 알림 수 | 제한 (5개) | 무제한
조건 세분화 | ❌ | ✅

**구현:**
```typescript
if (user.isPremium) {
  sendPushImmediately();
} else {
  schedulePush(5 * 60); // 5분 지연
}
```

**효과:**
- 직거래 앱에서 "시간 = 돈"
- 체감 차이 큼
- 비용 거의 없음

### 2️⃣ 검색 결과 노출 가중치 (자연스러운 과금)

**점수 보정:**
```typescript
if (product.isBoosted) score += 15;
```

**원칙:**
- 1~2개만 위로
- 절대 상단 고정 ❌
- "광고" 표시 ❌

**효과:**
- 사용자는 "이건 그냥 잘 맞는 결과네?" 느낌
- 자연스러운 과금

### 3️⃣ 조건 저장 슬롯 제한 (은근 강력)

**구분** | **무료** | **프리미엄**
--- | --- | ---
저장 검색 | 2개 | 10개
반경 설정 | 고정 (5km) | 자유
키워드 수 | 제한 (5개) | 확장 (10개)

**구현:**
```typescript
if (savedSearchCount >= limit) {
  showUpgradeModal();
}
```

**효과:**
- 강요 아님
- 필요할 때만 결제 유도
- 거부감 낮음

### 4️⃣ 핫딜 순간 부스트 (이벤트형 과금)

**판매자 버튼:**
```
🔥 24시간 노출 강화 (₩2,900)
```

**효과:**
- 검색 점수 +20
- 알림 우선 매칭
- "핫딜" 배지

**특징:**
- 소액
- 충동 결제
- 거래 성사율 ↑
