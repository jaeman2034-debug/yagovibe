# 🏆 첫 팀 레퍼런스 페이지 문구

## 📌 사용 목적

- 초기 10팀 확보 시 "레퍼런스"로 활용
- 다른 팀에게 보여줄 "성공 사례" 페이지
- 신뢰 구축 + 전환율 향상

---

## 🎯 레퍼런스 페이지 구조

### 1. 헤더 (신뢰 구축)

```text
실제 사용 중인 팀들의 후기

초기 10팀 중 7팀이 "이거 없으면 다시 귀찮겠다"고 말했습니다.
```

---

### 2. 팀별 후기 카드 (3~5개)

#### 카드 템플릿

```text
[팀명] | [종목] | [연령대]

"{{운영자 한 줄 후기}}"

- 사용 기간: {{기간}}
- 자동 생성 글: {{개수}}개
- 블로그 조회수: {{조회수}}회
```

---

### 3. 실제 후기 예시 (복붙용)

#### 예시 1: 60대 축구 팀

```text
소흘 60대 FC | 축구 | 60대

"경기 끝나고 후기 쓰는 게 제일 귀찮았는데,
이제는 출석만 체크하면 AI가 알아서 써주네요."

- 사용 기간: 2주
- 자동 생성 글: 5개
- 블로그 조회수: 127회
```

---

#### 예시 2: 50대 당구 동호회

```text
포천 당구 클럽 | 당구 | 50대

"팀 소개 글을 매번 새로 쓰는 게 번거로웠는데,
이제는 자동으로 관리되니까 정말 편해요."

- 사용 기간: 2주
- 자동 생성 글: 4개
- 블로그 조회수: 89회
```

---

#### 예시 3: 40대 풋살 팀

```text
청평 풋살 모임 | 풋살 | 40대

"주간 요약이 자동으로 나오니까,
이번 주 우리 팀이 뭐했는지 한눈에 보여요."

- 사용 기간: 2주
- 자동 생성 글: 6개
- 블로그 조회수: 203회
```

---

### 4. 통계 섹션 (신뢰 구축)

```text
초기 10팀 체험 결과

✅ 7팀: "이거 없으면 다시 귀찮겠다"
✅ 3팀: "이건 계속 쓰겠다" (Pro 전환)
✅ 평균 자동 생성 글: 4.5개/팀
✅ 평균 블로그 조회수: 142회/팀
```

---

### 5. CTA 섹션 (전환 유도)

```text
당신의 팀도 이렇게 자동으로 관리할 수 있습니다.

14일 무료 Pro 체험 시작하기
(카드 등록 없이, 언제든 해지 가능)
```

---

## 📋 레퍼런스 페이지 운영 원칙

1. **과장 금지** → 실제 데이터만 사용
2. **다양성** → 종목/연령대/지역 다양하게
3. **구체성** → 숫자/기간/결과 명시
4. **신뢰성** → 실제 팀명/운영자 이름 (동의 시)
