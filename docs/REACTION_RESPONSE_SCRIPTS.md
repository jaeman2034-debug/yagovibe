# 🎯 실전 반응별 답변 스크립트 (거절/보류/즉결)

## 📌 사용 원칙

- **설득하지 않는다** → 선택권은 항상 상대에게
- **압박하지 않는다** → "나중에라도" 프레임 유지
- **공감한다** → "그럴 수 있다" 먼저 말하기

---

## 1️⃣ 거절 반응별 대응

### ❌ "지금은 그냥 무료만 써볼게요"

**정답:**
```text
네, 그게 제일 좋은 접근인 것 같아요.

지금 단계에서는
"설명 듣고 바로 결제"보다는,

그냥 2주 정도 써보시고
- 이게 있으면 편한지,
- 없어지면 아쉬운지

그 감만 한 번 느껴보시면 됩니다. 🙂

나중에라도,
"이거 없으면 다시 힘들겠다"는 느낌이 생기면,
그때 편하게 Pro를 고민하셔도 전혀 늦지 않습니다.
```

**핵심:**
- 거절을 거절로 받지 않음
- "나중에" 프레임 유지
- 압박 제로

---

### ❌ "우리는 아직 돈 쓰기는 좀…"

**정답:**
```text
충분히 이해합니다.

그래서 지금은
"돈을 써서 뭔가를 바꾸자"보다는,

그냥 **운영자님 시간을 얼마나 줄여 줄 수 있는지**만
한 번 같이 보는 단계라고 생각해 주시면 좋을 것 같아요.

나중에라도,
"이거 없으면 다시 귀찮겠다"는 느낌이 생기면,
그때 편하게 Pro를 고민하셔도 전혀 늦지 않습니다.
```

**핵심:**
- "돈" 프레임을 "시간" 프레임으로 전환
- 공감 먼저
- 선택권은 상대에게

---

### ❌ "블로그 같은 건 우리가 안 해요"

**정답:**
```text
맞아요. 요즘은 '블로그'까지 일부러 하시는 팀은 거의 없죠.

그래서 이건 굳이 블로그라기보다는
**'팀 홍보용 한 장짜리 페이지'**라고 생각해 주시면 좋아요.

지금도,
- 카카오톡에 링크 한 번 보낼 때,
- "우리 팀이 어떤 팀인지" 설명할 때,

매번 글을 새로 쓰시잖아요?
그걸 이 페이지 하나로 대신하는 느낌이라고 보시면 됩니다.
```

**핵심:**
- "블로그" 프레임을 "홍보 페이지"로 전환
- 현재 불편함을 상기시킴
- 기능 설명이 아닌 "대체" 프레임

---

## 2️⃣ 보류 반응별 대응

### 🤔 "한 달 정도 더 써보고 결정할게요"

**정답:**
```text
네, 그게 제일 좋은 접근인 것 같아요.

지금은 Free 플랜으로도
- 팀 소개 글
- 활동 후기 (월 2회)

까지는 계속 쓰실 수 있으니까,
그냥 편하게 써보시면서
"이 정도면 Pro로 올려볼 만하다" 싶으실 때
그때 결정하셔도 전혀 늦지 않습니다.

그동안 궁금한 거 있으시면
언제든 편하게 물어보세요. 🙂
```

**핵심:**
- 보류를 긍정적으로 받아들임
- Free 플랜도 가치 있음을 언급
- 관계 유지

---

### 🤔 "다른 팀들도 쓰는지 궁금해요"

**정답:**
```text
지금은 초기 단계라서
아직 많이 쓰시는 팀은 많지 않아요.

대신에,
지금 체험 중인 몇 팀에서
"이거 없으면 다시 귀찮겠다"는 피드백을 받고 있어서,
그 팀들 기준으로는 계속 쓰실 예정이에요.

운영자님도 한 번 써보시고,
"이 정도면 계속 쓸 만하다" 싶으시면
그때 같이 결정하시면 될 것 같아요. 🙂
```

**핵심:**
- 솔직함 (과장 없음)
- 다른 팀의 긍정적 피드백 언급
- 선택권은 상대에게

---

## 3️⃣ 즉결 반응별 대응

### ✅ "이거 좋네요, 바로 Pro로 할게요"

**정답:**
```text
감사합니다, 운영자님. 🙏

그럼 지금 바로 Pro로 전환해 드릴게요.

이제부터는
- 주간 요약
- 무제한 활동 후기
- SNS 공유용 요약

까지 모두 자동으로 제공됩니다.

앞으로 쓰시다가
- 불편한 점
- 이런 것도 자동으로 됐으면 좋겠다 싶은 점

생기시면,
"고객"이 아니라 **파트너**라고 생각하시고
그냥 편하게 말씀해 주세요.

운영자님 기준으로 진짜 편한 도구가 되도록
같이 계속 만들어가면 좋겠습니다.
```

**핵심:**
- 감사 표현
- "파트너" 프레임으로 관계 전환
- 지속적 피드백 요청

---

### ✅ "좋은데, 가격 좀 낮춰주면 바로 할게요"

**정답:**
```text
운영자님, 가격에 대한 의견 감사합니다.

지금은 초기 단계라서
가격 정책이 아직 완전히 고정된 건 아니에요.

운영자님 기준으로
"이 정도면 낼 수 있다" 싶은 가격이
대략 어느 정도인지 알려주시면,
그걸 참고해서 정책을 조정해 볼게요.

일단은 지금 가격으로라도
"이 정도면 쓸 만하다" 싶으시면
그렇게 진행하시고,

나중에 가격이 조정되면
그때 알려드릴게요. 🙂
```

**핵심:**
- 가격 협상은 "피드백"으로 받아들임
- 유연성 보여주기 (하지만 즉시 약속은 안 함)
- 관계 유지

---

## 4️⃣ 애매한 반응별 대응

### 😕 "음… 잘 모르겠네요"

```text
그 감만 한 번 느껴보시면 될 것 같아요.

그동안 궁금한 거 있으시면
언제든 편하게 물어보세요. 🙂
```

**핵심:**
- 애매함을 긍정적으로 받아들임
- "체험" 프레임 유지
- 압박 제로

## 🎯 공통 원칙

1. **설득하지 않는다** → 선택권은 항상 상대에게
2. **압박하지 않는다** → "나중에라도" 프레임 유지
3. **공감한다** → "그럴 수 있다" 먼저 말하기
4. **관계를 유지한다** → 거절해도 "나중에" 문구로 마무리
