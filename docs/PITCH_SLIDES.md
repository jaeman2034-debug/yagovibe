# 🎯 투자 피치 슬라이드 (PPT 문장)

**총 슬라이드**: 12장  
**소요 시간**: 10분

---

## 슬라이드 1: 타이틀

**제목**: AI 기반 로컬 거래 슈퍼앱

**부제**: 이미지만 올리면 10초 만에 거래가 시작되는 지역 마켓플레이스

**로고 + 연락처**

---

## 슬라이드 2: 문제 (Problem)

**헤드라인**: 중고거래의 3대 치명 약점

**내용**:
- 신뢰 부족: 사기 위험, 노쇼율 40%+
- 사용자 경험 저하: 수동 입력, 저품질 사진, 가격 혼란
- 지역 상권 연결 부재: 온라인 ↔ 오프라인 단절

**시각화**: 기존 플랫폼 사용자 불만 통계

---

## 슬라이드 3: 솔루션 (Solution)

**헤드라인**: AI 자동화 + 신뢰 인프라 + 생태계

**3개 핵심**:
1. **AI 자동화**: 사진 → 제목/카테고리/가격 10초 자동 생성
2. **신뢰 인프라**: 평판 + 에스크로 + 오프라인 인증
3. **생태계**: 광고 + 파트너 API + 프랜차이즈

**시각화**: 솔루션 아키텍처 다이어그램

---

## 슬라이드 4: 제품 데모

**헤드라인**: 10초 등록, 1분 거래 시작

**플로우**:
1. 사진 1장 업로드
2. AI 자동 추천 (제목/카테고리/가격)
3. 등록 완료
4. 추천 피드 노출
5. 안전결제 (에스크로)
6. 오프라인 인증
7. 거래 확정

**시각화**: 실제 앱 스크린샷

---

## 슬라이드 5: 시장 기회

**헤드라인**: 55조원 시장, 7조원 SAM

**시장 규모**:
- TAM: 55조원 (중고거래 20조 + 광고 5조 + 생활서비스 30조)
- SAM: 7조원 (로컬 중고거래 5조 + 지역 광고 2조)
- SOM: 500억원 (1년 목표)

**시각화**: 시장 규모 파이 차트

---

## 슬라이드 6: 트랙션 지표

**헤드라인**: 목표 지표 vs 기존 플랫폼

**핵심 지표**:
- 등록 전환율: 72% (기존 30%)
- 거래 성공률: 65% (기존 40%)
- 노쇼율: -80% (기존 40% → 8%)
- DAU/MAU: 38% (기존 20%)

**시각화**: 지표 비교 바 차트

---

## 슬라이드 7: 비즈니스 모델

**헤드라인**: 다각화 수익 구조

**수익원**:
- 거래 수수료 3% (60%)
- 지역 광고 (25%)
- 보관/검수 서비스 (10%)
- 파트너 API (5%)

**예상 수익**:
- 월 매출: 2.3억원
- 연간 매출 (ARR): 27.6억원

**시각화**: 수익 구조 파이 차트

---

## 슬라이드 8: 경쟁우위

**헤드라인**: 3가지 진입장벽

**차별화 포인트**:
1. **기술적 진입장벽**: 이미지 AI 파이프라인 (사진 → 자동 등록)
2. **신뢰 인프라**: 오프라인 인증 (GPS + QR) + 에스크로
3. **생태계 네트워크**: 파트너 API + 프랜차이즈 모델

**경쟁사 비교표**: 우리 vs 경쟁사 A vs 경쟁사 B

---

## 슬라이드 9: 로드맵

**헤드라인**: 24개월 성장 계획

**타임라인**:
- **0~6개월**: 지역 확장 (수도권 3개 지역, 지점 10개)
- **6~12개월**: 파트너 생태계 (파트너 50개, API 플랫폼)
- **12~18개월**: 프랜차이즈 확장 (지점 100개, 검수/보관 확대)
- **18~24개월**: 해외 확장 (동남아 3개국)

**시각화**: 타임라인 다이어그램

---

## 슬라이드 10: 팀

**헤드라인**: 핵심 인재 구성

**팀 구성**:
- **개발팀**: 5명 (AI 2명, 백엔드 2명, 프론트엔드 1명)
- **운영팀**: 3명 (PM 1명, 운영 2명)
- **비즈니스**: 2명 (BD 1명, 영업 1명)

**핵심 인재**: 각 팀 리더 프로필

---

## 슬라이드 11: 투자 제안

**헤드라인**: Seed 라운드 10억원

**투자 조건**:
- **규모**: 10억원
- **Runway**: 18개월
- **목표 ARR**: 50억원 (18개월 후)
- **평가**: 협의

**자금 사용 계획**:
- 개발 40% (4억원)
- 마케팅 30% (3억원)
- 운영 20% (2억원)
- 예비비 10% (1억원)

**시각화**: 자금 사용 계획 파이 차트

---

## 슬라이드 12: 핵심 메시지

**헤드라인**: 왜 지금, 왜 우리인가?

**3가지 핵심**:
1. **기술적 진입장벽**: AI 자동화로 사용자 경험 혁신
2. **신뢰 인프라**: 오프라인 인증으로 사기/노쇼 95% 감소
3. **생태계 확장**: 파트너 API + 프랜차이즈로 무한 확장

**시각화**: 핵심 가치 제안 다이어그램

**연락처**: 이메일, 전화번호
