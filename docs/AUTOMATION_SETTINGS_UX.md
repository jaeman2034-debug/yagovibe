# ⚙️ 자동화 설정 화면 (관리자 UX)

## 0️⃣ 화면 목적 (한 줄)

**설정하는 화면이 아니라, 안심하는 화면**

- ❌ 옵션 많음
- ❌ 전문 용어
- ⭕ 상태 확인 + 최소 토글

---

## 📋 목적
관리자가 "아무것도 안 해도 되는 느낌"을 받게 만드는 마지막 퍼즐

---

## 🎯 원칙

### UX 목표

- 설정이 복잡하지 않음
- "자동으로" 강조
- 선택권은 있지만 기본값이 최적
- 설명은 최소화

---

### 톤 가이드

- 담백함
- 과장 없음
- 신뢰감
- 편안함

---

## 1️⃣ 화면 구조 (위 → 아래 고정)

1. [ 자동화 상태 요약 ]
2. [ 기록 자동화 ]
3. [ 공유 자동화 ]
4. [ 성과 요약 수신 ]
5. [ Pro 상태 ]

---

## 2️⃣ 자동화 상태 요약 (상단 카드)

### 상태 문구 (자동)

```
🟢 자동 기록 중

최근 기록: 7월 21일

다음 자동 생성: 7월 28일 예정
```

### 보조 문구

```
별도 설정 없이 자동으로 진행됩니다.
```

**📌 목적: 불안 제거**

---

## 3️⃣ 기록 자동화 (핵심 토글)

### 토글

```
활동 기록 자동 생성 ⏽ ON
```

### 설명 (항상 노출)

```
경기·모임·연습이 있으면
담백한 기록이 자동으로 생성됩니다.
```

### 보조 안내 (작게)

```
과장된 표현 없음
사진·실명 사용 안 함
```

**❌ 세부 설정 ❌**
**❌ 미리보기 ❌**

---

### 토글 OFF 시 문구

```
자동 기록이 중단됩니다.
기존 기록은 그대로 유지됩니다.

언제든지 다시 켤 수 있습니다.
```

---

## 4️⃣ 공유 자동화 (2번째 토글)

### 토글

```
외부 공유 자동 생성 ⏽ ON
```

### 설명

```
기록이 카드 형태로 만들어져
카카오톡·밴드 등에서 공유됩니다.
```

### 보조 안내

```
광고처럼 보이지 않도록 설계되어 있습니다.
```

---

### 토글 OFF 시 문구

```
공유 자동 생성이 중단됩니다.
기존 기록은 그대로 유지됩니다.

언제든지 다시 켤 수 있습니다.
```

---

## 5️⃣ 성과 요약 수신 (선택)

### 토글

```
주간 요약 받기 ⏽ ON / OFF
```

### 설명

```
지난 7일의 방문과 반응을
간단한 숫자로만 알려드립니다.
```

**📌 메일/알림 설정 ❌**
**→ 자동 채널 고정**

---

### 토글 OFF 시 문구

```
주간 요약 수신이 중단됩니다.

언제든지 다시 켤 수 있습니다.
```

---

## 2️⃣ 설정 화면 문구

### 기본 상태 (모든 자동화 ON)

**메인 메시지:**
```
모든 것이 자동으로 설정되어 있습니다.

활동만 하시면 기록이 자동으로 쌓입니다.
```

**설명:**
```
별도로 설정하실 필요는 없습니다.
필요하실 때만 변경하시면 됩니다.
```

---

### 설정 변경 안내

**제목:**
```
설정 변경
```

**설명:**
```
자동 기록 방식을 변경하실 수 있습니다.

기본 설정이 가장 권장됩니다.
```

---

### 설정 저장 안내

**저장 시:**
```
설정이 저장되었습니다.

변경 사항은 다음 기록부터 적용됩니다.
```

---

## 3️⃣ 토글 설명 문구

### 토글 1: 자동 기록 활성화

**ON 상태:**
```
활동이 있으면 자동으로 기록이 생성됩니다.

관리자는 따로 할 일이 없습니다.
```

**OFF 상태:**
```
자동 기록이 비활성화되었습니다.

필요하실 때 수동으로 생성하실 수 있습니다.
```

---

### 토글 2: 공개 페이지 자동 업데이트

**ON 상태:**
```
기록이 생성되면 공개 페이지에 자동으로 반영됩니다.

외부에서도 바로 확인할 수 있습니다.
```

**OFF 상태:**
```
공개 페이지는 수동으로 업데이트됩니다.

필요하실 때만 반영하실 수 있습니다.
```

---

### 토글 3: 공유 카드 자동 생성

**ON 상태:**
```
기록이 생성되면 공유 카드가 자동으로 만들어집니다.

카카오톡이나 밴드로 바로 공유할 수 있습니다.
```

**OFF 상태:**
```
공유 카드는 수동으로 생성됩니다.

필요하실 때만 만들 수 있습니다.
```

---

## 4️⃣ 설정 화면 레이아웃

### 기본 구조

```
┌─────────────────────────────────┐
│  자동 기록 설정                  │
│  활동이 있으면 자동으로          │
│  기록됩니다.                     │
├─────────────────────────────────┤
│                                 │
│  ✅ 자동 기록                    │
│  활동이 있으면 자동으로          │
│  기록이 생성됩니다.              │
│                                 │
│  ✅ 공개 페이지 자동 업데이트    │
│  기록이 생성되면 공개 페이지에   │
│  자동으로 반영됩니다.            │
│                                 │
│  ✅ 공유 카드 자동 생성          │
│  기록이 생성되면 공유 카드가     │
│  자동으로 만들어집니다.          │
│                                 │
│  [ 설정 저장 ]                  │
│                                 │
└─────────────────────────────────┘
```

---

### 모바일 최적화

- 토글 크기: 48px 이상
- 설명 텍스트: 읽기 쉬운 크기
- 저장 버튼: 하단 고정

---

## 5️⃣ 설정 변경 플로우

### 변경 전

**안내:**
```
현재 모든 자동화가 활성화되어 있습니다.

변경하시겠어요?
```

---

### 변경 중

**확인:**
```
설정을 변경하시면 다음 기록부터 적용됩니다.

계속하시겠어요?
```

---

### 변경 후

**완료:**
```
설정이 저장되었습니다.

변경 사항은 다음 기록부터 적용됩니다.
```

---

## 6️⃣ 기본값 설명

### 기본값 (모든 자동화 ON)

**설명:**
```
기본 설정은 가장 권장되는 방식입니다.

활동만 하시면 모든 것이 자동으로 처리됩니다.
```

---

### 기본값 변경 안내

**경고 (부드럽게):**
```
기본 설정을 변경하시면

일부 기능이 수동으로 처리됩니다.

계속하시겠어요?
```

---

## 7️⃣ 설정 초기화

### 초기화 버튼

**위치:** 설정 화면 하단

**문구:**
```
기본 설정으로 되돌리기
```

**확인:**
```
모든 설정을 기본값으로 되돌리시겠어요?

변경 사항은 모두 초기화됩니다.
```
