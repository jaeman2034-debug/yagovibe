## 📋 구현된 기능

### 랭킹 표시
- 순위 (1-3위는 트로피 아이콘)
- 팀명 (로고, 종목, 지역)
- 경기 수
- 승/무/패
- 승률 (%)
- 득점/실점
- 득실차 (색상 구분: 양수=초록, 음수=빨강)
- 연속 기록 (연승/연패/연무)

### 정렬 기준
1. **승률** (기본)
   - 승률이 같으면 경기 수가 많은 순

2. **득실차**
   - 득실차가 같으면 승률 순

3. **승수**
   - 승수가 같으면 승률 순

4. **경기 수**
   - 경기 수가 같으면 승률 순

### 필터
- **종목**: 전체, 축구, 농구, 야구
- **최소 경기 수**: 1경기, 3경기, 5경기, 10경기 이상

## 🔄 데이터 흐름

```
teams 컬렉션
   ↓
teams.stats 필드 (denormalized)
   ↓
클라이언트 정렬
   ↓
랭킹 테이블 표시
```

**특징**:
- 별도 랭킹 컬렉션 불필요
- `teams.stats`만으로 랭킹 계산
- 실시간 업데이트 (teams.stats가 업데이트되면 자동 반영)
