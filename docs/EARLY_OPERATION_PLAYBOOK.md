# 📋 초기 운영 플레이북 (1인 운영 기준)

## 📋 목적
하루 30분으로 10개 팀을 돌보는 반복 가능한 구조

---

## 0️⃣ 운영 철학 (가장 중요)

- ❌ 잘해주려 하지 말 것
- ❌ 반응을 만들려 애쓰지 말 것
- ⭕ 시스템이 일하게 두기

**운영자는 "관리자"가 아니라 기록이 잘 흘러가게 두는 사람**

---

## 🎯 원칙

### 시간 제한

- **하루 30분** (절대 초과 금지)
- 반복 가능한 루틴
- 자동화 우선

---

### 우선순위

1. **기록 생성** (가장 중요)
2. **반응 확인** (긍정/부정)
3. **Pro 전환** (자연스럽게)

---

## 1️⃣ 하루 30분 운영 루틴 (고정)

### ⏱ 0–5분 | 상태 체크

**체크리스트:**
- [ ] 자동 글 생성 성공 여부
- [ ] 깨진 페이지 / 오류 여부
- [ ] Pro 만료/결제 실패 알림

**✅ 대응 없음이 정상**

---

### ⏱ 5–15분 | 팀 1~2곳만 확인

**체크리스트:**
- [ ] 최근 활동 데이터 들어왔는지
- [ ] 글 톤 이상 여부 (과장/어색)

**👉 수정 필요 시 한 문장만 손보기**  
**❌ 전체 수정 금지**

---

### ⏱ 15–25분 | 관리자 접점 (선별)

**체크리스트:**
- [ ] 지난 24시간 반응 있는 팀만 확인
- [ ] 링크 공유
- [ ] 관리자 페이지 방문

**무반응 팀 ❌ 접근 금지**

**📌 기준:**
"신호가 온 곳만 응답한다"

---

### ⏱ 25–30분 | 기록

**체크리스트:**
- [ ] 오늘 특이사항 1줄
- [ ] 반복 문의 있으면 메모

---

## 2️⃣ 주간 루틴 (주 1회, 30분)

### Week Check

**체크리스트:**
- [ ] 자동 글 생성 실패율
- [ ] 공유 카드 클릭률
- [ ] Pro 업셀 노출 → 클릭

**조정 가능한 것 (1개만):**
- Hero 문구
- 자동 글 톤
- 공유 카드 문구

**❌ 2개 이상 동시에 수정 금지**

---

## 3️⃣ 팀별 케어 레벨 (자동 분류)

| 레벨 | 기준 | 대응 |
|------|------|------|
| A | 공유 + 방문 + 반응 | 유지 |
| B | 방문만 | 방치 |
| C | 무반응 | 자동 중단 |

**👉 B, C에 시간 쓰지 말 것**

---

## 4️⃣ 관리자 문의 대응 템플릿 (복붙)

### 기능 질문

```
특별히 설정하실 건 없습니다.
활동만 있으면 자동으로 기록됩니다.
```

---

### 효과 질문

```
지금 페이지 흐름은 정상입니다.
조금 더 기록이 쌓이면 자연스럽게 반응이 옵니다.
```

---

### 가격 질문

```
지금은 사용 여부를 판단하실 단계예요.
필요하실 때 바로 켜실 수 있습니다.
```

---

## 5️⃣ Pro 전환 타이밍 운영 규칙

### ❌ 먼저 말하지 말 것

- "Pro 어떠세요?"
- "업그레이드 하세요"

---

### ⭕ 말해도 되는 순간

- 관리자 페이지 2회 이상 방문
- 성과 리포트 스크롤 70% 이상
- 공유 후 "사람이 봤다" 언급

**👉 이때만 Pro 안내 자동 노출**

---

## 6️⃣ 위기 대응 플레이북 (중요)

### 글 톤 불만

```
말씀 주셔서 감사합니다.
조금 더 기록 위주로 조정해둘게요.
```

**→ 즉시 반영, 설명 ❌**

---

### "홍보 같아요"

```
그렇게 느껴지셨다면 죄송합니다.
기록만 남기도록 바로 조정하겠습니다.
```

**→ 시스템 철학 재확인**

---

## 7️⃣ 30일 유지 성공 기준

- 운영자 하루 평균 ≤ 25분
- 자동 글 중 수동 수정 ≤ 10%
- Pro 팀 3팀 이상
- 운영자가 "귀찮다"는 느낌 ❌

---

## 8️⃣ 절대 하면 안 되는 것 (운영자 금기)

- 팀에 대신 글 써주기
- 개별 팀 맞춤 설계
- 예외 처리 남발
- 장문의 설명

**이 순간부터 사람이 일하기 시작함**

---

## 9️⃣ 운영자 멘탈 가이드 (중요)

**반응 없음 = 실패 ❌**  
**반응 없음 = 기본값**

**👉 기록은 쌓일수록 힘이 생김**

---

## 🔟 내부 공유용 한 문장

**"우리는 팀을 관리하지 않는다. 시스템이 팀을 기록하게 둔다."**

---

## 📊 상태별 대응 (자동화)

### 상태 A: 새 요청

**트리거:** DM/댓글에서 "팀 이름 + 활동 날짜" 요청

**액션:**
1. 팀명/지역/활동 1건 수동 입력 (5분)
2. 랜딩 생성 (자동)
3. 자동 글 1건 생성 (자동)
4. 공유 카드 생성 (자동)
5. 링크 전달 (1분)

**총 시간:** 6분

---

### 상태 B: 긍정 반응

**트리거:** "이거 좋네요" 또는 "기록으로 남는 게 좋네요"

**액션:**
1. "앞으로도 자동으로 남으면 좋을까요?" 질문 (1분)
2. 긍정 응답 시 → Pro 안내 (2분)

**총 시간:** 3분

---

### 상태 C: 부정 반응

**트리거:** "필요 없어요" 또는 "복잡해요"

**액션:**
1. "알겠습니다. 필요해지면 언제든 말씀 주세요." (1분)
2. 기록 중단 (자동)

**총 시간:** 1분

---

### 상태 D: 무반응

**트리거:** 3일 이상 반응 없음

**액션:**
- 추가 연락 ❌
- 링크 회수 ❌
- 자동 기록 중단 (자동)

**총 시간:** 0분

---
