# 💰 요금제 결제 UX / 카피 가이드

**"지금 필요해서 결제"하는 느낌을 주는 카피**

---

## 🎯 핵심 원칙

- ❌ 억지 / 설득
- ⭕ **필요한 순간에, 필요한 이유로 결제**
- ⭕ **"구독"이 아니라 "운영"이라는 단어 사용**

---

## 📍 트리거별 카피

### 트리거 1: 엑셀 다운로드 클릭 시 (최강)

**모달 제목:**
```
엑셀 다운로드
```

**설명:**
```
대회 운영에 필요한
✔ 선수 명단 엑셀
✔ 참가팀 요약 엑셀
을 한 번에 받아보세요.
```

**안내 문구:**
```
엑셀 다운로드는 BASIC 플랜부터 이용 가능합니다.
```

**👉 "지금 필요해서 결제"**

---

### 트리거 2: 알림 자동화 첫 사용 시

**모달 제목:**
```
알림 자동화
```

**설명:**
```
미제출 팀에 자동으로 알림을 보내
전화/문자 업무를 줄여보세요.
```

**안내 문구:**
```
알림 자동화는 BASIC 플랜 기능입니다.
```

---

### 트리거 3: 결제 연동 사용 시 (PRO)

**모달 제목:**
```
참가비 결제 연동
```

**설명:**
```
입금 확인과 정산을 자동화해보세요.
```

**안내 문구:**
```
참가비 결제 연동은 PRO 플랜에서 제공됩니다.
```

---

## 💳 요금제 선택 화면 카피

### FREE 플랜

**제목:**
```
🟢 FREE
```

**기능:**
```
✓ 대회 1개
✓ 참가 신청 / 승인
✓ 선수 명단 관리
```

**버튼:**
- 현재 플랜: `[ 현재 사용 중 ]` (비활성)
- 다른 플랜 사용 중: `[ 선택하기 ]`

---

### BASIC 플랜 ⭐ 추천

**제목:**
```
🔵 BASIC ⭐ 추천
```

**가격:**
```
월 49,000원
또는 대회당 99,000원
```

**기능:**
```
✓ 대회 무제한
✓ 엑셀 다운로드
✓ 알림 자동화
✓ 플랫폼 로고 제거
```

**버튼:**
- 현재 플랜: `[ 현재 사용 중 ]` (비활성)
- 업그레이드: `[ 이 기능 사용하기 ]` ⭐

**👉 "구독"이 아니라 "기능 사용"**

---

### PRO 플랜

**제목:**
```
🟣 PRO
```

**가격:**
```
월 149,000원
```

**기능:**
```
✓ BASIC 전부 포함
✓ 참가비 결제 연동
✓ 현장 체크인 (QR)
✓ 자동 영수증
✓ 우선 지원
```

**버튼:**
- 현재 플랜: `[ 현재 사용 중 ]` (비활성)
- 업그레이드: `[ 대회 운영 시작하기 ]` ⭐

**👉 "운영"이라는 단어로 공감대 형성**

---

## 🛡️ 심리적 안전장치 (하단 문구)

**필수 문구:**
```
✓ 언제든지 플랜 변경이 가능합니다.
✓ 대회 종료 후 자동 결제되지 않습니다.
```

**👉 지자체/협회 결제 승인률 급상승**

---

## 🎨 레이아웃 원칙

1. **FREE는 항상 왼쪽**
   - 기본값으로 인식
   - "무료로 시작" 느낌

2. **BASIC에 ⭐ "추천"**
   - 가장 많이 팔릴 플랜
   - 시각적으로 강조

3. **PRO는 "조금 비싸 보이게"**
   - 업셀용
   - "프리미엄" 느낌

---

## 💬 버튼 카피 가이드

### ❌ 피해야 할 카피

- "결제하기"
- "구독 시작"
- "업그레이드"
- "플랜 변경"

### ⭕ 권장 카피

- "이 기능 사용하기" (BASIC)
- "대회 운영 시작하기" (PRO)
- "선택하기" (플랜 선택)
- "나중에" (취소)

**👉 협회는 "구독"에 거부감 있음**
**👉 "운영"이라는 단어에 즉시 공감**

---

## 🔥 최종 결론

좋은 SaaS는
**'가격을 설명하지 않아도
결제 이유가 보이는 UX'**를 가진다.

지금 이 플랫폼은:
- 기능 설계 ❌ 수준 아님
- **"운영 문제를 해결하고, 돈을 받는 구조"**까지 완성
