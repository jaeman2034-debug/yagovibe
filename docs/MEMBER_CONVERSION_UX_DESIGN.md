# 🔄 비회원 → 회원 전환 UX 3단계 설계

**생성일**: 2025-01-27  
**목적**: 설명 없이 차이 체감 → 행동 연결, 마찰 최소화  
**원칙**: 협회 운영 규칙을 UX로 자동 반영

---

## 🧭 전환 UX 3단계 구조

```
[STEP 1: 상태 인지]
  ↓ (비회원팀이 🟢 슬롯/협회 우선 시설 접근)
[STEP 2: 가치 확인]
  ↓ ("회원팀 전환 안내" 버튼 클릭)
[STEP 3: 문의 → 승인 → 권한 변경]
  ↓ (협회 승인 완료)
[즉시 권한 반영]
```

---

## 📐 STEP 1: 상태 인지 화면

### 트리거 시점

1. **시설 상세 화면**에서 비회원팀이 협회 우선 대관 시설 확인
2. **대관 캘린더 화면**에서 비회원팀이 🟢 슬롯 클릭
3. **운동장 대관 메인**에서 비회원팀이 협회 우선 대관 시설 카드 접근
