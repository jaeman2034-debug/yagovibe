# 💰 팀 블로그 Pro 전환 트리거 설계

## 📋 목적
무료 사용자가 Pro 플랜으로 전환하는 최적의 타이밍과 방법을 설계하여 수익화 최적화

---

## 1️⃣ 전환 트리거 타이밍 (우선순위 순)

### 트리거 1: 첫 블로그 글 생성 직후 (가장 강력)

**시점:**
- AI가 첫 소개 글을 생성한 직후
- 사용자가 "생성 완료" 메시지를 본 직후

**표시 위치:**
- 팀 블로그 관리 페이지 (`TeamBlogManagement`)
- 첫 글 생성 완료 모달/토스트 하단

**메시지:**
```
✨ 첫 글이 생성되었습니다!

이제 Pro로 업그레이드하면:
• 주간 자동 포스팅 (매주 자동)
• SEO 최적화 (검색 노출 강화)
• 사진 기반 AI 설명 자동 생성
• 브랜딩 제거

[ Pro 시작하기 - 월 19,000원 ]
```

**전환율 예상:** 15-25% (첫 가치 체감 시점)

---

### 트리거 2: 블로그 조회수 증가 시

**시점:**
- 블로그 조회수가 50회 이상
- 또는 일주일 내 조회수 20회 이상

**표시 위치:**
- 팀 블로그 관리 페이지 상단
- 통계 카드 옆

**메시지:**
```
🔥 이 팀 블로그가 반응을 얻고 있어요!

지금까지 50명이 방문했습니다.
Pro로 업그레이드하면:
• 검색 노출 강화 (더 많은 방문자)
• 주간 자동 포스팅 (지속적인 콘텐츠)
• 상세 통계 (방문자 분석)

[ Pro로 더 키워보기 - 월 19,000원 ]
```

**전환율 예상:** 10-20% (성과 확인 시점)

---

### 트리거 3: 두 번째 글 생성 시도 시

**시점:**
- 무료 플랜에서 두 번째 글 생성 버튼 클릭
- 또는 "주간 포스트 생성" 버튼 클릭

**표시 위치:**
- Paywall 모달 (기존 구조 활용)

**메시지:**
```
💡 무료 플랜은 월 2개 글까지만 가능합니다.

Pro로 업그레이드하면:
• 무제한 자동 포스팅
• 주간 요약 자동 생성
• SEO 최적화

[ Pro 시작하기 - 월 19,000원 ]
[ 무료로 계속하기 ]
```

**전환율 예상:** 20-30% (제한 경험 시점)

---

### 트리거 4: 외부 공개 블로그 방문자 증가

**시점:**
- 공개 블로그 페이지 조회수 일주일 내 30회 이상
- 또는 가입 요청 3건 이상

**표시 위치:**
- 팀 대시보드 알림 영역
- 또는 블로그 관리 페이지

**메시지:**
```
🎉 팀 블로그가 관심을 받고 있어요!

이번 주 30명이 방문했고, 가입 요청 3건이 들어왔습니다.
Pro로 업그레이드하면:
• 검색 노출 강화 (더 많은 방문자)
• 자동 홍보 (SNS 공유 최적화)
• 상세 통계

[ Pro로 더 키워보기 - 월 19,000원 ]
```

**전환율 예상:** 12-18% (외부 반응 확인 시점)

---

### 트리거 5: 관리자 대시보드 진입 시 (선택적)

**시점:**
- 팀 대시보드 진입 시 (첫 진입 또는 주기적)
- 블로그 탭 클릭 시

**표시 위치:**
- 블로그 관리 페이지 상단 (배너 형태)
- 또는 사이드바 (작은 배너)

**메시지:**
```
🤖 AI가 블로그를 관리 중입니다

Pro로 업그레이드하면:
• 주간 자동 포스팅
• SEO 최적화
• 브랜딩 제거

[ Pro 시작하기 - 월 19,000원 ]
```

**전환율 예상:** 5-10% (노출 빈도 높지만 전환 낮음)

## 2️⃣ 무료 vs Pro 기능 비교 (명확한 경계선)

### 무료 플랜

**제공 기능:**
- ✅ 팀 블로그 1개
- ✅ 첫 소개 글 자동 생성 (1회)
- ✅ 활동 후기 자동 생성 (월 2회까지)
- ✅ 공개 블로그 페이지
- ✅ 기본 가입 CTA
- ❌ 주간 자동 포스팅
- ❌ SEO 최적화
- ❌ 브랜딩 제거
- ❌ 상세 통계
- ❌ 사진 기반 AI 설명

**제한사항:**
- 월 2개 글까지만 생성 가능
- "Powered by YAGO" 표시 필수
- 기본 SEO만 제공

---

### Pro 플랜

**제공 기능:**
- ✅ 팀 블로그 1개
- ✅ 첫 소개 글 자동 생성 (무제한)
- ✅ 활동 후기 자동 생성 (무제한)
- ✅ 주간 자동 포스팅 (매주 자동)
- ✅ SEO 최적화 (키워드, 메타 태그)
- ✅ 브랜딩 제거
- ✅ 상세 통계 (방문자, 조회수, 전환율)
- ✅ 사진 기반 AI 설명 자동 생성
- ✅ 예약 발행
- ✅ SNS 공유용 요약 자동 생성

**추가 혜택:**
- 무제한 글 생성
- 커스텀 도메인 (향후)
- 스폰서용 리포트 (향후)

**일반 멤버:**

## 6️⃣ Analytics 추적

### 추적 이벤트

1. **paywall_impression**
   - Paywall 모달/배너 노출
   - 파라미터: `trigger`, `teamId`, `plan: "free"`

2. **upgrade_click**
   - "Pro 시작하기" 버튼 클릭
   - 파라미터: `trigger`, `teamId`, `plan: "pro"`

3. **upgrade_cancel**
   - "나중에" 또는 닫기 버튼 클릭
   - 파라미터: `trigger`, `teamId`

4. **upgrade_success**
   - Pro 결제 완료
   - 파라미터: `teamId`, `plan: "pro"`

## 7️⃣ 전환율 최적화 전략

### A/B 테스트 항목

1. **가격 표시**
   - A안: "월 19,000원"
   - B안: "연간 180,000원 (월 15,000원)"

2. **메시지 톤**
   - A안: 가치 중심 ("자동화하세요")
   - B안: 결과 중심 ("더 많은 방문자")

3. **버튼 텍스트**
   - A안: "Pro 시작하기"
   - B안: "무제한 포스팅 시작하기"
