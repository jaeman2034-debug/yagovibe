# 🌍 공개 팀 블로그 랜딩 페이지 - 실전 문구 & 전환 설계

## 📋 목적
외부인이 처음 보는 페이지에서 "이 팀 들어가고 싶다"는 욕구를 만드는 구조

---

## 1️⃣ 히어로 영역 (첫 3초 승부)

### 메인 메시지
```
{팀명}
{연령대}에도 우리는 계속 뛰고 있습니다.
```

**예시:**
- `소흘 60대 FC`
- `60대에도 우리는 계속 뛰고 있습니다.`

### 신뢰 배지 (3개)
- `✔ 주 1회 활동`
- `✔ 실제 팀 운영 중`
- `✔ AI 관리 블로그` (Pro 플랜만)

### 메인 CTA 버튼
**텍스트:** `함께 운동하기`  
**스타일:** 흰색 배경, 파란색 텍스트, 큰 버튼, 그림자

### 서브 CTA 버튼
**텍스트:** `팀 소개 보기`  
**스타일:** 반투명 흰색 배경, 흰색 텍스트

---

## 2️⃣ 팀 핵심 정보 카드 (가입 장벽 제거용)

### 제목
`팀 핵심 정보`

### 정보 항목 (4개)
1. **활동 지역**
   - 아이콘: 📍
   - 라벨: `활동 지역`
   - 값: `{team.location}` (예: "포천")

2. **평균 연령**
   - 아이콘: 👥
   - 라벨: `평균 연령`
   - 값: `{team.ageGroup}` (예: "60대")

3. **활동 빈도**
   - 아이콘: 📅
   - 라벨: `활동 빈도`
   - 값: `주 1회` (고정)

4. **분위기**
   - 아이콘: 🏆
   - 라벨: `분위기`
   - 값: `즐겁게 / 안전하게` (고정)

---

## 3️⃣ 중간 CTA 섹션 (전환 핵심)

### 제목
`🙋 이런 분을 기다립니다`

### 추천 대상 리스트 (3개)
- `• {연령대} 남성` (팀 데이터 기반)
- `• 운동 경험 무관`
- `• 즐겁게 오래 하실 분`

### CTA 버튼
**텍스트:** `이 팀에 가입 신청하기`  
**스타일:** 파란색 배경, 흰색 텍스트, 큰 버튼

### 클릭 시 동작
1. **로그인 안 됨** → `/start` (로그인 페이지)
2. **로그인됨** → `team_requests` 컬렉션에 가입 요청 생성
   - 성공 메시지: `가입 요청이 전송되었습니다! 관리자의 승인을 기다려주세요.`
   - 중복 요청: `이미 가입 요청을 보내셨습니다. 관리자의 승인을 기다려주세요.`

---

## 4️⃣ 블로그 콘텐츠 영역

### 탭 구조
1. **팀 소개** (기본)
   - 첫 글 (postType: "intro") 표시
   - 없으면: `팀 소개를 준비 중입니다.`

2. **활동**
   - 활동 포스트 목록 (postType !== "intro")
   - 없으면: `활동 내역이 없습니다.`

3. **후기**
   - 후기 포스트 목록
   - 없으면: `후기가 없습니다.`

### 포스트 표시 형식
- 제목 (큰 글씨)
- 요약 (excerpt)
- 본문 (HTML 렌더링)
- 발행일 (작은 글씨)

---

## 5️⃣ 하단 Pro CTA (관리자만 보임)

### 표시 조건
- `isOwner === true` (현재 사용자가 팀 소유자/관리자)

### 제목
`이 팀은 AI가 관리합니다 🤖`

### 기능 리스트 (3개)
- `✓ 자동 블로그`
- `✓ 자동 홍보`
- `✓ 자동 요약 리포트`

### CTA 버튼
**텍스트:** `우리 팀도 AI 관리 시작하기 (Pro)`  
**스타일:** 보라색 배경, 흰색 텍스트  
**클릭 시:** 팀 대시보드로 이동

---

## 6️⃣ 전환 타이밍 설계

### 첫 방문 (비로그인)
1. 히어로 영역 → `함께 운동하기` 클릭
2. 로그인 페이지로 이동
3. 로그인 완료 후 → 원래 블로그 페이지로 복귀
4. 자동으로 `이 팀에 가입 신청하기` 버튼 강조 표시

### 로그인 상태
1. `이 팀에 가입 신청하기` 클릭
2. 즉시 가입 요청 생성
3. 성공 메시지 표시
4. 버튼 비활성화 (이미 요청함)

### 관리자 (소유자)
1. 하단 Pro CTA 표시
2. `우리 팀도 AI 관리 시작하기` 클릭
3. 팀 대시보드 → 블로그 관리 탭으로 이동

## 8️⃣ SEO 메타 태그

### 동적 설정
- **title:** `{팀명} 팀 블로그` 또는 첫 글의 SEO title
- **description:** 첫 글의 excerpt 또는 팀 설명
- **keywords:** 첫 글의 SEO keywords (Pro 플랜만)
- **og:title:** title과 동일
- **og:description:** description과 동일
- **og:url:** `/teams/{teamSlug}/blog`

---

## 9️⃣ Analytics 이벤트

### 추적 이벤트
1. **team_blog_view**
   - 팀 블로그 페이지 진입
   - 파라미터: `teamSlug`, `source: "public"`

2. **team_blog_join_click**
   - "함께 운동하기" 또는 "이 팀에 가입 신청하기" 클릭
   - 파라미터: `teamId`, `teamSlug`

3. **team_blog_join_request_sent**
   - 가입 요청 생성 성공
   - 파라미터: `teamId`, `teamSlug`

4. **team_blog_share**
   - 공유 버튼 클릭
   - 파라미터: `teamId`, `teamSlug`, `method: "web_share" | "clipboard"`
