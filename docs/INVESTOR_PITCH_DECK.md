> **"AI 기반 로컬 거래 슈퍼앱: 이미지만 올리면 10초 만에 거래가 시작되는 지역 마켓플레이스"**

---

## 2️⃣ 문제 (Problem)

### 기존 플랫폼의 3대 치명 약점

#### 1. 신뢰 부족
- 중고거래 = 사기 위험
- 노쇼율 40%+
- 평판 시스템 부재

#### 2. 사용자 경험 저하
- 수동 입력 (제목, 카테고리, 가격)
- 저품질 사진
- 가격 혼란 (시세 정보 없음)

#### 3. 지역 상권 연결 부재
- 온라인 ↔ 오프라인 단절
- 지역 상점과의 연계 없음
- 물류/검수 서비스 부재

### 시장 기회

- **국내 중고거래 시장**: 20조원+ (연간)
- **로컬 광고 시장**: 5조원+ (연간)
- **생활서비스 연계**: 무한 확장 가능

---

## 3️⃣ 솔루션 (Solution)

### 핵심 차별화: AI 자동화 + 신뢰 인프라 + 생태계

#### 🤖 AI 자동화 레이어
- **이미지 → 제목/카테고리/가격 자동 생성**
  - 사진 1장 업로드 → 10초 내 등록 완료
  - 브랜드/아이템 자동 인식
  - 유사 매물 기반 가격 가이드
- **시맨틱 검색**
  - 의미 기반 검색 (제목이 달라도 찾기)
  - 임베딩 기반 추천
- **개인화 추천 피드**
  - 관심사 + 거리 + 평판 종합 점수
  - 실시간 학습 (클릭/채팅/거래 이벤트)

#### 🛡️ 신뢰 인프라
- **평판 시스템**
  - 종합 점수 (평점 + 거래수 + 응답속도 - 노쇼율)
  - 자동 업데이트 (거래/매칭/노쇼 이벤트)
- **에스크로 결제**
  - 구매자 결제 → 에스크로 보관 → 만남/수령 → 확정 → 정산
  - 보증금 10% (노쇼 방지)
- **오프라인 인증**
  - GPS 위치 확인 (100m 이내)
  - QR 코드 상호 인증
  - 체크인 시 평판 부스트

#### 🌐 생태계 확장
- **지역 광고 플랫폼**
  - 지역 타겟팅 (위치 + 반경)
  - CPM/CPC 과금
  - 랭킹 결합 (organic 0.7 + adBid 0.3)
- **파트너 API**
  - 외부 파트너 연동 (세탁소, 수리점, 택배, 중고상)
  - 웹훅 이벤트 전송
  - 거래 수수료 2% + API 호출 과금
- **프랜차이즈 모델**
  - 지점 등록 및 인증
  - 검수/보관/직거래/AS 서비스
  - 직거래 수수료 3%

## 4️⃣ 제품 데모 플로우

```
1. 사진 1장 업로드
   ↓
2. AI 자동 추천 (제목/카테고리/가격)
   ↓
3. 10초 내 등록 완료
   ↓
4. 추천 피드 노출 (관심사 + 거리 + 평판)
   ↓
5. 안전결제 (에스크로 보관)
   ↓
6. 오프라인 인증 (GPS 체크인 + QR 인증)
   ↓
7. 거래 확정 → 정산
```

**핵심 가치**: 사용자 노력 90% 감소, 거래 성공률 2배 향상
