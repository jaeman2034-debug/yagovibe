# 🔴 Google Maps 웹 제약 사항 (정책/제약, 버그 아님)

## 핵심 원인

Google Maps 웹(브라우저)에서 **좌표(lat,lng) 직접 입력 + 자동차 모드 + "지금 출발"** 조합은 제한됩니다.

### 제한되는 경우

```
출발: 37.5665000, 126.9780000 (좌표)
도착: 37.5799313, 126.9306570 (좌표)
모드: 자동차 (DRIVING)
환경: 웹 브라우저
```

→ **"지금 출발" 버튼이 비활성화됨**

### 정상 작동하는 경우

| 환경 | 모드 | 상태 |
|------|------|------|
| 모바일 앱 | 자동차 | ⭕ 정상 |
| 웹 브라우저 | 도보 | ⭕ 정상 |
| 웹 브라우저 | 대중교통 | ⭕ 정상 |
| 웹 브라우저 | 자동차 (주소명) | ⭕ 정상 |

---

## ✅ 해결 방법 (이미 구현됨)

### 1. 모바일 앱으로 열기 (기본 동작)

```ts
google.com/maps/dir/?api=1&destination=lat,lng
```

모바일에서 열면:
- Google Maps **앱으로 자동 전환**
- "지금 출발" 정상 작동
- **실사용 환경 = 정상**

### 2. 도보/대중교통 모드 (웹 테스트용)

웹에서 테스트할 때는:
- 🚗 자동차 → "지금 출발" 비활성 (제한)
- 🚶 도보 → "지금 출발" 활성 ⭕
- 🚌 대중교통 → "지금 출발" 활성 ⭕

### 3. 주소명 사용 (선택적)

좌표 대신 주소명을 사용하면 웹에서도 활성화되지만, **실서비스에서는 굳이 변경 불필요** (모바일에서 정상 작동하므로)

### ✅ 테스트 시나리오

1. **모바일 실기기 테스트 (필수)**
   - [ ] 자동차 모드 선택
   - [ ] Google Maps 앱으로 전환 확인
   - [ ] "지금 출발" 버튼 활성화 확인
   - [ ] 네비게이션 시작 확인

2. **웹 브라우저 테스트 (참고용)**
   - [ ] 도보 모드 → "지금 출발" 활성화 확인
   - [ ] 대중교통 모드 → "지금 출발" 활성화 확인
   - [ ] 자동차 모드 → "지금 출발" 비활성화 (정상, 제한 사항)

### ⚠️ 버그로 오인하지 말 것

- ❌ "지금 출발" 버튼이 안 눌린다 → 버그 아님
- ❌ 자동차 모드에서 오류 → Google Maps 웹 제한
- ✅ 모바일에서 정상 작동 → 정상
