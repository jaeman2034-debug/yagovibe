## 🎯 권한 전이의 정답 원칙

### 1. 위임은 "팀장만" 가능
- `transferOwner` 함수에서 `ownerData.role !== "owner"` 체크
- 권한 없으면 `permission-denied` 에러

### 2. 동시에 팀장은 단 1명
- 트랜잭션으로 원자적 처리
- 기존 owner → coach
- 대상 팀원 → owner

### 3. 위임 순간
- 기존 팀장 → 팀원(P2)
- 대상 팀원 → 팀장(P3)

### 4. 되돌리기 UX 없음
- 확인 모달에서 명확히 경고
- 신중한 액션 유도
