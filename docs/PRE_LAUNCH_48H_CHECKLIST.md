# 기록장 — 출시 직전 48시간 체크리스트

## 목적 (한 줄)

**기능이 아니라 '톤·흐름·신뢰'가 깨지지 않았는지 확인**

---

## T-48h : 구조 & 흐름 점검 (한 번만)

### 🔲 페이지 구조

- [ ] Hero에 활동 증거가 보이는가
- [ ] CTA가 강요 없이 2~3회 자연 노출되는가
- [ ] 데이터 없을 때 섹션이 사라지는가 (빈칸 ❌)

### 🔲 상태별 화면

- [ ] Guest / Member / Admin 전환 정상
- [ ] 공유 유입 시 완화 Hero 적용
- [ ] Admin에게만 Pro 노출

---

## T-36h : 문구 & 톤 검수 (자동 + 육안)

### 🔲 자동 검수 결과

- [ ] block 없음
- [ ] warn ≤ 허용치 (로그만 존재)

### 🔲 육안 체크 (중요)

- [ ] "홍보/마케팅/성공" 단어 없음
- [ ] 감탄사/느낌표 남용 없음
- [ ] "지금 하세요" 압박 없음

**📌 기준**

읽고 나서 조용한가?

**조용하면 통과.**

---

## T-24h : 자동화 실제 동작 확인

### 🔲 자동 블로그

- [ ] 실제 글 1건 생성
- [ ] 톤 가이드 위반 없음
- [ ] 날짜/지역 정상

### 🔲 공유 카드

- [ ] 이미지 없어도 카드 생성
- [ ] CTA 문구 없음
- [ ] 날짜/기록성 명확

---

## T-18h : 알림 & 침묵 설계 점검

- [ ] Day 0~2 알림 없음
- [ ] 첫 알림은 안심 메시지
- [ ] CTA 알림 ≤ 1회

**📌 알림이 많아 보이면 ❌**

---

## T-12h : Pro 흐름 검증

- [ ] Pro 미사용도 기능 붕괴 없음
- [ ] Pro 문구에 손실/압박 표현 없음
- [ ] 해지 안내 명확

---

## T-6h : 실제 사용자 시뮬레이션 (10분)

### 시나리오

1. 공유 카드 클릭
2. 팀 페이지 스크롤
3. 블로그 1개 클릭
4. 아무 행동 없이 나가기

### 체크

- [ ] 불편함 없음
- [ ] 압박 없음
- [ ] "광고 같다" 느낌 없음

---

## T-1h : 마지막 질문 (이것만)

**이 서비스는 지금보다 더 말을 하고 싶은가?**

- **NO** → ⭕ 배포
