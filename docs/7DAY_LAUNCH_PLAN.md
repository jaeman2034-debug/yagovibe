# 기록장 — 7일 실행 플랜 (던지고 관찰)

## 원칙 (다시 한 번)

- ❌ 설득하지 않는다
- ❌ 고치지 않는다
- ⭕ 켜고, 두고, 본다

---

## Day 1 — 켜기 (30분)

### 할 일
- [ ] 도메인 연결
- [ ] 메인/팀 페이지 공개
- [ ] 자동 기록·공유 ON

### 체크
- [ ] 빈 데이터 섹션 숨김 OK
- [ ] 공유 카드 생성 OK

**오늘은 아무에게도 알리지 않는다**

---

## Day 2 — 최소 노출 (10분)

### 할 일
- [ ] 1곳만 공유 (가장 조용한 곳)
- [ ] 예: 지역 밴드 게시글 1개

**문구 템플릿**:
```
팀 활동 기록을 남기고 있습니다.
필요하신 분만 참고하세요.
```

### 금지
- ❌ 댓글 대응
- ❌ 추가 공유

---

## Day 3 — 자동 기록 확인 (10분)

### 할 일
- [ ] 자동 글 1건 생성 확인
- [ ] 톤 위반만 체크 (수정 ❌)

### 성공 신호
**"아무것도 안 했는데 생김"**

---

## Day 4 — 공유 유입 관찰 (5분)

### 볼 것
- [ ] 공유 유입 있/없
- [ ] 블로그 클릭 있/없

### 하지 말 것
- ❌ 수치 비교
- ❌ 전환 판단

---

## Day 5 — 침묵 유지 (0분)

**아무것도 하지 않는다.**

**불안하면 성공 중**

---

## Day 6 — 관리자 관점 확인 (10분)

### 할 일
- [ ] 관리자 대시보드 1회 접속
- [ ] 상태 문구 읽기만

### 질문
**"설명하고 싶어졌나?"**

- YES → 문구 과다 (메모만)
- NO → 정상

---

## Day 7 — 판단은 하나만

### 질문
**"기록이 생겼는가?"**

- YES → 유지
- NO → 자동화 점검만 (UX/카피 손대지 않기)

---

## 📌 7일 동안 보는 것 / 안 보는 것

### 본다
- 기록 생성 여부
- 공유 발생 여부
- 관리자 재방문 여부

### 안 본다
- 전환율
- 성장률
- "잘 되는지" 감정

---

## 🚨 실패가 아닌 신호

- 문의 없음
- 조용함

**이건 설계 의도 그대로**
