### 화면 구성 (상단 → 하단 순서)

#### 1. 헤더 영역 (최상단)
```

│  노원구축구협회 운영 현황                        │

**설명**:

- 날짜는 "기준"으로 표시 (IT 용어인 "스냅샷", "실시간" 금지)

### 화면 디자인 원칙

1. **색상**: 
   - 배경: 흰색 또는 연한 회색
   - 강조: 협회 로고 색상 (파란색 계열 추천)
   - 경고: 미납, 비활동은 연한 빨간색

2. **폰트**:
   - 숫자: 크고 굵게 (최소 24px)
   - 라벨: 작고 가늘게 (12-14px)
   - 요약 문장: 읽기 쉬운 크기 (16px)

3. **레이아웃**:
   - 카드 간격: 충분히 넓게 (20px 이상)
   - 여백: 넉넉하게 (모바일에서도 읽기 쉽게)
   - 정렬: 왼쪽 정렬 (중앙 정렬 금지)

## 📊 [요청 작업 2] UX 흐름 설계

### 데이터 자동 수집 흐름

```
[1단계] 클럽 활동 데이터 자동 수집
   ↓
   - 클럽이 대회 참가 신청 시 자동 기록
   - 회원 등록 시 자동 기록
   - 회비 납부 시 자동 기록
   ↓
[2단계] 대회 데이터 자동 수집
   ↓
   - 대회 개최 시 자동 기록
   - 참가 신청 시 자동 기록
   - 결과 입력 시 자동 기록
   ↓
[3단계] 회원 데이터 자동 수집
   ↓
   - 회원 가입 시 자동 기록
   - 회원 정보 변경 시 자동 기록
   - 활동 기록 자동 수집
   ↓
[4단계] 회계 데이터 자동 수집
   ↓
   - 회비 납부 시 자동 기록
   - 지출 승인 시 자동 기록
   - 정산 완료 시 자동 기록
   ↓
[5단계] AI 자동 요약 생성
   ↓
   - 매일 자정에 전날 데이터 수집
   - AI가 핵심 지표 추출
   - 자연어 문장으로 요약 생성
   ↓
[6단계] 리포트 화면 자동 업데이트
   ↓
   - 협회 임원이 화면 열면 최신 데이터 표시
   - 별도 "새로고침" 버튼 없음
   - 자동으로 최신 상태 유지
```

### 협회가 해야 할 일 (최소화)

**거의 없음:**
- ✅ 화면 열기만 하면 됨
- ✅ 필요시 "전체 클럽 보기" 클릭

**가끔:**
- ⚠️ 대회 결과 입력 (대회 종료 후 1회)
- ⚠️ 회비 미납자 확인 (월 1회)

**절대 하지 않아도 됨:**
- ❌ 데이터 입력
- ❌ 리포트 작성
- ❌ 통계 계산
- ❌ 그래프 그리기

---

### 자동화 포인트

1. **클럽 활동 자동 감지**
   - 클럽이 대회 참가 신청 → 자동으로 "활동 중" 상태로 변경
   - 3개월 이상 활동 없음 → 자동으로 "비활동" 상태로 변경

2. **회원 수 자동 집계**
   - 회원 가입/탈퇴 시 실시간 반영
   - 카테고리별(성인/실버/유소년) 자동 분류

3. **회비 납부율 자동 계산**
   - 납부 완료 시 자동 반영
   - 목표 대비 자동 비교

4. **대회 데이터 자동 수집**
   - 대회 개최 시 자동 기록
   - 참가 신청 시 자동 집계
   - 결과 입력 시 자동 반영

5. **AI 요약 자동 생성**
   - 매일 자정에 전날 데이터 분석
   - 핵심 지표 추출
   - 자연어 문장으로 요약 생성
   - 작년 대비, 목표 대비 자동 비교

## 🔄 [요청 작업 4] 확장 구조 설명

### 복제 가능한 구조

#### 현재: 노원구축구협회 (구·군 단위)

```
노원구축구협회
  ├─ 산하 클럽 24개
  ├─ 회원 1,247명
  ├─ 대회 8개/년
  └─ 리포트 화면 1개
```

---

#### 확장 1: 다른 구·군 축구협회

**구조 복제:**
```
강남구축구협회
  ├─ 산하 클럽 (자동 수집)
  ├─ 회원 (자동 수집)
  ├─ 대회 (자동 수집)
  └─ 리포트 화면 (동일 구조, 데이터만 다름)
```

**차이점:**
- 클럽 수, 회원 수만 다름
- 화면 구조는 동일
- AI 요약 문장만 자동으로 다르게 생성

**복제 방법:**
1. 노원구 리포트 화면을 템플릿으로 저장
2. 새로운 구·군 협회 생성 시 템플릿 복제
3. 데이터만 자동 연결
4. 화면은 즉시 사용 가능

---

#### 확장 2: 서울시축구협회 (시·도 단위)

**구조 변경:**
```
서울시축구협회
  ├─ 산하 구·군 협회 (노원구, 강남구, ...)
  │   ├─ 각 구·군의 클럽 수
  │   ├─ 각 구·군의 회원 수
  │   └─ 각 구·군의 대회 수
  ├─ 전체 집계
  │   ├─ 총 클럽 수 (모든 구·군 합산)
  │   ├─ 총 회원 수 (모든 구·군 합산)
  │   └─ 총 대회 수 (모든 구·군 합산)
  └─ 리포트 화면 (구·군별 + 전체 집계)
```

**화면 구성:**
```
┌─────────────────────────────────────────┐
│ 서울시축구협회 운영 현황                  │
│                                          │
│ [구·군별 현황 카드들]                    │

│                                          │
│ [전체 집계]                              │

│ 총 회원: 8,234명                         │
│                                          │
│ [AI 요약]                                │
│ 서울시 전체 축구협회 현황...              │
└─────────────────────────────────────────┘

**데이터 흐름:**
```

```

### 복제 가능한 핵심 구조

#### 2. 리포트 생성 계층 (템플릿 기반)

```
데이터 수집
   ↓
템플릿 적용 (구·군 / 시·도 / 중앙)
   ↓
AI 요약 생성
   ↓
화면 표시
```

**템플릿만 다르고, 로직은 동일**

---

#### 3. 화면 구성 계층 (컴포넌트 재사용)

```
[핵심 지표 카드] 컴포넌트
   ↓ (재사용)
구·군 리포트
시·도 리포트
중앙 리포트
```

**컴포넌트는 동일, 데이터만 다름**

---

### 확장 시 추가 작업

**최소 작업:**
1. 새로운 협회 생성 (구·군 / 시·도 / 중앙 선택)
2. 템플릿 자동 적용
3. 데이터 연결
4. 완료

**코드 변경 불필요:**
- 리포트 생성 로직은 동일
- 화면 구성은 템플릿 기반
- AI 요약은 자동 생성
