## 🗓️ DAY 1 — 📝 첫 자동 글 생성 (와우 포인트)

### 액션

1. **팀 블로그 탭 → ✨ 첫 글 생성 버튼 클릭**
2. **5초 침묵** (운영자 반응 기다리기)
3. **운영자 반응 체크**

### 예상 반응

**✅ 성공 신호:**
- "이거 누가 쓴 거예요?"
- "우리 팀 정보 맞네요?"
- "와, 이거 좋네요"

**→ 둘 중 하나 나오면 성공**

### 절대 하지 말 것

- ❌ 프롬프트 설명
- ❌ 기술 설명
- ❌ "AI가..." 같은 기술 용어 남발

### 대응 멘트 (반응별)

**"이거 누가 쓴 거예요?"**
```text
팀 정보랑 활동 데이터 기반으로,
방금 버튼 한 번 누른 걸로,
AI가 자동으로 쓴 겁니다.
저는 아무 글자도 안 쳤어요.
```

**"우리 팀 정보 맞네요?"**
```text
네, 맞습니다.
팀 대시보드에 입력하신 정보를 기반으로
자동으로 생성된 거예요.
```

## 🗓️ DAY 2 — 🌍 공개 페이지 확인

### 액션

1. **공개 페이지 URL 열기**
2. **"외부에서 보이는 화면" 보여주기**

### 멘트

```text
이건 이제
검색이나 공유로
누구나 볼 수 있어요.

이 링크 하나로
- 신규 회원 모집
- 팀 소개
- 활동 홍보

모두 해결됩니다.
```

### 👉 핵심
- **홍보 가치 인식** 유도
- "링크 하나로 해결" 프레임

## 🗓️ DAY 3 — ⚽ 활동 후기 자동 생성

### 조건

- 일정/출석 **1회만** 있어도 OK
- 사진 있으면 베스트 (Pro 기능)

### 액션

1. **일정 완료 확인**
2. **출석 체크 확인**
3. **활동 후기 자동 생성 확인**
4. **생성된 후기 링크 전달**

### 멘트

```text
[오늘 모임 활동 후기 자동 생성 완료 ✅]

운영자님, 오늘 모임 기준으로
AI가 이런 활동 후기를 자동으로 써놨습니다. 👇

👉 {후기 링크}

이제부터는
활동만 하시면
후기는 자동으로 쌓입니다.

운영자님이 따로 후기 쓰실 필요 없이,
이걸 그대로 공유만 하셔도 됩니다. 🙂
```

### 👉 핵심
- **운영 스트레스 제거** 인식 유도
- "이제부터는 자동" 프레임

## 🗓️ DAY 4 — 📊 통계 한 줄 노출

### 보여줄 것

```text
📊 이번 주 팀 블로그 조회수: {조회수}회

이 중 {외부 방문자}명은 외부에서 처음 들어온 방문자입니다.
```

### 멘트

```text
운영 안 하셨는데
이 정도 반응이 나왔어요.

운영자님이 일부러 홍보 안 하셔도,
이 블로그가 쌓이면 쌓일수록
"우리 팀이 살아 있는 팀이다"는 신뢰가 계속 올라가요.
```

### 👉 핵심
- **Pro 전환 씨앗** 심기
- "운영 안 했는데 반응" 프레임

## 🗓️ DAY 5 — 🤖 주간 요약 (Pro 체험 핵심)

### 자동 생성

- "이번 주 이렇게 활동했습니다" 자동 생성
- 주간 요약 링크 전달

### 멘트

```text
[이번 주 팀 활동 요약이 자동으로 만들어졌습니다 ✅]

이번 한 주 동안의
- 모임 횟수
- 출석 상황
- 주요 활동

을 AI가 이렇게 정리해 줍니다. 👇

👉 {주간 요약 링크}

이 요약은 **Pro 플랜에서만 매주 자동으로 제공**됩니다.
"이번 주 우리 팀, 뭐했는지" 한 번에 정리할 수 있어서
운영자님들이 제일 좋아하시는 부분이에요.
```

### 👉 핵심
- **차이점 명확화** (Free vs Pro)
- "이 기능은 Pro에서 계속" 프레임

## 🗓️ DAY 7 — 🚀 전환 or 예약

### CTA

```text
🚀 Pro 계속 사용하기 – 월 19,000원

(언제든 해지 가능, 카드 1번 등록)
```

### 결과 케이스별 대응

#### ✅ 즉결 결제

**멘트:**
```text
감사합니다, 운영자님. 🙏

이제 [팀명] 블로그는
AI가 계속 알아서 관리하는 상태가 되었습니다.

앞으로 쓰시다가
- 불편한 점
- 이런 것도 자동으로 됐으면 좋겠다 싶은 점

생기시면,
"고객"이 아니라 **파트너**라고 생각하시고
그냥 편하게 말씀해 주세요.
```

#### ⭕ 다음 달 결제 예약

**멘트:**
```text
네, 그럼 다음 달에 다시 연락드릴게요.

그동안은 Free 플랜으로
- 팀 소개 글
- 활동 후기 (월 2회)

까지는 계속 쓰실 수 있으니까,
편하게 써보시면서
"이 정도면 Pro로 올려볼 만하다" 싶으실 때
그때 결정하셔도 전혀 늦지 않습니다.
```

#### ⭕ "조금 더 써볼게요"

**멘트:**
```text
네, 그게 제일 좋은 접근인 것 같아요.

지금 단계에서는
"설명 듣고 바로 결제"보다는,

그냥 좀 더 써보시고
- 이게 있으면 편한지,
- 없어지면 아쉬운지

그 감만 한 번 느껴보시면 됩니다. 🙂

나중에라도,
"이거 없으면 다시 귀찮겠다"는 느낌이 생기면,
그때 편하게 Pro를 고민하셔도 전혀 늦지 않습니다.
```

### 👉 핵심
- **관계 유지** (거절해도)
- "나중에" 프레임 유지
- 파트너 관계 전환 (전환 성공 시)
