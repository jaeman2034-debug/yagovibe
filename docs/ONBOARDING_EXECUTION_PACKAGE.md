## ① 첫 팀 맞춤 AI 첫 글 프롬프트 (운영용)

### 프롬프트 템플릿

```text
너는 지역 스포츠 팀의 공식 홍보 블로그를 운영하는 AI다.
목표는 신뢰를 주고, 참여를 유도하는 것이다.

입력 데이터:
- 팀명: {{teamName}}
- 종목: {{sportType}}
- 지역: {{location}}
- 연령대: {{ageGroup}}
- 멤버 수: {{memberCount}}
- 분위기 키워드: {{toneKeywords}} (예: "친근함", "열정", "건강", "즐거움")

출력 형식:
1) 제목: SEO 친화적, 질문형 (예: "[팀명]은 어떤 팀인가요?")
2) 본문: 3~5문단 소개글
   - 첫 문단: 팀 인사 + 핵심 정체성
   - 중간 문단: 활동 지역, 종목, 연령대, 멤버 구성
   - 마지막 문단: 팀의 철학/목표
3) "이런 분께 추천합니다!" 섹션: 불릿 3~5개
4) 하단 참여 CTA: 1줄 (예: "현재 [팀명]은 새로운 멤버를 모집 중입니다. 관심 있으시다면 아래 버튼을 통해 언제든지 참여를 신청해 주세요!")

톤 규칙:
- 과장 금지: "최고", "최강" 같은 표현 지양
- 진정성: 실제 팀의 모습을 솔직하게
- 지역/사람 중심: 지역 커뮤니티와 사람 간의 유대감 강조
- 친근함: 거리감 없는, 편안한 말투

완료 기준: 운영자가 수정 안 해도 바로 공개 가능한 수준.
```

## ② 첫 팀 접촉 DM/게시글 원문 (A/B 2안)

### A안: 운영자 공감형

```text
안녕하세요 운영자님.

팀 공지/후기 관리 때문에
항상 손이 많이 가시죠?

저희는 팀 활동만 하면
AI가 자동으로

- 팀 소개
- 활동 후기
- 주간 요약

까지 대신 만들어주는 서비스를 운영 중입니다.

초기 팀 대상으로
카드 등록 없는 무료 체험 열어드려요.

필요하시면 바로 링크 드릴게요 🙂
```

**사용 시점:**
- 네이버/다음 카페 게시글
- 1:1 DM 첫 접촉
- 운영자가 "공지 관리가 귀찮다"고 언급한 경우

---

### B안: 결과 제시형

```text
운영자님 안녕하세요.

팀 블로그/공지,
요즘은 AI가 대신 관리합니다.

활동만 하면

- 팀 소개 글 자동 생성
- 후기 자동 포스팅
- 공개 홍보 페이지 운영

초기 체험 팀을 찾고 있어요.

관심 있으시면 바로 써보실 수 있게 도와드릴게요.
```

**사용 시점:**
- SNS 홍보 (트위터/X, 인스타그램)
- 빠른 관심 유도가 필요한 경우
- "결과" 중심으로 설명하고 싶을 때

### 게시글 본문 (네이버/다음 카페용)

```text
안녕하세요, 동호회 회장님/총무님들께만 도움이 될 만한 서비스라 조심스럽게 글 남깁니다.

혹시 이런 고민 있으신가요?

- 경기/모임 끝나고 후기 글 쓰는 게 귀찮다
- 신규 회원 모집하려고 글 좀 써야 하는데, 항상 밀린다
- 공지/후기를 단톡방, 카페, 밴드에 매번 따로 써야 해서 번거롭다

저희가 만든 서비스는,

👉 "팀 활동만 하면, AI가 대신 팀 블로그를 운영해주는 서비스" 입니다.

- 팀 소개 글 자동 생성
- 매 모임/경기 후 활동 후기 자동 작성
- 매주 주간 활동 요약 글 자동 생성 (Pro)

현재 **초기 10개 팀을 대상으로 14일 무료 Pro 체험**을 열어두고 있습니다.
카드 등록도 필요 없고, 체험이 끝나면 자동으로 Free로 돌아갑니다.

대신 조건은 딱 하나입니다.

> "써보시고 좋았던 점/아쉬웠던 점을 솔직히 알려주세요."

관심 있으신 회장님/총무님은 댓글로
`[팀명 / 종목 / 연령대 / 활동 지역]` 정도만 남겨주시면
1:1로 자세히 안내드리겠습니다. 🙂
```
