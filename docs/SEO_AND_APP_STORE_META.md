## 2️⃣ SEO 디스크립션 (Description)

### 홈페이지 메인

```
팀 활동을 자동으로 기록하고 공개하는 서비스입니다. 홍보나 모집 없이, 활동만 있으면 자동으로 기록이 쌓이고 자연스럽게 공유됩니다. 주 1회 이상 모이는 팀에게 추천합니다.
```

**특징:**
- 핵심 기능 설명
- 차별점 명시
- 타겟 명확

---

### 팀 블로그 페이지

```
[팀명]의 활동 기록을 확인하세요. [지역]에서 활동하는 [종목] 팀으로, [연령대] 분들이 주 [빈도] 모입니다. 최근 활동 기록과 팀 정보를 확인할 수 있습니다.
```

**예시:**
```
소흘 60대 FC의 활동 기록을 확인하세요. 포천에서 활동하는 축구 팀으로, 60대 분들이 주 1회 모입니다. 최근 활동 기록과 팀 정보를 확인할 수 있습니다.
```

---

### 관리자 페이지

```
[팀명] 기록장 관리 페이지입니다. 팀 활동 기록을 확인하고, 공개 설정을 관리할 수 있습니다.
```
