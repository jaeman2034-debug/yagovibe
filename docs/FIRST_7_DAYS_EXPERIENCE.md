# 📅 첫 7일 경험 시나리오

## 📋 목적
아무 일도 안 일어나는데, 신뢰가 쌓이는 흐름

---

## 🎯 원칙

### 경험 목표

- 자연스러운 신뢰 형성
- 침묵도 신뢰의 일부
- 과도한 알림 없음
- 기록이 쌓이는 증거

---

### 톤 가이드

- 담백함
- 과장 없음
- 신뢰감
- 편안함

---

## 0️⃣ 설계 원칙 (절대 규칙)

- ❌ 환영 메일 폭탄
- ❌ 기능 설명 알림
- ❌ "지금 해보세요" 유도
- ⭕ 침묵이 기본값
- ⭕ 알림은 확인용, 안심용만

---

## 첫 7일 경험의 핵심

**"아무 일도 안 일어나는데, 신뢰가 쌓이는 흐름"**

- ❌ 과도한 알림
- ❌ 강요하는 행동
- ⭕ 자연스러운 기록 쌓임
- ⭕ 조용한 증거

---

## Day 0 — 온보딩 종료 직후

### 사용자 상태

- 설정 완료
- "이제 할 일 없음" 상태

### 시스템 동작

- 자동화 상태 정상 체크
- 다음 기록 예상일 계산

### 사용자 노출

```
❌ 알림 없음
❌ 메일 없음
```

**📌 의도적 침묵**
**→ 신뢰의 시작**

### 신뢰 형성

- "설정이 끝났다"는 안심감
- "이제 할 일 없다"는 확신

---

## Day 1 — 아무 일 없음 (중요)

### 시스템

- 백그라운드에서 활동 데이터 대기

### 사용자

- 서비스 잊어도 됨

**📌 성공 신호**
**사용자가 "아, 맞다"라고 생각 안 함**

### 신뢰 형성

- "아직 아무 일도 안 일어났지만 괜찮다"
- "기다리면 될 것 같다"

---

## Day 2 — 첫 안심 신호 (조건부)

### 조건

- 최근 7일 내 활동 데이터 있음

### 알림 (1줄, 선택 채널)

```
🟢 기록장이 정상 작동 중입니다
최근 활동이 자동으로 대기 중이에요.
```

**❌ CTA 없음**
**❌ 링크 없음**

**📌 기계적 확인 메시지**

---

## Day 3 — 자동 기록 생성 (핵심)

### 시스템

- 자동 블로그 글 1건 생성
- 공유 카드 1장 생성
- 팀 페이지 업데이트

### 사용자 노출 (1개만)

```
📝 새로운 팀 기록이 자동으로 남겨졌습니다
```

**버튼:**
```
[ 기록 보기 ] (선택)
```

**📌 "아무것도 안 했는데 생김" 경험**

### 신뢰 형성

- "아, 진짜 자동으로 되네"
- "내가 안 했는데 기록이 생겼다"

---

## Day 4 — 외부 반응 발생 (조건부)

### 조건

- 공유 유입 발생 OR
- 방문자 5명 이상

### 알림 (관리자만)

```
👀 누군가 팀 기록을 보고 있습니다
(지난 24시간 5명 방문)
```

**❌ 숫자 과장 ❌**
**❌ 행동 유도 ❌**

**📌 관찰의 관찰**

---

## Day 5 — 완전 침묵

### 알림

```
❌ 알림 없음
❌ 메일 없음
❌ 노출 없음
```

**📌 시스템에 대한 신뢰 강화**

### 신뢰 형성

- "계속 쌓이고 있구나"
- "내가 안 해도 되네"

---

## Day 6 — 주간 요약 (옵션 ON인 경우만)

### 제목

```
이번 주, 기록 요약
```

### 내용 (3줄 고정)

```
팀 기록 1건
방문자 12명
공유 1회
```

### 하단 문구

```
이 흐름은 자동으로 유지됩니다.
```

**📌 해석 없음, 판단 강요 없음**

### 신뢰 형성

- "뭔가 계속 돌아가고 있구나"
- "신경 쓰지 않아도 되네"

---

## Day 7 — Pro 전환 가능성 노출 (조건부)

### 조건 (1개 이상)

- 관리자 페이지 2회 방문
- 주간 요약 열람
- 공유 발생

### 노출 위치

- 관리자 대시보드 하단 카드

### 문구

```
📈 기록이 잘 쌓이고 있습니다
이 흐름을 계속 유지할 수 있어요.
```

**CTA:**
```
[ Pro 알아보기 ] (작게)
```

**📌 처음이자 마지막 '제안'**

### 신뢰 형성

- "일주일 동안 알아서 쌓였구나"
- "이제 정말 신뢰할 수 있겠다"

## 9️⃣ 침묵의 가치

### 침묵이 신뢰를 만드는 순간

**DAY 1-2:**
- 아무 일도 안 일어남
- 하지만 불안하지 않음
- "기다리면 될 것 같다"

**DAY 4-5:**
- 알림 없음
- 하지만 기록은 쌓임
- "조용히 돌아가고 있구나"

---

### 침묵의 원칙

- 불필요한 알림은 신뢰를 해침
- 조용한 증거가 더 강력함
- 사용자가 찾아오는 것이 이상적
