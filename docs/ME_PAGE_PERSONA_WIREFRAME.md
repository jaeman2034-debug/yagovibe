# 🔥 마이페이지 Persona별 와이어프레임 (STEP 2)

## 📐 3-Layer 고정 구조

```
┌─────────────────────────────────────────┐
│ ① Identity Layer (항상 있음)            │
│ - 프로필 카드                            │
│ - 활동 요약 (팀 수 / 대회 수 / 기록 수)  │
│ - 설정 / 로그아웃                        │
├─────────────────────────────────────────┤
│ ② Persona Layer (핵심 차별 영역)        │
│ - P0/P1/P2/P3/P4별로 다름                │
├─────────────────────────────────────────┤
│ ③ Opportunity Layer (유도 영역)         │
│ - 아직 안 했지만 할 수 있는 것           │
└─────────────────────────────────────────┘
```

## 🔧 Persona 감지 로직

**컴포넌트**: `<MePersonaResolver />`

```typescript
function resolvePersona(user: User, data: {
  hasTeam: boolean;
  isTeamCaptain: boolean;
  isAssociationAdmin: boolean;
  hasApplications: boolean;
  profileComplete: boolean;
}): Persona {
  // P4: 협회 관리자 (최우선)
  if (data.isAssociationAdmin) {
    return 'P4';
  }
  
  // P3: 팀장
  if (data.hasTeam && data.isTeamCaptain) {
    return 'P3';
  }
  
  // P2: 팀 소속 선수
  if (data.hasTeam && !data.isTeamCaptain) {
    return 'P2';
  }
  
  // P1: 개인 체육인 (팀 없음, 의도적)
  if (!data.hasTeam && data.profileComplete) {
    return 'P1';
  }
  
  // P0: 완전 신규
  return 'P0';
}
```
