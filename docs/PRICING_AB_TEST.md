## 🎯 A/B 테스트 설계

### A안: 현재 가격 (19,000원/월)

**가격 표시:**
```text
월 19,000원 / 팀

(연간 결제 시 월 15,000원)
```

**지원 문구:**
```text
커피 한 잔 값으로 AI가 팀 홍보를 대신 관리합니다.
```

**CTA:**
```text
🚀 Pro 시작하기 – 월 19,000원
```

---

### B안: 낮춘 가격 (14,900원/월)

**가격 표시:**
```text
월 14,900원 / 팀

(연간 결제 시 월 12,000원)
```

**지원 문구:**
```text
하루 500원으로 AI가 팀 홍보를 대신 관리합니다.
```

**CTA:**
```text
🚀 Pro 시작하기 – 월 14,900원
```

## 📊 테스트 지표

### 1차 지표 (결정적)
- **전환율**: 체험 → Pro 전환 비율
- **결제 완료율**: Checkout 시작 → 결제 완료 비율

### 2차 지표 (참고)
- **체험 시작률**: DM/게시글 → 체험 시작 비율
- **체험 완료율**: 체험 시작 → 14일 완료 비율
- **평균 체험 기간**: 체험 시작 → 전환/종료까지 기간

## 🎯 테스트 시나리오

### 시나리오 1: 동일 채널 A/B 테스트

**방법:**
- 네이버/다음 카페 게시글 2개 버전 작성
- 각각 다른 가격으로 제시
- 동일 시간대에 게시
- 전환율 비교

**예상 결과:**
- A안 (19,000원): 전환율 낮지만, 수익성 높음
- B안 (14,900원): 전환율 높지만, 수익성 낮음

---

### 시나리오 2: 순차 테스트

**방법:**
- 첫 5팀: A안 (19,000원) 제시
- 다음 5팀: B안 (14,900원) 제시
- 전환율 비교

**예상 결과:**
- A안: 1~2팀 전환
- B안: 2~3팀 전환

## 🎯 예상 결과 및 대응

### 케이스 1: A안이 더 좋음 (19,000원)

**조건:**
- A안 전환율 ≥ B안 전환율
- A안 수익성 > B안 수익성

**대응:**
- 현재 가격 유지
- "커피 한 잔 값" 프레임 강화

---

### 케이스 2: B안이 더 좋음 (14,900원)

**조건:**
- B안 전환율 > A안 전환율
- B안 수익성 ≥ A안 수익성 (전환율 차이로 상쇄)

**대응:**
- 가격 조정 검토
- "하루 500원" 프레임 강화

---

### 케이스 3: 비슷함

**조건:**
- A안 전환율 ≈ B안 전환율
- 수익성 차이 미미

**대응:**
- 현재 가격 유지 (19,000원)
- 추가 테스트 필요 (연간 결제 할인율 조정 등)

## 📋 최종 결정 기준

**가격 결정 시 고려사항:**
1. 전환율 최대화
2. 수익성 최대화 (가격 × 전환율)
3. 운영자 심리적 장벽 최소화
4. 경쟁 서비스 대비 경쟁력

**결정 시점:**
- 초기 10팀 테스트 완료 후
- 최소 20팀 데이터 수집 후 (통계적 유의성)

## 🎯 운영 원칙

- **데이터 기반 결정** → 감이 아닌 숫자로
- **지속적 테스트** → 가격은 고정이 아님
- **운영자 피드백 반영** → 가격도 제품의 일부
