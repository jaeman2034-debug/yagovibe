# ✍️ AI 자동 생성 블로그 글 템플릿 & 톤 가이드

## 📋 목적
노인 스포츠 팀에 맞는 인간적이고 따뜻한 AI 글 생성 가이드

---

## 1️⃣ 글 유형 분류 (자동 선택)

### PostType 정의

```typescript
type PostType =
  | "match_recap"      // 경기 후기
  | "practice_log"    // 훈련/모임 기록
  | "member_story"    // 멤버 이야기
  | "health_tip"      // 건강/스트레칭 팁
  | "intro"           // 팀 소개 (첫 글)
  | "weekly"          // 주간 요약 (Pro 전용)
```

**자동 선택 로직:**
- 최근 경기 데이터 있음 → `match_recap`
- 최근 훈련/모임 데이터 있음 → `practice_log`
- 멤버 생일/기념일 → `member_story`
- 건강 관련 데이터 → `health_tip`
- 첫 글 → `intro`
- 주간 요약 → `weekly`

## 3️⃣ 글 유형별 템플릿

### 템플릿 A: 팀 소개 글 (첫 글)

**목적:**
- 팀 정체성 전달
- 신뢰 구축
- 가입 유도

**구조:**
```
1. 제목 (SEO 친화, 질문형)
2. 팀 소개 (3~4문단)
3. 활동 내용
4. 이런 분을 기다립니다
5. 가입 CTA
```

**톤:**
- 따뜻하고 진정성 있게
- 과장 없이
- 지역/사람 중심

---

### 템플릿 B: 경기 후기 (match_recap)

**목적:**
- 활동성 증명
- 팀 분위기 전달
- 재방문 유도

**제목 규칙:**
```
[날짜] ○월 ○일 정기 경기 기록
```

**본문 구조:**
```
1. 오늘의 한 줄 요약
   → 예: 무더운 날씨였지만, 큰 탈 없이 잘 마무리했습니다.

2. 경기 분위기
   → 날씨, 컨디션, 참여 인원 간단 언급

3. 기억에 남는 장면
   → 골/승패 ❌
   → 웃음, 배려, 소통 위주

4. 마무리
   → 다음 일정 또는 감사 인사
```

**톤:**
- 담백한 서술형
- 결과보다 과정 중심
- 구체적인 에피소드 (웃음, 배려, 소통)

**예시 문단:**
```
무더운 날씨였지만, 모두 각자 페이스를 지키며 경기를 진행했습니다.
중간중간 휴식을 충분히 가지며 큰 무리 없이 마무리할 수 있었습니다.
```

---

### 템플릿 C: 훈련/모임 기록 (practice_log)

**제목:**
```
[모임 기록] ○월 ○일 함께 땀 흘린 하루
```

**본문 포인트:**
- 준비 운동 언급 필수
- "부상 없이" 문구 포함
- 개인 실력 평가 ❌

---

### 템플릿 D: 멤버 이야기 (member_story)

**제목:**
```
[팀 이야기] 함께한 지 ○개월
```

**구조:**
```
1. 간단 소개 (이니셜 or 별칭)
2. 팀에 오게 된 계기
3. 요즘 느끼는 점
```

**규칙:**
- 실명 ❌
- 사진 ❌ (프라이버시)
- 미담 과장 ❌

---

### 템플릿 E: 건강/스트레칭 팁 (health_tip)

**제목:**
```
[건강 기록] 요즘 저희가 신경 쓰는 것
```

**내용 가이드:**
- 의학 조언 ❌
- "저희는 이렇게 합니다" 관점
- 출처 언급 ❌ (전문가 흉내 금지)

---

### 템플릿 F: 주간 요약 (weekly - Pro 전용)

**목적:**
- 지속적인 활동 증명
- 팀 활성도 강조
- SEO 콘텐츠

**구조:**
```
1. 제목 (이번 주 활동 요약)
2. 주요 활동 (불릿 포인트)
3. 멤버 소식
4. 다음 주 일정
```

**톤:**
- 간결하고 명확하게
- 정보 중심
- 신뢰감 있게

## 4️⃣ 안전장치 (AI 자동 필터)

### 자동 제거 키워드

**금지 키워드:**
- "최고", "전설", "감동", "눈물"
- 과도한 승패 표현
- 의학적 조언 표현

**필터 로직:**
```typescript
const forbiddenKeywords = [
  "최고", "전설", "감동", "눈물",
  "불타는", "열정", "전설의"
];

function filterContent(content: string): string {
  let filtered = content;
  forbiddenKeywords.forEach(keyword => {
    filtered = filtered.replace(new RegExp(keyword, 'g'), '');
  });
  return filtered;
}
```

---

### 자동 삽입 키워드

**권장 키워드:**
- "무리하지 않고"
- "각자 컨디션에 맞게"
- "꾸준히"
- "부상 없이"

**삽입 로직:**
```typescript
function enhanceContent(content: string): string {
  // 적절한 위치에 안전 키워드 삽입
  if (!content.includes("무리하지 않고")) {
    // 적절한 위치에 삽입
  }
  return content;
}
```

## 7️⃣ 톤 가이드 (노인 스포츠 팀 특화)

### 기본 원칙

**✅ 해야 할 것:**
- 존중과 배려의 톤
- 경험과 지혜 인정
- 건강과 즐거움 강조
- 지역 공동체 의식

**❌ 하지 말아야 할 것:**
- 과장된 표현
- 젊은 세대 용어
- 경쟁/승부 중심
- 복잡한 문장

---

### 톤 예시

**좋은 예:**
```
"60대에도 우리는 계속 뛰고 있습니다.
건강하게, 즐겁게, 오래 함께하는 것이 우리의 목표입니다."
```

**나쁜 예:**
```
"최고의 팀! 항상 승리하는 강팀!"
"젊은이들 못지않게 빠르고 강합니다!"
```

### 프롬프트 A: 팀 소개 글

```
너는 지역 스포츠 팀의 홍보 블로그를 전문으로 작성하는 AI다.

다음 정보를 바탕으로 팀 소개 블로그 글을 작성해라.

[팀 정보]
- 팀명: {teamName}
- 종목: {sportType}
- 연령대: {ageGroup}
- 활동 지역: {location}
- 팀 성격: {toneKeywords}
- 멤버 수: {memberCount}

[작성 규칙]
- 제목 포함 (SEO 친화, 질문형)
- 부드럽고 따뜻한 톤
- 너무 홍보 같지 않게
- 마지막 문단에 자연스러운 가입 유도 문장 포함
- 길이: 600~900자
- 존중과 배려의 톤 유지
- 경쟁보다 건강과 즐거움 강조

[톤 가이드]
- 과장 금지
- 진정성
- 지역/사람 중심
- 노인 스포츠 팀에 맞는 따뜻한 표현
```

---

### 프롬프트 B: 경기 후기

```
너는 지역 스포츠 팀의 활동 후기를 작성하는 AI다.

다음 정보를 바탕으로 경기 후기 블로그 글을 작성해라.

[경기 정보]
- 날짜: {date}
- 이벤트: {event}
- 참여 인원: {participantCount}
- 주요 내용: {summary}
- 인상 깊은 순간: {highlights}

[작성 규칙]
- 제목: "{date} {event} 후기"
- 즐겁고 활기찬 톤
- 구체적인 에피소드 포함
- 긍정적 에너지 전달
- 길이: 400~600자
- 다음 일정 안내 포함

[톤 가이드]
- 승부보다 즐거움 강조
- 멤버 간 소통과 배려 표현
- 건강한 활동 중심
```

```
[주간 활동]
- 주요 활동: {activities}
- 참여 멤버: {members}
- 다음 주 일정: {nextSchedule}

[작성 규칙]

- 간결하고 명확한 톤
- 불릿 포인트로 주요 활동 정리
- 다음 주 일정 안내
- 길이: 300~500자

[톤 가이드]
- 정보 중심
- 신뢰감 있게
- 지속적인 활동 증명
```

## 4️⃣ 실제 글 예시

### 예시 1: 팀 소개 글

**제목:**
```
소흘 60대 FC를 소개합니다 ⚽️
```

**본문:**
```
소흘 지역에서 활동하는 60대 축구 동호회, 소흘 60대 FC입니다.

우리는 건강하게, 즐겁게, 오래 함께하는 것을 목표로 합니다.
승부보다는 팀워크와 건강한 운동을 중시하며, 매주 정기적으로 모여 함께 뛰고 있습니다.

주로 포천 지역의 체육시설에서 활동하며, 연령대가 비슷한 분들과 함께 운동할 수 있어 더욱 즐겁습니다. 
경기 후에는 간단한 식사와 대화를 나누며 친목을 다집니다.

이런 분을 기다립니다:
• 60대 이상 남성
• 축구 경험 무관
• 건강하고 즐겁게 운동하고 싶은 분

함께 뛰며 건강하고 즐거운 시간을 보내고 싶으신 분들의 참여를 기다립니다.
```

---

### 예시 2: 경기 후기

**제목:**
```
7월 21일 정기 경기 후기
```

**본문:**
```
이번 주 일요일, 무더위 속에서도 즐거운 경기를 가졌습니다.

총 15명이 참여한 이번 경기에서는 팀워크가 돋보였습니다.
서로를 배려하며 안전하게 경기를 진행했고, 경기 후에는 시원한 음료를 마시며 대화를 나누는 시간을 가졌습니다.

특히 이번 경기에서는 새로 합류하신 멤버분이 적극적으로 참여하시는 모습이 인상 깊었습니다.
모두가 따뜻하게 환영해주셔서 더욱 즐거운 분위기였습니다.

다음 주 일요일(7월 28일)에도 정기 경기가 예정되어 있습니다.
많은 분들의 참여를 기다립니다!
```

---

### 예시 3: 주간 요약

**제목:**
```
이번 주 활동 요약 (7월 15일~7월 21일)
```

**본문:**
```
이번 주 소흘 60대 FC는 다음과 같이 활동했습니다.

• 정기 경기 1회 (7월 21일, 15명 참여)
• 팀 모임 1회 (경기 후 친목 시간)
• 새 멤버 1명 합류

이번 주에도 건강하고 즐거운 시간을 보냈습니다.
다음 주 일정은 다음과 같습니다:

• 7월 28일(일): 정기 경기 예정

많은 분들의 참여를 기다립니다!
```
