## 30일 유지의 핵심

**"여전히 귀찮지 않게, 계속 쓰게 만드는 구조"**

- ❌ 과도한 알림
- ❌ 강요하는 행동
- ⭕ 자연스러운 기록 쌓임
- ⭕ 조용한 증거

---

## 1️⃣ Week 2 (Day 8-14) — "계속 굴러간다" 인식

### 시스템 동작

- 자동 기록 1회 추가 생성
- 공유 카드 1회 자동 생성
- 팀 페이지 누적 기록 표시 (2→3건)

### 사용자 노출 (1회만)

```
🗂️ 팀 기록이 3건 쌓였습니다
지난 기록들도 그대로 보관되어 있습니다.
```

**📌 목적**
- 지속성 체감
- "사라지지 않는다"는 안심

### 신뢰 형성

- "계속 돌아가고 있구나"
- "내가 신경 쓰지 않아도 되네"

---

## 2️⃣ Week 3 (Day 15-21) — "남들도 본다" 인식

### 조건

- 누적 방문자 ≥ 20명 OR
- 공유 2회 이상

### 관리자 노출 (대시보드, 무알림)

```
👀 이 팀의 기록을
지난 3주간 23명이 확인했습니다.
```

**❌ 푸시 ❌ 메일**
**→ 들어온 사람만 보게 함**

**📌 목적**
- 사회적 증거 최소치 제공

### 신뢰 형성

- "이제 정말 신뢰할 수 있겠다"
- "계속 쓰고 싶다"

---

## 3️⃣ Week 4 (Day 22-30) — "이건 자산이다" 인식

### 시스템 동작

- 자동 기록 누적 4-5건
- 페이지 상단에 월 단위 묶음 표시

### 노출 문구 (페이지 상단, 작게)

```
📅 7월 기록 4건이 보관되었습니다
```

**📌 목적**
- 기록을 콘텐츠가 아닌 자산으로 인식

### 신뢰 형성

- "한 달 동안 알아서 쌓였구나"
- "이제 정말 편하다"

---

## 4️⃣ 이탈 방지 최소 장치 (딱 3개)

### ① 침묵 감지 장치

**조건:**
- 관리자 방문 14일 없음

**노출 (1회, 메일 or 알림):**
```
기록장은 계속 작동 중입니다.
최근 기록도 정상적으로 보관되어 있어요.
```

**❌ 링크 ❌ CTA**

---

### ② 기록 중단 감지

**조건:**
- 21일간 활동 데이터 없음

**문구:**
```
최근 기록이 잠시 멈춰 있습니다.
다시 시작되면 자동으로 이어집니다.
```

**📌 "문제"라는 표현 ❌**

---

### ③ Pro 만료 전 알림 (Pro만)

**시점:**
- 만료 3일 전

**문구:**
```
자동 기록이 곧 중단될 예정입니다.
기존 기록은 그대로 유지됩니다.
```

**👉 선택권 강조**

---

### ❌ 절대 하지 않는 이탈 방지

- "돌아오세요"
- "지금 확인하세요"
- "놓치고 있어요"

**👉 불안 유발 금지**

---

## 5️⃣ 30일 알림 총량 제한

|------|------|

## 6️⃣ 30일 성공 판단 기준 (정량)

- 자동 기록 ≥ 4건
- 관리자 방문 ≥ 3회
- Pro 팀 ≥ 30%

**📌 전부 안 돼도 OK**
**→ 기록이 쌓였으면 성공**

## 7️⃣ 이탈 위험 신호

### 위험 신호 1: 기록 생성 중단

**조건:**
- 14일 이상 기록 생성 없음
- 활동 데이터는 있음

**대응:**
```
❌ 알림 없음
✅ 대시보드에 "기록 생성 대기 중" 표시
```

---

### 위험 신호 2: 접속 중단

**조건:**
- 30일 이상 접속 없음
- 기록은 계속 생성됨

**대응:**
```
❌ 알림 없음
✅ 기록은 계속 쌓임
```

---

### 위험 신호 3: 설정 변경

**조건:**
- 자동 기록 OFF
- 공유 자동 생성 OFF

**대응:**
```
❌ 알림 없음
✅ 설정 변경 확인 메시지 (1회만)
```
