# 🔥 네비 UI 상태 머신 설계

## 📐 전체 상태 흐름

```
[IDLE] ──검색 시작──> [SEARCHING] ──장소 선택──> [CONFIRMED] ──출발──> [NAVIGATING] ──도착──> [ARRIVED]
  ↑                                                                                              ↓
  └─────────────────────────────────────────────── 리셋 ─────────────────────────────────────────┘
```

---

## 1️⃣ IDLE (대기 상태)

### 화면 구성

```
┌─────────────────────────────────────────┐
│  [헤더: "지도"]                          │
├─────────────────────────────────────────┤
│                                         │
│  🟢 "말해보세요"                        │
│  "떠오르는 장소 말해보세요"              │
│                                         │
│  ┌─────────────────────────────────┐   │
│  │ 🔍 장소를 검색하세요...         │   │
│  └─────────────────────────────────┘   │
│                                         │
│         [지도: 기본 줌, 내 위치 중심]    │
│                                         │
│                                         │
│                                         │
│                                         │
│                                         │
│                                         │
│  [내 위치 버튼]                          │
└─────────────────────────────────────────┘
```

### UI 요소
- ✅ 상단: 녹색 음성 버튼 ("말해보세요")
- ✅ 검색바: 표시 (비활성)
- ✅ 지도: 기본 줌, 내 위치 중심
- ✅ 하단: 없음

### 사용자 행동
- 검색바 클릭/포커스 → **SEARCHING** 전환
- 음성 버튼 클릭 → 음성 인식 시작 (SEARCHING 유지)

---

## 2️⃣ SEARCHING (검색 중)

### 화면 구성

```
┌─────────────────────────────────────────┐
│  [헤더: "지도"]                          │
├─────────────────────────────────────────┤
│                                         │
│  🟢 "말해보세요"                        │
│                                         │
│  ┌─────────────────────────────────┐   │
│  │ 🔍 상암축구장                    │   │ ← 활성 검색어
│  └─────────────────────────────────┘   │
│                                         │
│  ┌─────────────────────────────────┐   │
│  │ 📍 상암월드컵경기장              │   │
│  │    서울특별시 마포구...          │   │
│  │    약 1.2km                      │   │
│  ├─────────────────────────────────┤   │
│  │ 📍 상암축구공원                  │   │
│  │    서울특별시 마포구...          │   │
│  │    약 0.8km                      │   │
│  └─────────────────────────────────┘   │
│                                         │
│         [지도: 검색 결과 마커 표시]      │
│                                         │
│                                         │
│                                         │
│                                         │
│  [내 위치 버튼]                          │
└─────────────────────────────────────────┘
```

### UI 요소
- ✅ 상단: 녹색 음성 버튼 (유지)
- ✅ 검색바: 활성 (입력 중)
- ✅ 검색 결과 리스트: 표시 (하단 스크롤)
- ✅ 지도: 검색 결과 마커 표시
- ✅ 하단: 없음

### 사용자 행동
- 검색 결과 클릭 → **CONFIRMED** 전환
- 검색 취소 (X 버튼) → **IDLE** 복귀

---

## 3️⃣ CONFIRMED (목적지 확정 / 출발 대기)

### 화면 구성

```
┌─────────────────────────────────────────┐
│  [헤더: "지도"]                          │
├─────────────────────────────────────────┤
│                                         │
│                                         │
│                                         │
│         [지도: 목적지 마커 강조]         │
│         [지도: 출발지 → 목적지 화살표]   │
│                                         │
│                                         │
│                                         │
│                                         │
│  ┌─────────────────────────────────┐   │
│  │ 🎯 서울월드컵경기장             │   │
│  │    서울특별시 마포구...          │   │
│  │    약 1.2km                      │   │
│  │                                 │   │
│  │  [출발]  [다른 곳]              │   │
│  └─────────────────────────────────┘   │
│                                         │
│  [내 위치 버튼]                          │
└─────────────────────────────────────────┘
```

### UI 요소
- ✅ 상단: 없음 (검색바 숨김)
- ✅ 지도: 목적지 마커 강조, 출발지 → 목적지 방향 힌트
- ✅ 하단 카드: 목적지 정보 + 출발 버튼

### 사용자 행동
- "출발" 버튼 클릭 → **NAVIGATING** 전환
- "다른 곳" 버튼 클릭 → **SEARCHING** 복귀
- 카드 닫기 (X) → **SEARCHING** 복귀

### 시각적 변화
- ✅ 검색바 사라짐
- ✅ 하단 카드 등장 (애니메이션)
- ✅ 지도 줌 조정 (출발지 + 목적지 fitBounds)

---

## 4️⃣ NAVIGATING (안내 중)

### 화면 구성

```
┌─────────────────────────────────────────┐
│  [헤더: "지도"]                          │
├─────────────────────────────────────────┤
│  ┌─────────────────────────────────┐   │
│  │ 🚗 차로 · 1.2km · 15분          │   │ ← 상단 상태바
│  │    서울월드컵경기장              │   │
│  └─────────────────────────────────┘   │
│                                         │
│         [지도: 파란 경로선 표시]        │
│         [지도: 출발지 마커 (파란 점)]    │
│         [지도: 목적지 마커 (빨간 깃발)]  │
│         [지도: 카메라 자동 추적]        │
│                                         │
│                                         │
│                                         │
│                                         │
│  ┌─────────────────────────────────┐   │
│  │ 🎯 서울월드컵경기장             │   │
│  │    🚗 이동 중...                │   │
│  │    1.2km · 15분                 │   │
│  │                                 │   │
│  │  [중지]                         │   │
│  └─────────────────────────────────┘   │
│                                         │
│  [내 위치 버튼]                          │
└─────────────────────────────────────────┘
```

### UI 요소
- ✅ 상단 상태바: 이동 수단 + 거리 + 시간 + 목적지 이름
- ✅ 검색바: 숨김
- ✅ 지도: 파란 경로선, 출발지/목적지 마커, 카메라 추적
- ✅ 하단 카드: 목적지 이름 + "이동 중..." + 거리/시간 + 중지 버튼

### 사용자 행동
- "중지" 버튼 클릭 → 확인 다이얼로그 → **IDLE** 복귀
- 지도 드래그 → 카메라 자유 모드 (자동 추적 해제)

### 시각적 변화
- ✅ 상단 상태바 등장 (애니메이션)
- ✅ 검색바 사라짐
- ✅ 파란 경로선 표시
- ✅ 하단 카드 업데이트 ("이동 중..." + 거리/시간)
- ✅ 지도 줌 조정 (경로 전체 fitBounds)

### 경로 계산 중 상태

```
┌─────────────────────────────────────────┐
│  ┌─────────────────────────────────┐   │
│  │ 🚗 차로 · ⏳ 경로 계산 중...    │   │ ← 로딩 스피너
│  └─────────────────────────────────┘   │
│                                         │
│         [지도: 로딩 중...]              │
│                                         │
│  ┌─────────────────────────────────┐   │
│  │ 🎯 서울월드컵경기장             │   │
│  │    ⏳ 경로 계산 중...            │   │
│  └─────────────────────────────────┘   │
└─────────────────────────────────────────┘
```

### 사용자 행동
- "확인" 버튼 클릭 → **IDLE** 복귀

### 시각적 변화
- ✅ 상단 상태바 사라짐
- ✅ 경로선 제거
- ✅ 하단 카드 업데이트 ("도착했습니다" + 초록색 테두리)

## 🔥 상태 전환 규칙

### 전환 가능한 상태

| 현재 상태 | 다음 상태 | 트리거 |
|---------|---------|--------|
| IDLE | SEARCHING | 검색바 포커스/클릭 |
| SEARCHING | IDLE | 검색 취소 |
| SEARCHING | CONFIRMED | 검색 결과 선택 |
| CONFIRMED | SEARCHING | "다른 곳" / 카드 닫기 |
| CONFIRMED | NAVIGATING | "출발" 버튼 클릭 |
| NAVIGATING | IDLE | "중지" 버튼 클릭 |
| NAVIGATING | ARRIVED | 목적지 도착 감지 |
| ARRIVED | IDLE | "확인" 버튼 클릭 |
| ARRIVED | SEARCHING | "주변 보기" 버튼 클릭 |

### 전환 불가능한 상태

- ❌ IDLE → CONFIRMED (검색 없이 바로 목적지 확정 불가)
- ❌ CONFIRMED → ARRIVED (출발 없이 도착 불가)
- ❌ SEARCHING → NAVIGATING (목적지 확정 없이 안내 시작 불가)

## 📱 모바일 최적화

### Safe Area 고려
- 상단 상태바: `calc(var(--header-h, 56px) + env(safe-area-inset-top, 0px) + 8px)`
- 하단 카드: `bottom: calc(80px + env(safe-area-inset-bottom, 0px))`

### 터치 영역
- 버튼 최소 높이: 44px
- 카드 패딩: 16px 이상
