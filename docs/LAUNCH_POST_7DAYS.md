## 📊 핵심 지표 추적

### 일일 확인 지표

1. **신규 가입자 수**
   - Firestore `users` 컬렉션 신규 문서 수
   - 전날 대비 증가율

2. **SMS 인증 성공률**
   - `auth_logs` 컬렉션 분석
   - `sms_success` / `sms_request` 비율

3. **온보딩 완료율**
   - `onboardingCompleted: true` 유저 수
   - 전체 가입자 대비 비율

4. **에러 발생률**
   - `sms_error` / `verify_error` 발생 수
   - 전체 요청 대비 비율

### 주간 확인 지표

1. **D1 리텐션**
   - 가입 후 1일 내 재방문율
   - `eventLogs` 분석

2. **온보딩 이탈률**
   - 각 단계별 이탈률
   - 이탈 지점 분석

3. **SMS 비용**
   - 일일 SMS 전송량
   - 비용 추이
