# 🎯 엘리베이터 피치 5종 (상황별)

## 📋 목적
10초 안에 끝나는 설명

---

## 🎯 원칙

### 피치 목표

- 10초 안에 끝남
- 상황별 맞춤
- 핵심만 전달
- 자연스러운 대화

---

### 톤 가이드

- 담백함
- 과장 없음
- 명확함
- 간결함

---

## 0️⃣ 엘리베이터 피치의 핵심

**10초 안에 끝나는 설명**

- ❌ 기능 나열
- ❌ 과장 표현
- ⭕ 핵심 가치만
- ⭕ 자연스러운 대화

---

## 1️⃣ 처음 만난 사람에게 (가장 기본)

### 상황

- 일상 대화
- 첫 소개
- 간단한 질문

### 피치

```
팀이 한 활동을 홍보 말고,
날짜가 남는 기록으로 자동으로 남겨주는 서비스예요.
```

**📌 포인트**
- 홍보 ❌
- 자동 ⭕
- 기록 ⭕

**길이: 약 8초**

---

## 2️⃣ 동호회/팀 관리자에게

### 상황

- 동호회 운영자 만남
- 문제 공감
- 해결책 제시

### 피치

```
활동은 계속하는데 남는 게 없잖아요.
기록장은 그걸 자동으로 정리해 줍니다.
```

**📌 포인트**
- 공감 → 해결
- 기능 설명 없음

**길이: 약 8초**

---

## 3️⃣ '그게 왜 필요해?'라고 묻는 사람에게

### 상황

- 의문 제기
- 필요성 질문
- 가치 설명

### 피치

```
기록이 쌓이면,
팀을 설명하지 않아도 사람들이 이해하더라고요.
```

**📌 포인트**
- 결과만 말함
- 논리 설명 ❌

**길이: 약 8초**

---

## 4️⃣ 지자체·공공 관계자에게

### 상황

- 지자체 협업 제안
- 공공성 강조
- 비용 부담 없음

### 피치

```
행사가 아니라,
지역에서 실제로 계속 일어나는 활동을 남기는 기록입니다.
```

**📌 포인트**
- 연속성
- 공공 데이터 감각

**길이: 약 10초**

---

## 5️⃣ 투자자·기획자에게 (최소 설명)

### 상황

- 투자 설명
- 파트너십 제안
- 비즈니스 모델

### 피치

```
팀 활동 기록을 자동화해서
신뢰가 쌓이게 만드는 구조입니다.
```

**📌 포인트**
- 구조 중심
- 시장/숫자 언급 ❌

**길이: 약 8초**
