# 🎬 데모 시나리오 & 협회 시연 스크립트 (노원구축구협회)

**생성일**: 2025-01-27  
**목적**: 노원구축구협회 시연용 데모 플로우 및 설명 스크립트  
**대상**: 협회 임원 (사무국장, 회장 등)  
**원칙**: 5초 이해, 설명 최소화, 화면 중심

---

## 📋 시연 개요

### 시연 시간
- **총 소요 시간**: 10~15분
- **화면 설명**: 각 화면당 30초~1분
- **질문 시간**: 5분

### 시연 환경
- **데이터**: 노원구축구협회 시드 데이터 사용
- **화면**: 실제 UI (디자인 완료 후)
- **데모 모드**: 실제 데이터 기반

---

## 🎯 시연 목표

1. **협회 중심 구조 이해** (조직 컨텍스트 바)
2. **운동장 대관이 핵심 가치임을 인식** (우선 대관 시설)
3. **회원/비회원 차이를 상태로 이해** (설명 없이)
4. **전환 UX의 자연스러움** (강요 없음)

## 📐 시연 화면 순서

### 화면 1: 우리 팀 관리 메인 (30초)

**화면 설명**:

```
"이 화면은 노원구축구협회의 전체 운영 현황을 보여줍니다."

[화면 표시]
- 조직 컨텍스트 바: "협회 ▼ 노원구축구협회"
- 운동장 대관 카드 (최상단, 가장 크게)
  - "이번 주 12개 대관"
  - "회원팀 8개 · 아카데미 3개 · 비회원 1개"
- 팀 목록
  - 회원팀 (3개): 노원FC, 강북FC, 공릉FC 🟢
  - 비회원팀 (3개): 동부FC, 중앙FC, 북부FC ⚪
  - 아카데미 (2개): 노원유스, 실버축구단 🔵
```

**설명 포인트** (간단히):

> "협회가 중심이고, 운동장 대관이 가장 중요한 기능입니다.  
> 색상만으로 회원/비회원/아카데미를 구분할 수 있습니다."

**침묵 시간**: 5초 (화면을 보게 함)

---

### 화면 2: 운동장 대관 메인 (1분)

**화면 설명**:

```
"운동장 대관 화면입니다."

[화면 표시]
- 이번 주 대관 요약
  - 회원팀 8개 🟢
  - 아카데미 3개 🔵
  - 비회원 1개 ⚪
- 협회 우선 대관 시설 (녹색 보더, 좌측 세로 바)
  - 육군사관학교 축구장
  - 경기기계공업고등학교 축구장
  - 서울과학기술대학교 운동장
- 일반 대관 시설
  - 마들 스타디움
```

**설명 포인트**:

> "이 세 시설은 협회가 우선 대관권을 가진 시설입니다.  
> 회원팀은 이 시설에 우선적으로 신청할 수 있습니다."

**시연 액션**:

1. "육군사관학교 축구장" 카드 클릭
2. → 시설 상세 화면으로 이동

---

### 화면 3: 시설 상세 - 회원팀 (30초)

**화면 설명**:

```
[회원팀(공릉FC)으로 로그인한 상태]

[화면 표시]
- 시설 정보: 육군사관학교 축구장
- 상태 메시지 카드 (녹색 배경)
  - "🟢 우선 배정 대상입니다"
  - [대관 신청] 버튼
```

**설명 포인트**:

> "회원팀은 이 시설에 바로 대관 신청이 가능합니다.  
> 설명 없이 상태만 보여줍니다."

**시연 액션**:

1. "전체 캘린더 보기" 클릭
2. → 대관 캘린더 화면으로 이동

---

### 화면 4: 대관 캘린더 - 회원팀 (1분)

**화면 설명**:

```
[화면 표시]
- 캘린더 뷰
  - 🟢 회원팀 배정 시간대
  - 🔵 아카데미 배정 시간대
  - ⚪ 잔여 슬롯
- 범례: 🟢 회원팀  🔵 아카데미  ⚪ 잔여
```

**설명 포인트**:

> "색상만으로 권한을 구분합니다.  
> 회원팀은 녹색 시간대에 바로 신청할 수 있습니다."

**시연 액션**:

1. 🟢 슬롯 클릭
2. → "대관 신청" 모달 표시
3. 모달 닫기

---

### 화면 5: 시설 상세 - 비회원팀 (1분 30초) ⭐ 핵심

**화면 설명**:

```
[비회원팀(동부FC)으로 로그인한 상태]

[화면 표시]
- 시설 정보: 육군사관학교 축구장
- 상태 메시지 카드 (회색 배경)
  - "⚪ 잔여 시간대만 이용 가능"
  - "협회 우선 대관 시설 · 회원팀 우선 배정"
  - [대기 신청] 버튼
  - [회원팀 전환 안내] 버튼 (작게)
```

**설명 포인트** (중요):

> "비회원팀은 협회 우선 대관 시설을 바로 신청할 수 없습니다.  
> 하지만 강요하지 않고, 전환 안내만 제공합니다."

**시연 액션**:

1. "회원팀 전환 안내" 버튼 클릭
2. → 전환 모달 표시 (STEP 2)

---

### 화면 6: 회원팀 전환 모달 - 가치 확인 (1분)

**화면 설명**:

```
[모달 표시]
- 헤드라인: "회원팀 전환 시 달라지는 점"
- 체크리스트 3가지:
  - 🟢 협회 우선 대관 시설 이용
  - 🟢 잔여 대기 없음
  - 🟢 협회 대회 참가 가능
- 비용: "연회비 240만원" (작게, 강조 없음)
- 버튼: [전환 문의] [나중에]
```

**설명 포인트**:

> "가격이 아니라 얻는 것을 먼저 보여줍니다.  
> 비용은 작게 표시하고, 강요하지 않습니다."

**시연 액션**:

1. "전환 문의" 버튼 클릭
2. → 전환 문의 폼 표시 (STEP 3)

---

### 화면 7: 전환 문의 폼 (30초)

**화면 설명**:

```
[폼 표시]
- 팀명: "동부FC" (자동 입력, 수정 불가)
- 담당자 연락처: "010-1234-5678" (자동 입력, 수정 불가)
- 추가 메모: (선택 입력)
- 버튼: [문의 접수] [취소]
```

**설명 포인트**:

> "최소한의 입력만 요구합니다.  
> 팀명과 연락처는 자동으로 채워집니다."

**시연 액션**:

1. 메모 입력: "협회 우선 대관 시설 이용을 위해 전환 신청합니다."
2. "문의 접수" 버튼 클릭
3. → 성공 메시지 표시
4. → 팀 목록으로 돌아가기

---

### 화면 8: 전환 문의 후 상태 변화 (30초)

```
[팀 목록 화면]

[화면 표시]
- 동부FC 카드
  - 상태: ⚪ (회색)
  - 배지: "전환 문의 중" (노란색)
```

**설명 포인트**:

> "전환 문의가 접수되면 배지로 상태를 표시합니다.

### 화면 9: 전환 승인 후 즉시 반영 (1분) ⭐ 핵심

**시연 액션** (협회 관리자 화면):

1. 전환 문의 승인 (시뮬레이션)
2. → 팀 상태 변경: PENDING → MEMBER

**화면 설명**:

```
[동부FC로 다시 로그인]

[화면 표시]
- 팀 카드 위치 변경: 비회원팀 섹션 → 회원팀 섹션으로 이동
- 색상 변경: 회색 → 녹색 🟢
- 배지 제거: "전환 문의 중" 배지 사라짐
```

**시연 액션**:

1. "육군사관학교 축구장" 시설 상세로 이동
2. → 상태 메시지 변경: "⚪ 잔여 시간대만 이용 가능" → "🟢 우선 배정 대상입니다"
3. → CTA 변경: [대기 신청] + [회원팀 전환 안내] → [대관 신청] (단일 버튼)

**설명 포인트** (중요):

> "승인 즉시 권한이 반영됩니다.  
> 페이지 새로고침 없이 자동으로 변경됩니다."

- PermissionDecision: VIEW_ONLY
