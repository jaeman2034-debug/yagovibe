# 🛡️ 신뢰 표시 (UI 마이크로카피)

## 📋 목적
사용자는 느끼지만, 의식하지 않는 신뢰 장치

---

## 🎯 원칙

### UI 목표

- 자연스러운 신뢰 형성
- 의식하지 않는 신뢰 장치
- 담백한 표현
- 사실 기반 표시

---

### 톤 가이드

- 담백함
- 과장 없음
- 신뢰감
- 편안함

---

## 0️⃣ 설계 원칙 (절대 규칙)

- ❌ 선언형 문구 ("신뢰합니다", "안전합니다")
- ❌ 배지 남발
- ❌ 약관 링크 강조
- ⭕ 상황 설명형
- ⭕ 사실 전달형
- ⭕ 짧고 조용하게

**신뢰는 보여주는 게 아니라, 설명 안 해도 느껴져야 한다**

---

## 신뢰 표시의 핵심

**사용자는 느끼지만, 의식하지 않는 신뢰 장치**

- ❌ "신뢰할 수 있습니다" 같은 직접적 표현
- ⭕ 자연스러운 마이크로카피
- ⭕ 사실 기반 표시

---

## 1️⃣ Hero 영역 — 첫 인상 신뢰

### 상태 배지 (작게)

```
🟢 최근 활동 기록 중
```

### 마이크로카피 (캐치프레이즈 하단)

```
이 페이지는 팀의 실제 활동을
있는 그대로 기록한 공간입니다.
```

**📌 효과**
- 광고 오해 차단
- "관찰해도 되는 곳" 인식

---

## 2️⃣ 최근 활동 요약 — 사실성 강화

### 날짜 옆 마이크로카피

```
자동으로 기록됨
```

**❌ "업데이트됨" ❌**
**→ 사람 개입 최소화 암시**

---

## 3️⃣ TeamMetaGrid — 과장 방지

### Grid 하단 (공통)

```
팀 정보는
변경될 때만 갱신됩니다.
```

**📌 효과**
- "항상 좋게 보이게 손보는 거 아님" 신호

---

## 4️⃣ 블로그 카드 — 기록성 강조

### 카드 하단 (날짜 아래)

```
팀 활동 기록
```

**또는 (공유 유입 시)**

```
이 팀의 일상 기록입니다
```

**📌 효과**
- 콘텐츠 소비 압박 제거
- 홍보글 오인 차단

---

## 5️⃣ 블로그 본문 — AI 오해 차단 (아주 작게)

### 글 하단 회색 텍스트

```
이 글은 팀 활동 데이터를 바탕으로
자동 정리되었습니다.
```

**❗ "AI" 단어 직접 노출 ❌**
**→ 자동화 투명성 + 불편 최소화**

---

## 6️⃣ CTA 주변 — 선택권 강조

### CTA 하단 마이크로카피

```
지금 결정하지 않아도 괜찮습니다.
```

**📌 효과**
- 클릭 압박 제거
- 체류 시간 ↑

---

## 7️⃣ 관리자 페이지 — 안심 언어

### 자동화 상태 옆

```
별도 설정 없이 유지됩니다.
```

### 토글 OFF 시

```
기존 기록은 그대로 남습니다.
```

**📌 효과**
- "실수하면 망한다" 불안 제거

---

## 8️⃣ 공유 유입 전용 — 경계 해제

### Hero 하단

```
공개 기록 페이지 · 누구나 볼 수 있습니다
```

**❌ "가입 없이 보기" ❌**
**→ 행동 유도 없음**

---

## 9️⃣ 공공/지역 페이지 — 중립성 확보

### 페이지 상단

```
이 페이지는
지역 팀들의 활동 기록을 모아 보여줍니다.
```

### 하단

```
비교·순위·평가는 하지 않습니다.
```

**📌 효과**
- 공공 데이터 인식
- 상업성 차단
