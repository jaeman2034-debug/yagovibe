# 🧪 테스트 가이드 (Google Maps 연동)

## 핵심 원칙

> **모바일 실기기 테스트가 기준이다.**
> 웹 브라우저 테스트는 참고용일 뿐이다.

---

## ✅ 필수 테스트 시나리오 (모바일)

### 1. 기본 길찾기 플로우

1. 앱 실행
2. 장소 검색 또는 마커 선택
3. 교통수단 선택 (도보/자동차/대중교통)
4. **Google Maps 앱으로 전환 확인**
5. "지금 출발" 버튼 활성화 확인
6. 네비게이션 시작 확인

### 2. 복귀 UX 테스트

1. Google Maps에서 뒤로가기
2. 앱으로 복귀
3. Phase 33 상태 복원 확인
4. "다시 길찾기" 버튼 작동 확인
5. 음성 명령 "계속 안내" 작동 확인

### 3. 실패 대응 테스트

1. 네트워크 끊김 시뮬레이션
2. Phase 37 상태 표시 확인
3. "다시 시도" 버튼 작동 확인

---

## ⚠️ 웹 브라우저 테스트 (참고용)

### 제한 사항

웹 브라우저에서 테스트할 때:
- **자동차 모드**: "지금 출발" 버튼 비활성화 (정상, Google 제한)
- **도보 모드**: "지금 출발" 버튼 활성화 ⭕
- **대중교통 모드**: "지금 출발" 버튼 활성화 ⭕

### 웹 테스트 체크리스트

- [ ] 도보 모드 → 정상 작동
- [ ] 대중교통 모드 → 정상 작동
- [ ] 자동차 모드 → "지금 출발" 비활성화 (버그 아님)

---

## 🎯 테스트 우선순위

### 1순위: 모바일 실기기
- iOS Safari
- Android Chrome
- 실제 Google Maps 앱 연동

### 2순위: 웹 브라우저 (참고)
- Chrome (데스크탑)
- Safari (데스크탑)
- **자동차 모드 제한 인지**

---

## 📋 버그 vs 제한 구분

### 버그로 오인하지 말 것

- ❌ "지금 출발" 버튼이 안 눌린다 (웹 + 자동차)
- ❌ 자동차 모드에서 오류 메시지
- ✅ **이것들은 Google Maps 웹 제한입니다**

### 실제 버그 체크

- ✅ 모바일에서 "지금 출발" 비활성화
- ✅ 앱 복귀 시 상태 복원 실패
- ✅ 음성 명령 미작동
- ✅ Phase 37 실패 감지 미작동

---

## 🔍 디버깅 팁

### "지금 출발" 비활성화 확인

Google Maps 웹에서 왼쪽 패널 확인:
- "현재 제공 중인 운전 경로 검색 서비스 범위를 벗어난 검색입니다."
- → 이것은 **정상 제한**입니다

### 모바일에서 확인

1. 모바일 브라우저에서 동일 URL 열기
2. Google Maps 앱으로 자동 전환 확인
3. "지금 출발" 버튼 활성화 확인
4. → **정상 작동**입니다
