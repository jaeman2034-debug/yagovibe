# 💰 플랫폼 수익 모델 / 요금제 설계 (SaaS 현실 최적안)

## 📋 개요

**핵심 원칙:**
- 협회는 부담 없이 시작
- 플랫폼은 자연스럽게 수익 발생
- 운영 규모 커질수록 수익 자동 증가
- **"쓰다 보면 결제하게 되는 SaaS"**

---

## 🎯 수익 모델의 축 (3개)

### 1. 대회 단위 요금 (Competition-based)
- 월 구독 또는 대회당 결제
- 기본 기능 제공

### 2. 결제 수수료 (Transaction-based)
- 참가비 결제 시 플랫폼 수수료 2~5%
- 또는 건당 1,000~2,000원

### 3. 프로 기능 업셀 (Feature-based)
- 고급 기능은 PRO 플랜에서만 제공

---

## 💳 요금제 구조

### 🟢 FREE (진입용 / 마케팅)

**가격:** 무료

**제한:**
- 대회 1개
- 참가 신청 / 승인
- 선수 명단 관리
- 엑셀 Export ❌
- 알림 자동화 ❌
- 플랫폼 로고 표시 ⭕

**목적:** 유입 + 레퍼런스 확보

---

### 🔵 BASIC (가장 많이 팔릴 플랜 ⭐)

**가격:**
- 월 49,000원 또는
- 대회당 99,000원

**포함:**
- 대회 무제한
- 참가 신청 / 승인
- 선수 명단
- 엑셀 Export ⭕
- 승인/미제출 알림 ⭕
- 기본 어드민
- 플랫폼 로고 제거 ⭕

**타겟:** 협회/지자체가 가장 편하게 결제하는 구간

---

### 🟣 PRO (운영 자동화 풀옵션)

**가격:** 월 149,000원

**포함:**
- BASIC 전부 포함
- 결제 연동 ⭕
- 자동 영수증 / 입금 확인
- 현장 체크인 (QR)
- 알림 커스터마이징
- 우선 지원

**타겟:** 사람 1명 줄이는 비용 < 이 요금

---

## 💰 결제 수수료 모델

### 구조
- 참가비 결제 시 플랫폼 수수료 2~5%
- 또는 건당 1,000~2,000원

### 예시
```
팀 50팀 × 300,000원 = 1,500만 원
수수료 3% = 45만 원
```

**협회 관점:**
- "사람 안 써도 됨"
- "입금 관리 안 해도 됨"

---

## 🎨 과금 타이밍 (UX)

### ❌ 피해야 할 것
- 가입하자마자 결제 요청 → 이탈

### ⭕ 정답: "가치 체험 후" 과금

**예시 흐름:**
```
대회 생성
  → 참가 신청 받음
    → 승인 몇 팀 처리
      → [엑셀 다운로드] 클릭
        → 요금 안내 모달
```

**👉 지금 필요해서 결제하게 됨**

## 🔧 기술 구현

### 데이터 모델

```typescript
interface Association {
  plan: "free" | "basic" | "pro";
  planStartedAt?: Timestamp;
  billing?: {
    status: "active" | "cancelled" | "past_due";
    provider?: "toss" | "kakao" | "stripe";
    customerId?: string;
    subscriptionId?: string;
  };
}
```

### 기능 제한 로직

**프론트엔드:**
```typescript
if (association.plan !== 'free') {
  showExportButton();
}
```

**백엔드 (Firestore Rules):**
```javascript
function hasPlan(plan) {
  return resource.data.plan == plan;
}

allow read: if hasPlan('basic') || hasPlan('pro');
```

**Cloud Functions:**
```typescript
if (association.plan === 'free') {
  throw new HttpsError('failed-precondition', 'Upgrade required');
}
```

## 🚀 업그레이드 플로우

1. **기능 사용 시도** (예: 엑셀 Export)
2. **Plan 체크** → FREE면 Paywall 표시
3. **요금제 선택** → 결제 진행
4. **결제 완료** → Plan 업데이트
5. **기능 사용 가능**
