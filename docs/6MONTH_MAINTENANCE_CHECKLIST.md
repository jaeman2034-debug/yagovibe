# 💤 6개월 무개입 운영 체크리스트

## 목적

존재를 증명하지 않고, 시간이 증명하게 둔다

**핵심 원칙**: "잘 되는지 보지 말고, 망가지지 않는지만 본다"

---

## 전체 원칙 (가장 중요)

- ❌ 성장 목표 없음
- ❌ 기능 추가 없음
- ❌ 카피 수정 없음
- ⭕ 에러만 제거

---

## 🗓️ Month 1 — 완전 방치

### 해야 할 것

**서버 에러 로그 주 1회 확인**
- Firebase Functions 로그 확인
- 에러 발생 시에만 대응
- 정상 작동 시 아무것도 하지 않음

**Firestore 용량 초과 여부 확인**
- 월 1회 용량 확인
- 초과 시 알림만 확인
- 자동 삭제 정책 확인

### 하지 말 것

- ❌ 방문자 수 보기
- ❌ 공유 횟수 보기
- ❌ "왜 반응 없지?" 생각

**조용하면 정상**

---

## 🗓️ Month 2 — 자동화 생존 확인

### 체크 (월 1회, 5분)

**주간 기록 생성이 멈추지 않았는가**
- Cloud Scheduler 실행 로그 확인
- 최근 4주간 기록 생성 여부 확인
- 실패 시에만 대응

**특정 팀만 계속 실패하지는 않는가**
- 에러 로그에서 패턴 확인
- 시스템 문제인지 개별 문제인지 구분
- 시스템 문제일 때만 대응

### 금지

- ❌ 실패 팀에 연락
- ❌ 수동 개입
- ❌ "왜 안 되지?" 조사

**실패는 시스템 문제일 때만 다룬다**

---

## 🗓️ Month 3 — 외부 시선 점검 (비개입)

### 볼 수 있는 신호

**외부 링크 유입 1회라도 있었는가**
- Google Analytics (최소 설정)
- Referrer 확인
- 공공 링크 공유 여부

**"이런 기록 있더라"라는 말이 있었는가**
- 검색어 유입 확인
- 직접 URL 접근 확인
- 외부 언급 여부

### ❌ 하지 말 것

- ❌ 해석
- ❌ 확대
- ❌ 대응

**관측만 한다**

---

## 🗓️ Month 4 — 관리자 안심 유지

### 체크

**관리자 문의가 없었는가**
- 이메일 확인
- 문의 없음 = 정상 작동

**"잘 되고 있나요?" 질문이 없었는가**
- 질문 없음 = 설명 불필요
- 질문 있음 = 에러 가능성

**질문이 없다는 건 설명이 필요 없다는 뜻**

---

## 🗓️ Month 5 — 유지 비용 재확인

### 확인

**월 비용이 예산 범위 내인가**
- Firebase 비용 확인
- AI 비용 확인
- 예산 초과 여부 확인

**AI 비용이 급증하지 않았는가**
- OpenAI 사용량 확인
- 비정상적 증가 패턴 확인
- 원인 파악 (시스템 문제인지)

### 조치

**급증 시 → 생성 빈도 줄이기**
- 주간 기록 생성 빈도 조정
- 재시도 횟수 감소
- 불필요한 AI 호출 제거

**❌ 기능 추가로 해결하지 말 것**

---

## 🗓️ Month 6 — 단 하나의 판단

### 이 질문만 한다

**이게 없어지면, 아쉬운 사람이 있을까?**

**YES → 유지 성공**
- 누군가가 필요로 함
- 존재 가치 확인
- 계속 유지

**NO → 그래도 실패 아님**
- 아직 필요 시점이 아님
- 인프라는 느리게 필요해짐
- 시간이 증명하게 둠

   - ❌ 방향 전환

**브라우저 닫고 하루 쉬기**
