## 1️⃣ 조 수 자동 계산 규칙

```typescript
function calculateOptimalDivisionCount(totalTeams: number): number {
  if (totalTeams < 4) return 1;      // 1~3팀: 1조
  if (totalTeams < 8) return 2;      // 4~7팀: 2조
  if (totalTeams < 16) return 4;    // 8~15팀: 4조
  if (totalTeams < 32) return 8;    // 16~31팀: 8조
  return 8;                          // 32팀 이상: 8조 (최대)
}
```

### 예시

| 팀 수 | 자동 조 수 | 조별 팀 수 (평균) |
|------|----------|----------------|
| 4팀 | 2조 | 2팀/조 |
| 5팀 | 2조 | 3팀/조, 2팀/조 (편차 1) |
| 8팀 | 4조 | 2팀/조 |
| 9팀 | 4조 | 3팀/조, 2팀/조, 2팀/조, 2팀/조 (편차 1) |
| 14팀 | 4조 | 4팀/조, 4팀/조, 3팀/조, 3팀/조 (편차 1) |

## 2️⃣ 균등 분배 알고리즘 (라운드 로빈)

### 핵심 로직

```typescript
teams.forEach((team, idx) => {
  const divisionIdx = idx % divisionCount;
  divisions[divisionIdx].teams.push(team);
});
```

### 동작 원리

- **라운드 로빈 방식**: 순서대로 각 조에 1팀씩 배정
- **홀수팀 자동 처리**: 나머지 연산(`%`)으로 자동 균등 분배
- **최대 편차**: 항상 1팀 이하 (완벽한 균등 분배)

## 3️⃣ 알고리즘 레벨

### 레벨 0: 완전 랜덤

- **방식**: Fisher-Yates 셔플 → 라운드 로빈 배정
- **특징**: 시드 없음, 완전 랜덤
- **용도**: 테스트/데모

### 레벨 1: 시드 분산 + 랜덤 (권장)

- **방식**: 
  1. 시드팀을 각 조에 1팀씩 먼저 배치
  2. 나머지 팀을 랜덤 셔플 후 배정
- **특징**: 상위 시드는 서로 다른 조로 자동 분산
- **용도**: 실전 대회 (대부분의 생활체육 대회)

### 레벨 2: 시드 + 회피 규칙 + 균형 점수 (고급)

- **방식**:
  1. 시드팀 배치
  2. 100번 시도하여 최적 조합 탐색
  3. 점수 계산: `seedVariance * 0.6 + sameClubPenalty * 0.3 + regionPenalty * 0.1`
- **특징**: 
  - 동일 클럽/지역 회피
  - 조별 평균 시드 균형 최적화
- **용도**: 감사 대응이 중요한 대회

## 5️⃣ 홀수팀 처리 정리

### 원칙

1. **최대 편차 1팀**: 조별 팀 수 차이는 항상 1팀 이하
2. **앞 조 우선**: 나머지 팀은 앞 조부터 배정
3. **자동 처리**: 별도 로직 없이 라운드 로빈으로 자동 처리

### 수식

```
baseTeamsPerDivision = Math.floor(totalTeams / divisionCount)
remainder = totalTeams % divisionCount

앞 remainder개 조: baseTeamsPerDivision + 1팀
나머지 조: baseTeamsPerDivision팀
```
