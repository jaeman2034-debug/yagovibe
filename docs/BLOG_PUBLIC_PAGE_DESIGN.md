# 🌐 공개 팀 블로그 페이지 설계 문서

## 목적

홍보 페이지처럼 보이지 않으면서, 검색·공유·신뢰를 동시에 만족시키는 **기록 페이지** 설계

---

## 1️⃣ URL 구조 (SEO 핵심)

```
/teams/{teamSlug}
/teams/{teamSlug}/blog
/teams/{teamSlug}/blog/{postId}
```

**이유**:
- 팀 단위 SEO 축적
- 포스트 단위 공유 가능
- 나중에 지역/종목 묶기 쉬움

---

## 2️⃣ 팀 블로그 메인 페이지 UI

### ① Hero 영역 (Above the Fold)

**구성**:
- 팀명 (H1)
- 한 줄 설명: "이 팀의 활동 기록을 모아둔 페이지입니다."
- 상태 배지: "🟢 최근 활동 기록 중"

**제거 사항**:
- ❌ "AI가 생성"
- ❌ "홍보 블로그"
- ⭕ 기록임을 명확히

### ② 팀 메타 정보 (압축 신뢰)

**표시 항목** (DB 없으면 비노출):
- 📍 활동 지역
- 👥 연령대
- ⚽ 종목
- 🏃 주 1회

### ③ 기록 리스트 (핵심 콘텐츠)

**카드 구조**:
```
[제목]
소흘 60대 FC 팀 소개

[요약]
최근 팀 활동과 분위기를 간단히 정리했습니다.

[하단]
2025년 1월 · 팀 활동 기록
```

**규칙**:
- 날짜 항상 표시
- CTA 없음
- 최신순 정렬

**UX 규칙**:
- 기록 0개: "아직 기록이 없습니다"
- 기록 1개: Hero 강조
- 기록 3개+: 리스트

---

## 3️⃣ 개별 포스트 페이지 UI

**구성**:
- 상단: 팀명 + "팀 활동 기록"
- 제목 (H1)
- 본문 (단락 간격 넉넉)
- 하단 마이크로카피: "이 글은 팀 활동 데이터를 바탕으로 자동 정리되었습니다."

**특징**:
- 이미지 없어도 OK
- 과장 표현 없음
- 기록 톤 유지

---

## 4️⃣ SEO 메타 (실전용)

### 팀 블로그 메인

**Title**:
```
{teamName} 팀 활동 기록 | 기록장
```

**Description**:
```
{teamName}의 경기와 모임 활동을 기록으로 남깁니다.
홍보 없이 팀의 일상을 있는 그대로 정리합니다.
```

### 개별 포스트

**Title**:
```
{postTitle} | {teamName} 활동 기록
```

**Description**:
```
{teamName}의 활동과 분위기를 기록으로 정리했습니다.
```

---

## 5️⃣ 공유 카드(Open Graph) 규칙

**OG Title**:
```
{teamName} 팀 활동 기록
```

**OG Description**:
```
이 팀의 활동을 날짜가 남는 기록으로 정리했습니다.
```

**OG Image** (향후):
- 카드형
- 팀명 + 날짜
- CTA 없음

---

## 6️⃣ 접근 제어

**공개 페이지**: 누구나 접근

**제거 사항**:
- ❌ 댓글
- ❌ 좋아요
- ❌ 공유 버튼 강조

**관리자만**:
- "새 기록 생성" 버튼 노출

---

## 7️⃣ 실패를 막는 UX 규칙

| 상황 | 표시 |
|------|------|
| 기록 0개 | "아직 기록이 없습니다" |
| 기록 1개 | Hero 강조 |
| 기록 3개+ | 리스트 |

**제거 사항**:
- ❌ 빈 화면
- ❌ 로딩 스피너 남발

---

## 8️⃣ 왜 이 설계가 맞는가

**검색 유입** → 광고 오해 없음
**공유 클릭** → 설명 필요 없음
**팀 관리자** → 자부심만 남음
**지자체/외부** → 공공 기록처럼 인식

**핵심 인식**: "이 팀은 계속 활동하는구나"
