# 📋 구글 시트 템플릿 생성 및 연동 가이드

## 🎯 개요

구글 시트를 사용한 회원 일괄 등록은 **실전 운영의 핵심**입니다. 엑셀 파일보다 효율적이고, 실시간 공동 편집이 가능합니다.

## 📊 단계별 로드맵

### 1단계: 엑셀/CSV 업로드 (✅ 완료)
- 파일 업로드 기반 일괄 등록
- 즉시 안정화, 현장 사용 가능

### 2단계: 구글 시트 템플릿 제공 (✅ 완료)
- 표준 템플릿 링크 제공
- "사본 만들기" 방식
- 운영자 UX 급상승

### 3단계: 구글 시트 직접 연동 (✅ 완료)
- 시트 URL 붙여넣기
- 공개 시트에서 CSV 자동 다운로드
- 실시간 동기화 준비 완료

## 🔧 구글 시트 템플릿 생성 방법

### 1. 템플릿 시트 만들기

1. **새 구글 시트 생성**
   - https://sheets.google.com 접속
   - "빈 스프레드시트" 생성

2. **컬럼 구조 설정**
   ```
   A1: 이름*          (필수)
   B1: 전화번호
   C1: 배번
   D1: 역할
   E1: 회비플랜
   F1: 소속
   G1: 메모
   ```

3. **첫 행 고정**
   - 행 1 선택 → 보기 → 고정 → 1행

4. **드롭다운 설정** (선택사항)
   - **역할 컬럼 (D열):**
     - 데이터 → 데이터 유효성 검사
     - 조건: 목록에서 선택
     - 값: `일반,회장,부회장,총무,감독,코치,감사,상벌위원장`
   
   - **회비플랜 컬럼 (E열):**
     - 데이터 → 데이터 유효성 검사
     - 조건: 목록에서 선택
     - 값: `월회비,연회비,면제`
   
   - **소속 컬럼 (F열):**
     - 데이터 → 데이터 유효성 검사
     - 조건: 목록에서 선택
     - 값: `청룡,백호`

5. **예시 데이터 입력** (2-3행)
   ```
   손승호 | 010-1234-5678 | 10 | 회장 | 면제 | | 운영총괄
   김광일 | 010-2345-6789 | 7 | 일반 | 월회비 | 청룡 |
   ```

6. **설명 시트 추가** (선택사항)
   - 하단에 "설명" 시트 추가
   - 사용 방법, 주의사항 작성

### 2. 템플릿 공유 설정

1. **공유 버튼 클릭** (우측 상단)
2. **"링크가 있는 모든 사용자"** 선택
3. **권한: "뷰어"** 선택
4. **링크 복사**

### 3. 템플릿 ID 추출

URL에서 스프레드시트 ID 추출:
```
https://docs.google.com/spreadsheets/d/SPREADSHEET_ID_HERE/edit
                                    ↑ 이 부분이 템플릿 ID
```

### 4. 코드에 템플릿 ID 추가

`src/utils/googleSheetsTemplate.ts` 파일 수정:

```typescript
/**
 * 표준 회원 등록 템플릿
 */
export const MEMBER_TEMPLATE_ID = "YOUR_TEMPLATE_ID_HERE";

/**
 * 템플릿 "사본 만들기" 링크
 */
export function getMemberTemplateCopyLink(): string {
  return createGoogleSheetsCopyLink(MEMBER_TEMPLATE_ID);
}
```

`src/pages/team/TeamMembersPage.tsx`에서 템플릿 링크 사용:

```typescript
<a
  href={getMemberTemplateCopyLink()}
  target="_blank"
  rel="noopener noreferrer"
  className="text-xs text-blue-600 hover:text-blue-800 underline"
>
  📄 템플릿 사본 만들기 →
</a>
```

## 🔗 구글 시트 연동 사용 방법

### 사용자 가이드

1. **일괄 등록 모달 열기**
   - 회원 관리 페이지 → "일괄 등록" 버튼

2. **구글 시트 탭 선택**
   - "🔗 구글 시트" 탭 클릭

3. **시트 URL 입력**
   - 구글 시트에서 "공유" → "링크 복사"
   - URL 붙여넣기
   - 예: `https://docs.google.com/spreadsheets/d/SPREADSHEET_ID/edit`

4. **데이터 불러오기**
   - "불러오기" 버튼 클릭
   - 미리보기 테이블 확인

5. **저장**
   - "N명 저장" 버튼 클릭
   - 완료!

### ⚠️ 중요: 시트 공유 설정

구글 시트가 **공개**되어 있어야 합니다:

1. 시트에서 **"공유"** 버튼 클릭
2. **"링크가 있는 모든 사용자"** 선택
3. 권한: **"뷰어"** (읽기 전용)
4. **완료** 클릭

이 설정이 없으면 데이터를 불러올 수 없습니다.

## 🧠 고급: OAuth 2.0 직접 연동 (향후 구현)

현재는 공개 시트만 지원하지만, 향후 OAuth 2.0을 통해:

1. **사용자 인증**
   - Google 계정으로 로그인
   - 권한 위임

2. **비공개 시트 접근**
   - 사용자가 소유한 시트 직접 접근
   - Google Drive에서 시트 선택

3. **실시간 동기화**
   - 시트 변경 시 자동 반영
   - 양방향 동기화

## 🎯 실전 운영 시나리오

### 시나리오 1: 신규 회원 일괄 등록
1. 총무가 구글 시트 템플릿 사본 만들기
2. 신규 회원 정보 입력
3. 시트 URL 복사
4. 시스템에서 "구글 시트" 탭 → URL 붙여넣기 → 불러오기 → 저장

### 시나리오 2: 기존 회원 정보 업데이트
1. 감독이 구글 시트에서 배번/역할 수정
2. 시트 URL 복사
3. 시스템에서 불러오기 → 저장
4. 변경사항 즉시 반영

### 시나리오 3: 실시간 공동 편집
1. 회장, 총무, 감독이 동시에 시트 편집
2. 각자 담당 영역 수정
3. 시스템에서 주기적으로 동기화 (향후 기능)

## 🔧 기술 구현 상세

### CSV 다운로드 방식

공개 시트는 CSV로 직접 다운로드 가능:
```
https://docs.google.com/spreadsheets/d/{SPREADSHEET_ID}/gviz/tq?tqx=out:csv
```

### 데이터 파싱

1. CSV 텍스트 다운로드
2. 헤더 추출
3. 행별 데이터 파싱
4. 컬럼 자동 매핑
5. Firestore 일괄 등록

### 에러 처리

- **시트가 비공개인 경우:** 공유 설정 안내
- **URL 형식 오류:** 올바른 형식 안내
- **데이터 없음:** 빈 시트 안내
- **컬럼 불일치:** 자동 매핑 시도
