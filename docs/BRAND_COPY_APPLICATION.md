# 📝 브랜드 카피 일괄 적용본 (기록장 기준)

## 📋 목적
선택한 이름(기록장) 기준으로 홈페이지 메인 카피, 앱/웹 상단 문구, 공유 카드 워딩 일괄 적용

---

## 🏷️ 브랜드 정보 (확정)

**브랜드명:** 기록장 (Girok)

**슬로건:** "팀의 기록이 사람을 모읍니다"

**브랜드 문장:** "우리는 팀을 홍보하지 않습니다. 팀의 기록을 남깁니다."

---

## 1️⃣ 홈페이지 메인 카피

### Hero 영역 (상단)

**H1:**
```
팀의 기록이 사람을 모읍니다
```

**서브 카피:**
```
활동은 이미 충분합니다.
기록장은 그 시간을 조용히 남겨줍니다.
```

**보조 설명 (한 줄):**
```
홍보 없이, 과장 없이,
팀의 일상이 기록으로 쌓입니다.
```

**CTA:**
- Primary: 기록 남기기
- Secondary: 어떻게 작동하나요

---

### 홈 섹션 요약 카피 (3단)

**① 기록:**
```
활동이 있으면
자동으로 정리됩니다.
```

**② 공개:**
```
팀마다 하나의
공개 기록장이 생깁니다.
```

**③ 확산:**
```
기록은
자연스럽게 공유됩니다.
```

**❌ "성장", "확장", "마케팅" 단어 사용 금지**

---

### "왜 기록장인가요?" 섹션

**헤드라인:**
```
홍보하지 않아도
사람은 기록을 봅니다
```

**본문:**
```
단톡방에서 사라지는 활동 대신,
날짜와 맥락이 남는 기록을 만듭니다.

그 기록이 쌓이면
궁금한 사람들이 먼저 찾아옵니다.
```

---

### 사용 대상 명시 (명확)

**이런 팀에게 맞습니다:**
- 주 1회 이상 모이는 팀
- 기록이 남았으면 하는 팀
- 홍보 담당자가 없는 팀

**이런 팀에게는 맞지 않습니다:**
- 단기 모집이 목표인 팀
- 성과를 과장하고 싶은 팀
- 이벤트 중심 운영 팀

---

### Pro 안내 요약 (홈 하단)

**카피:**
```
기록을 멈추지 않게 하는 선택
```

**설명:**
```
Free로 시작하세요.
Pro는 기록과 공유를 자동으로 유지합니다.
```

**CTA:**
```
Pro 알아보기
```

## 3️⃣ 공유 카드 워딩 (자동 생성용)

### 기본형 (1:1)

**상단:**
```
기록장 · 팀 활동 기록
```

**메인:**
```
7월 21일 정기 모임 기록
```

**서브:**
```
무리 없이 잘 마무리했습니다
```

**하단:**
```
포천 · 소흘 60대 FC
```

---

### 대체형 (이미지 없음)

**메인:**
```
각자 페이스로
함께한 하루
```

**서브:**
```
팀 활동 기록
```

---

### 공유 카드 고정 규칙

- CTA 문구 ❌
- 감탄사 ❌
- 이모지 ❌
- 날짜/기록성 ⭕

---

### 공유 메시지 (카카오톡/밴드)

**기본 템플릿:**
```
[팀명]의 [날짜] 활동 기록

[요약 1줄]

기록장에서 더 보기
```

**예시:**
```
소흘 60대 FC의 7월 21일 활동 기록

무더운 날씨였지만, 큰 탈 없이 잘 마무리했습니다.

기록장에서 더 보기
```

---

### OG 메타 태그

**Title:**
```
[팀명] 기록장 - [날짜] 활동 기록
```

**Description:**
```
[요약 1줄] | 기록장에서 [팀명]의 활동 기록을 확인하세요.
```

**예시:**
```
소흘 60대 FC 기록장 - 7월 21일 활동 기록

무더운 날씨였지만, 큰 탈 없이 잘 마무리했습니다. | 기록장에서 소흘 60대 FC의 활동 기록을 확인하세요.
```
