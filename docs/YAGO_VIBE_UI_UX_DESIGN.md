### UI 컴포넌트 명세

#### 1. 조직 컨텍스트 바 (상단 고정)

**위치**: 화면 최상단  
**높이**: 64px  
**배경**: #FFFFFF  
**하단 보더**: 1px solid #E5E7EB  
**상태**: 상단 고정 (sticky)

**구성 요소**:
- 왼쪽: 현재 선택된 조직 표시
  - 텍스트: "협회" 또는 "내 팀"
  - 조직명: "노원구축구협회" (선택된 경우)
  - 아이콘: 드롭다운 화살표 (▼)
- 오른쪽: 전환 탭
  - "협회" 탭 / "내 팀" 탭
  - 선택된 탭: 배경 #F3F4F6, 텍스트 #111827
  - 미선택 탭: 배경 투명, 텍스트 #6B7280

**상호작용**:
- 탭 클릭 시 조직 컨텍스트 전환
- 조직명 클릭 시 조직 목록 표시 (드롭다운)

---

#### 2. 운동장 대관 카드 (최상단, 가장 크게)

**위치**: 조직 컨텍스트 바 아래 24px  
**높이**: 160px  
**너비**: 전체 (좌우 16px 여백)  
**배경**: 선형 그라데이션 (#ECFDF5 → #FFFFFF)  
**보더**: 1px solid #D1FAE5  
**보더 반경**: 12px  
**패딩**: 24px

**구성 요소**:
- 아이콘: 🏟️ (24px)
- 제목: "운동장 대관" (20px, Bold, #111827)
- 요약 정보:
  - "이번 주 12개 대관" (16px, Medium, #059669)
  - "회원팀 8개 · 아카데미 3개 · 비회원 1개" (14px, Regular, #6B7280)
- 우측 상단: "대관 관리 →" 링크 (14px, Medium, #059669)

**강조 효과**:
- 녹색 액센트 (#059669) 사용
- 배경 그라데이션으로 시각적 강조
- 다른 카드보다 40px 더 크게 (120px → 160px)

### UX 분기 로직

#### 조직 컨텍스트 전환

**"협회" 선택 시**:
- 협회 소속 팀 목록 표시
- 회원팀 / 비회원팀 / 아카데미 섹션 모두 표시
- 운동장 대관 카드 표시

**"내 팀" 선택 시**:
- 비소속 팀만 표시
- 운동장 대관 카드 숨김 (또는 비활성화)

### 화면 전환 흐름

**운동장 대관 카드 클릭**:
- → 운동장 대관 메인 화면으로 이동
- 전환 애니메이션: 슬라이드 오른쪽

**팀 카드 클릭**:
- → 팀 상세 화면으로 이동
- 전환 애니메이션: 페이드 인

**"전체 보기" 클릭**:
- → 전체 팀 목록 화면으로 이동
- 전환 애니메이션: 슬라이드 오른쪽

- **위치**: 모든 화면 상단
- **프로퍼티**: currentOrg, orgs, onOrgChange
- **스타일**: 상단 고정, 높이 64px

#### 2. FacilityBookingCard
- **위치**: 메인 화면 최상단
- **프로퍼티**: bookingCount, memberCount, academyCount, nonMemberCount
- **스타일**: 높이 160px, 녹색 액센트

#### 3. FacilityCard
- **위치**: 대관 메인 화면
- **프로퍼티**: name, location, bookingCount, isPriority, type
- **스타일**: 높이 140px, 우선 시설은 녹색 보더

#### 4. TeamTypeStatusCard
- **위치**: 시설 상세 화면
- **프로퍼티**: teamType (member/academy/nonMember)
- **스타일**: 팀 유형별 다른 색상

#### 5. BookingCalendarSlot
- **위치**: 캘린더 화면
- **프로퍼티**: time, status (member/academy/available), onClick
- **스타일**: 높이 48px, 상태별 색상

#### 6. TeamCard
- **위치**: 팀 목록
- **프로퍼티**: name, memberCount, type (member/academy/nonMember)
- **스타일**: 높이 100px, 타입별 색상
