# 🔒 실사용자 5명 테스트 진행 스크립트 (말하지 말고 관찰)

## 목적 (한 줄)

**QR로 들어온 초면 사용자가 설명 없이 '보고 → 행동 → 가입 → 복귀'까지 가는지 검증**

---

## 테스트 준비 (2분)

### 테스트 대상
- 서비스 모르는 사람 5명

### 기기
- 본인 스마트폰

### 안내 규칙

- ❌ 설명 금지
- ❌ 힌트 금지
- ⭕ 막히면 "계속 해보세요"만 말함

---

## STEP A — QR 진입 (가장 중요)

### 진행자 멘트 (이 말만 함)

**"이 QR 한 번 찍어보세요."**

### 관찰 포인트

- [ ] QR 스캔 후 3초 안에 화면 이해?
- [ ] 첫 화면에서 스크롤 없이 다음 행동 보임?
- [ ] "이게 뭐 하는 앱인지" 묻지 않는가?

### 합격 기준

- ✅ 멈춤 없이 화면 탐색 시작
- ✅ 혼란스러워하지 않음

---

## STEP B — 무가입 탐색

### 진행자 멘트

**없음** (말하지 않음)

### 관찰 포인트

- [ ] 로그인/가입 요구를 찾는가? (찾으면 실패 신호)
- [ ] 상품/채팅을 자연스럽게 눌러보는가?
- [ ] "일단 볼 수 있네?" 반응이 나오는가?
- [ ] 탐색 시간 (몇 분?)

### 합격 기준

- ✅ 1분 이상 자발적 탐색
- ✅ 가입 요구를 찾지 않음

---

## STEP C — 행동 시도

### 진행자 멘트

**없음**

### 유도 행동 (손짓만 가능)

- 채팅 보내기
- 판매하기
- 찜

### 관찰 포인트

- [ ] Lazy Signup 모달 등장 시 표정 변화
- [ ] "아…" 같은 부정 반응?
- [ ] 모달 문구를 읽지 않고도 CTA 누르는가?
- [ ] 모달 닫기 시도?

### 합격 기준

- ✅ 거부감 없이 "가입하고 계속" 클릭
- ✅ 부정 반응 없음

---

## STEP D — 가입 UX

### 진행자 멘트

**없음**

### 관찰 포인트

- [ ] 가입 중 이탈 시도?
- [ ] "귀찮다" 반응?
- [ ] 가입 완료까지 걸린 시간 (10초 내?)
- [ ] 중간 설명/프로필 요구에 멈춤?

### 합격 기준

- ✅ 끊김 없이 가입 완료
- ✅ 평균 가입 시간 10초 이내

---

## STEP E — 복귀 확인 (핵심)

### 진행자 멘트

**없음**

### 관찰 포인트

- [ ] 가입 후 아까 하려던 행동이 자동 실행되는가?
- [ ] 사용자가 "어?" 하고 놀라지 않는가?
- [ ] "다시 찾아야 하나?" 반응 없는가?
- [ ] 자연스럽게 행동 완료하는가?

### 합격 기준

- ✅ 자연스럽게 행동 완료
- ✅ 놀라움/혼란 없음

---

## STEP F — 종료 질문 (딱 1개만)

### 마지막에 이것만 물어본다

**"중간에 불편했던 순간 있었어요?"**

- ❌ "어땠어요?"
- ❌ "이해됐어요?"
- ❌ "개선점은?"

---

## 기록 방식 (초경량)

### 각 사용자마다 이것만 적는다

**예시:**

```
1번: QR 이해 O / 가입 망설임 X / 복귀 O
2번: 채팅에서 잠깐 멈춤 / 나머지 OK
3번: 전 과정 OK / 복귀에서 "어?" 반응
4번: QR → 탐색 → 가입 → 복귀 모두 자연스러움
5번: 가입 화면에서 2초 멈춤 / 나머지 OK
```

### 기록 템플릿

```
[사용자 번호]
- QR 진입: O / X / 멈춤(몇 초)
- 탐색: O / X / 시간(몇 분)
- 행동 시도: O / X / 망설임 있음
- 가입: O / X / 시간(몇 초) / 이탈 시도
- 복귀: O / X / 놀라움
- 전체: OK / NG / 특이사항
```

---

## 합격 판정 기준 (냉정)

### 5명 중:

- ✅ **4명 이상**이 설명 없이 끝까지 가면 → **출시 OK**
- ⚠️ **2명 이상**이 같은 지점에서 멈추면 → **그 지점만 UI 1줄 수정**
- ❌ 구조 수정
- ❌ 기능 추가
- ⭕ **문구/순서 미세 조정만**

---

## 실패 포인트별 대응

### 실패 포인트 1: QR 진입 후 멈춤

**원인:** 첫 화면이 불명확  
**대응:** 상단 문구 1줄 추가

### 실패 포인트 2: 탐색 중 가입 요구를 찾음

**원인:** UI에서 가입 버튼이 너무 눈에 띔  
**대응:** 가입 버튼 위치/크기 조정

### 실패 포인트 3: Lazy Signup 모달에서 망설임

**원인:** 모달 카피가 부담스러움  
**대응:** 카피 1줄 수정 (구조 변경 안 함)

### 실패 포인트 4: 가입 중 이탈

**원인:** 가입 화면이 복잡하거나 설명이 많음  
**대응:** 불필요한 문구 제거 (입력 필드 조정 안 함)

### 실패 포인트 5: 복귀에서 놀라움/혼란

**원인:** 복귀 로직이 작동하지 않음  
**대응:** `actionResumer.ts` 로직 확인

## 테스트 시 주의사항

### 절대 하지 말 것

- ❌ 설명하기
- ❌ 힌트 주기
- ❌ "여기 눌러보세요" 같은 안내
- ❌ 테스트 중 질문하기
- ❌ 결과를 설명하기

### 해도 되는 것

- ⭕ "계속 해보세요" (막혔을 때만)
- ⭕ 손짓으로 화면 가리키기 (행동 유도)
- ⭕ 종료 질문 1개만

## 최종 판정 체크리스트

- [ ] QR 진입 시도: 5명 모두 성공
- [ ] 탐색 1분 이상: 4명 이상
- [ ] Lazy Signup 모달 클릭: 4명 이상
- [ ] 가입 완료: 4명 이상
- [ ] 복귀 자연스러움: 4명 이상
