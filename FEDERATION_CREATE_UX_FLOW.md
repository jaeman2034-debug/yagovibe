```
[진입 화면]
    ↓
[AI 대화형 생성] ← 메인 플로우

[AI 분석 & 템플릿 추천]

```

## 🤖 화면 2: AI 대화형 생성 (Chat Wizard)

**URL**: `/platform/federations/create/ai`

**목적**: AI가 질문을 통해 협회 정보 수집

**레이아웃**:
```
┌─────────────────────────────────────┐
│  [Header]                           │
├─────────────────────────────────────┤
│                                     │
│  [진행률 표시]                       │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ │
│  [1/5]                              │
│                                     │
│  ┌─────────────────────────────┐   │
│  │ 🤖 AI                        │   │
│  │                              │   │
│  │ 어떤 종목 협회를 만드시나요?  │   │
│  └─────────────────────────────┘   │
│                                     │
│  [답변 옵션]                         │
│  ┌─────┐  ┌─────┐  ┌─────┐        │
│  │ ⚽  │  │ 🏀  │  │ ⚾  │        │
│  │축구 │  │농구 │  │야구 │        │
│  └─────┘  └─────┘  └─────┘        │
│                                     │
│  ┌─────┐  ┌─────┐                  │
│  │ 🏐  │  │기타 │                  │
│  │배구 │  │     │                  │
│  └─────┘  └─────┘                  │
│                                     │
│  [이전] [다음] (비활성)              │
│                                     │
└─────────────────────────────────────┘
```

**질문 시퀀스 (5단계)**:

### 질문 1: 종목
- **질문**: "어떤 종목 협회를 만드시나요?"
- **옵션**: ⚽ 축구, 🏀 농구, ⚾ 야구, 🏐 배구, 기타
- **기타 선택 시**: 텍스트 입력 필드 표시

### 질문 2: 참가 대상
- **질문**: "주 참가 대상은 누구인가요?"
- **옵션**: 
  - 성인 (생활체육)
  - 유소년 (초/중/고)
  - 대학생
  - 직장인
  - 혼합

### 질문 3: 운영 방식
- **질문**: "운영 방식은 어떤가요?"
- **옵션**:
  - 리그형 (정기 리그)
  - 토너먼트형 (대회 중심)
  - 혼합형

### 질문 4: 참가 단위
- **질문**: "팀 단위 참가인가요, 개인 신청인가요?"
- **옵션**:
  - 팀 단위
  - 개인 신청
  - 둘 다 가능

### 질문 5: 지역 기반
- **질문**: "지역 기반인가요, 학교 기반인가요?"
- **옵션**:
  - 지역 기반 (시/구/군)
  - 학교 기반
  - 기타

**상태 관리**:
- 각 질문별 답변 저장
- 진행률 표시 (1/5, 2/5, ...)
- 이전 버튼: 2단계부터 활성화
- 다음 버튼: 답변 선택 시 활성화

**액션**:
- 답변 선택 → 자동으로 다음 질문으로 이동 (또는 다음 버튼 활성화)
- `이전` 클릭 → 이전 질문으로 이동
- `다음` 클릭 → 다음 질문으로 이동
- 5단계 완료 → 화면 3 (AI 분석)으로 이동

## 📊 데이터 구조

### AI 대화형 수집 데이터
```typescript
interface AIConversationData {
  sport: string;              // 종목
  targetAudience: string;     // 참가 대상
  operationType: string;      // 운영 방식
  participationUnit: string;  // 참가 단위
  baseType: string;          // 지역/학교 기반
```
