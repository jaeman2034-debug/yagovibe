# 📢 팀 공지: MVP 범위 고정

> 슬랙/노션에 그대로 붙여서 사용

---

## 🎯 현재 MVP 범위

**현재 MVP는
'음성으로 주변 장소를 빠르게 찾는 지도'입니다.

이 목적에 직접 기여하지 않는 기능은
전부 출시 이후 논의합니다.**

---

## ✅ MVP에 포함되는 것

* 음성 검색 버튼
* 지도 핀 표시
* 결과 카드 1개
* 길찾기 연결

---

## ❌ MVP에 포함되지 않는 것

* AI 추천
* 상태 추론
* 자동 네비게이션
* 리스트
* 즐겨찾기
* 로그인
* 설정

---

## 🧪 기능 추가 전 질문

> "이거 없으면
> 사용자가
> 말로 장소 찾는 데 실패해?"

* YES → 고려
* NO → 보류

---

## 📋 개발 중 체크리스트

### 매일 아침
- [ ] 오늘 작업이 MVP 범위 내인가?

### 기능 제안 시
- [ ] 말로 주변 장소 찾기에 직접 필요함?
- [ ] 없으면 음성 검색이 작동 안 함?
- [ ] 사용자가 "뭘 말해야 하지?" 질문하게 만듦?

**3개 모두 YES → 넣기**
**하나라도 NO → 나중**

---

## 🚫 즉시 컷 문구

이 말이 나오면 **무조건 STOP**:

* "AI로 하면 더 멋질 것 같은데요"
* "사용자 상태를 알면 좋지 않을까요?"
* "추천도 같이 보여주면…"
* "리스트도 같이 넣죠"
* "로그인 있으면 좋을 듯"

---

## ⏱️ 출시 기준

* [ ] 키보드 없이 찾을 수 있나?
* [ ] 말하면 바로 핀이 찍히나?
* [ ] 카드가 1개만 나오나?
* [ ] "편하다"라는 말이 나왔나?

**전부 YES면 출시**

---
