# 📘 운영 가이드 — 토너먼트 Phase & 승인 팀 관리 (관리자용)

## 🎯 목적

- 관리자가 실수 없이 토너먼트를 진행하게 한다
- "왜 버튼이 안 눌리지?" 같은 문의를 사전에 차단한다
- 개발자 호출 없이 운영 가능하게 한다

---

## 1️⃣ 현재 단계 확인 방법

관리 화면에서 항상 아래 3가지를 먼저 확인한다.

### ✅ 확인 항목

1. **현재 단계 (Phase)**
   - 예: 팀원 등록 중 / 팀원 명단 잠금 / 체크인 시작

2. **승인된 팀 수**
   - 승인됨(APPROVED) 상태의 팀 개수

3. **최근 단계 변경 기록**
   - 누가 / 언제 / 어떤 단계로 변경했는지

👉 **이 3개가 항상 화면에 보이게 유지한다.**

---

## 2️⃣ 단계별 운영 규칙 (이것만 기억하면 됨)

### 🔓 팀원 등록 중 (ROSTER_OPEN)

**할 수 있는 것**:
- 팀 승인 / 반려
- 팀원 수정

**할 수 없는 것**:
- 체크인 시작

**다음 단계로 가는 조건**:
- 승인된 팀 ≥ 1

**❗ 중요**:
- 승인 팀이 0개면 → "팀원 명단 잠금" 버튼이 비활성화됨 (정상)

---

### 🔒 팀원 명단 잠금 (ROSTER_LOCKED)

**의미**:
- 팀 구성 완전 고정
- 더 이상 팀/팀원 변경 불가

**할 수 있는 것**:
- 체크인 준비

**주의**:
- 이 단계 이후 되돌릴 수 없음

**❗ 중요**:
- "이미 잠금됨" 메시지가 나오면 → 정상 상태 (이미 처리된 요청)

---

### ✅ 체크인 시작 (CHECKIN_OPEN)

**의미**:
- 실제 경기/행사 시작 준비

**전제**:
- 팀 명단 확정 완료

**이후 단계**:
- (대진 생성, 경기 시작 등 운영 정책에 따름)

---

## 2️⃣ 일반적인 작업 흐름

### Step 1: 팀원 등록 시작
1. 대회 상세 페이지 접속
2. "팀원 등록 시작" 버튼 클릭
3. Phase가 `ROSTER_OPEN`으로 변경됨

### Step 2: 팀 승인
1. "참가 신청 관리" 탭 이동
2. 승인할 팀 선택
3. "승인" 버튼 클릭
4. **중요**: 최소 1개 팀은 반드시 승인 필요

### Step 3: 팀원 명단 잠금
1. "팀원 등록" 탭으로 이동
2. "팀원 명단 잠금" 버튼 클릭
3. **조건**: 승인된 팀 ≥ 1개
4. Phase가 `ROSTER_LOCKED`로 변경됨

---

## 3️⃣ 자주 나오는 메시지 해석표 (운영자용)

| 메시지 | 의미 | 조치 |
|--------|------|------|
| 이미 진행 중입니다 | 이미 같은 단계 | 아무것도 안 해도 됨 |
| 이미 잠금되었습니다 | 이미 잠금 완료 | 정상 |
| 승인된 팀이 필요합니다 | 승인 팀 0 | 참가 신청 관리로 이동 |
| 권한이 없습니다 | 관리자 아님 | 권한 확인 |
| 잘못된 단계 전이입니다 | 순서를 건너뛰려고 시도 | 순서대로 진행 |

---

## 4️⃣ 운영 중 체크 포인트 (하루 1회)

다음 항목을 하루 1회 확인한다:

- [ ] 승인 팀 수가 0인데 잠금 상태가 아닌지
- [ ] 최근 phase 변경이 1회만 발생했는지
- [ ] 이상한 에러 메시지 반복 발생 여부

👉 **이상 시**:
- 되돌리려 하지 말고
- 개발팀에 "현재 단계 + 승인팀 수 + 변경 시각" 전달

---

## 5️⃣ 절대 하지 말 것 ❌

1. **DB 직접 수정**
2. **새로고침 연타로 버튼 강제 시도**
3. **승인 팀 없이 단계 진행 요청**

**위 행동은 시스템이 막도록 설계되어 있음**

막힌다면 "정상"이다.

---

## 6️⃣ 운영자에게 전달할 한 줄 요약

**"승인 팀이 있어야 잠글 수 있고, 잠그면 되돌릴 수 없으며, 버튼이 안 눌리면 이유가 화면에 나온다."**

---

## 7️⃣ Phase별 체크리스트 (상세)

### ROSTER_OPEN → ROSTER_LOCKED

**사전 확인**:
- [ ] 승인된 팀 ≥ 1개
- [ ] 모든 승인 팀의 팀원 수가 정상 범위 내 (MIN_PLAYERS ~ MAX_PLAYERS)

**실행**:
- [ ] "팀원 명단 잠금" 버튼 클릭
- [ ] Phase 변경 확인

**사후 확인**:
- [ ] Phase가 `ROSTER_LOCKED`로 변경됨
- [ ] 모든 팀의 `rosterLocked: true` 확인

## 8️⃣ FAQ

### Q: Phase를 되돌릴 수 있나요?
A: 현재는 되돌릴 수 없습니다. 신중하게 진행하세요.

### Q: 여러 관리자가 동시에 작업해도 되나요?
A: 가능하지만, 혼란을 피하기 위해 한 명만 실행하는 것을 권장합니다.

### Q: Phase 변경 후 취소할 수 있나요?
A: 불가능합니다. Phase 변경은 되돌릴 수 없으므로 신중하게 진행하세요.

### Q: 승인된 팀 수를 확인하는 방법은?
A: "팀원 등록" 탭에서 승인된 팀 수가 표시됩니다.
