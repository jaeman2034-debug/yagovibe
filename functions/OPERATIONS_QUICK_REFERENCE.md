# ⚡ 운영 빠른 참조 (한 눈에 보기)

## 📌 핵심 한 줄 요약

**"승인 팀이 있어야 잠글 수 있고, 잠그면 되돌릴 수 없으며, 버튼이 안 눌리면 이유가 화면에 나온다."**

---

## 🔍 현재 상태 확인 (3가지)

1. **현재 단계 (Phase)**: 팀원 등록 중 / 잠금 / 체크인 / 경기
2. **승인된 팀 수**: APPROVED 상태 팀 개수
3. **최근 변경 기록**: 누가 / 언제 / 어떤 단계로

---

## 📊 Phase 전이 흐름

```
[팀원 등록 중] ─→ [팀원 명단 잠금] ─→ [체크인 시작] ─→ [경기 진행]
승인 팀 ≥ 1 필요   자동               자동              최종
```

---

## 🚫 금지 사항

- ❌ DB 직접 수정
- ❌ 새로고침 연타로 버튼 강제 시도
- ❌ 승인 팀 없이 단계 진행 요청

**막힌다면 "정상"이다.**

---

## 💬 메시지 해석

| 메시지 | 의미 | 조치 |
|--------|------|------|
| 이미 진행 중입니다 | 이미 같은 단계 | 아무것도 안 해도 됨 |
| 이미 잠금되었습니다 | 이미 잠금 완료 | 정상 |
| 승인된 팀이 필요합니다 | 승인 팀 0 | 참가 신청 관리로 이동 |
| 권한이 없습니다 | 관리자 아님 | 권한 확인 |

---

## ✅ 하루 1회 체크

- [ ] 승인 팀 수가 0인데 잠금 상태가 아닌지
- [ ] 최근 phase 변경이 1회만 발생했는지
- [ ] 이상한 에러 메시지 반복 발생 여부

**이상 시**: 개발팀에 "현재 단계 + 승인팀 수 + 변경 시각" 전달
