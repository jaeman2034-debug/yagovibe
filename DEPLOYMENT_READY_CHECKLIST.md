### ✅ 3️⃣ AI 섹션
- 접힘 기본 / 선택 실행 / 적용 승인 구조 ✅
- 실패 시 toast only ✅
- → AI 리스크 0 설계 유지

### ✅ 4️⃣ 상세페이지
- 이미지 크기 제한 (maxHeight: 420px) ✅
- 비율 고정 (4 / 3) + object-contain ✅
- 핵심 정보 1스크린 노출 가능 상태 ✅

### ✅ 5️⃣ 스크롤
- min-h-screen + flex 구조 ✅
- 하단 padding 충분 (pb-24, pb-128) ✅
- h-screen, overflow-hidden 없음 ✅
- → 모바일/키보드/탭바 모두 안전
