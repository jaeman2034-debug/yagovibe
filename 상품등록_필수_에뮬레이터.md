# 🛒 상품 등록 테스트 - 필수 에뮬레이터

## ✅ 현재 상태

- ✅ **Storage**: ON (포트 9199) - 이미지 업로드용
- ❌ **Firestore**: OFF - **필수!** 상품 데이터 저장용
- ❌ **Auth**: OFF - **필수!** 사용자 인증용  
- ❌ **Functions**: OFF - **필수!** AI 이미지 분석용

## 🚀 해결 방법

### 에뮬레이터 재시작 (권장)

현재 실행 중인 에뮬레이터를 종료하고 다시 시작하세요:

```bash
# 터미널에서 Ctrl+C로 현재 에뮬레이터 종료
# 그 다음:
firebase emulators:start
```

이렇게 하면 `firebase.json`에 설정된 모든 에뮬레이터가 자동으로 시작됩니다:
- ✅ Storage (포트 9199)
- ✅ Firestore (포트 8080)
- ✅ Auth (포트 9099)
- ✅ Functions (포트 5001)

### 필요한 에뮬레이터만 명시적으로 시작

```bash
firebase emulators:start --only storage,firestore,auth,functions
```

## 📋 상품 등록 테스트에 필요한 기능

### 1. 사용자 인증 (Auth 필요)
- 로그인/회원가입
- 사용자 정보 확인

### 2. 이미지 업로드 (Storage 필요) ✅
- 상품 이미지 업로드
- 이미지 URL 생성

### 3. AI 이미지 분석 (Functions 필요)
- 이미지 분석 함수 호출
- 카테고리, 태그, 가격 예측

### 4. 상품 데이터 저장 (Firestore 필요)
- `marketProducts` 컬렉션에 저장
- 실시간 업데이트
