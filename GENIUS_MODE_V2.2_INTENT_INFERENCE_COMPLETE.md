# 🔥 천재 모드 2.2 - 의도 추론 완성

## ✅ 구현 완료

### 1. 의도 추론기

**기능**:
- 사용자가 말 안 해도 의도를 추측
- 상황(context)과 기억(memory) 기반 추론
- 자동 프로필 업데이트

**추론 규칙**:
- 저녁 + 펍 선호 → 경기 관람 의도
- 걸음 수 > 2000 → 운동 의도
- 위치: gym/park → 운동 의도
- 위치: stadium → 경기 관람 의도
- 위치: cafe → 휴식 의도
- 선호 시간대 + 선호 태그 → 의도 추론
- 주말 낮 → 운동, 주말 저녁 → 경기 관람
- 평일 저녁 → 휴식
- 비 오면 → 실내 휴식
- 맑은 낮 → 운동

**파일**: `src/utils/intentInference.ts` (신규)

---

### 2. 재랭킹 연결

**기능**:
- 추론된 의도로 프로필 업데이트
- 업데이트된 프로필로 재랭킹
- 의도 변경 시 로그 출력

**파일**: `src/utils/intentInference.ts`, `src/components/map/MapController.tsx`

---

### 3. 문장에 반영

**기능**:
- 추론된 의도에 맞는 문장 생성
- "몸 쓰기 좋은 루트예요 ✨" (운동)
- "경기 보기 전에 딱 ✨" (관람)
- "쉬는 날 무드 ✨" (휴식)
- "혼자만의 시간 ✨" (혼자)

**파일**: `src/utils/intentInference.ts`, `src/utils/courseBuilder.ts`

---

## 🎯 사용자 경험 흐름

```
1. 상황 감지
   - 시간: 저녁 7시
   - 기억: 펍 선호
   ↓
2. 의도 추론
   - "저녁 + 펍 선호" → 경기 관람 의도
   ↓
3. 프로필 업데이트
   - todayIntent: "watch" → "watch"
   ↓
4. 재랭킹
   - 경기 관람 의도 반영하여 재랭킹
   ↓
5. 문장 생성
   - "경기 보기 전에 딱 ✨"
   ↓
6. 사용자 체감
   - "운동 가는 중이시죠?"
   - "퇴근 후라 이런 루트"
   - "말 안 해도 알아듣는 느낌"
   ↓
7. 진짜 "작은 분신" ✨
```
