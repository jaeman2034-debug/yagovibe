# ✅ 1번 MVP 기능 목록 + 우선순위 (절대 안 흔들 버전)

## 🎯 1번 MVP의 한 줄 목표

> **"키보드 없이 말만 해도
> 내 주변 장소·시설을
> 지도에서 바로 찾을 수 있다."**

이 문장에 **안 들어가는 기능은 전부 버린다.**

---

# 🧱 MVP 기능을 3단으로 나눈다

* **P0 = 없으면 서비스가 성립 안 됨**
* **P1 = 있으면 확실히 좋아짐**
* **P2 = 성공 후에만**

---

## 🔴 P0 — 반드시 필요한 핵심 기능 (이게 MVP)

### 1️⃣ 음성 검색 (가장 중요)

**기능**

* 버튼 or 하단 영역 눌러서 음성 입력
* 또는 자동 listening (초기엔 버튼 OK)

**지원 문장 예**

* "근처 축구장"
* "헬스장"
* "배드민턴장"
* "카페"

👉 **자연어 완벽 이해 ❌**
👉 **키워드 매칭 ⭕ (처음엔 충분)**

**성공 기준:**
- [ ] 버튼 클릭 시 음성 입력 시작
- [ ] 키워드 매칭으로 카테고리 인식
- [ ] 인식 실패 시 자연스럽게 복귀

---

### 2️⃣ 내 위치 기반 장소 검색

**기능**

* 현재 위치 기준 반경 검색
* 카테고리별 장소 조회

**필수 카테고리 (초기)**

* ⚽ 축구장
* 🏋️ 헬스장
* 🏸 배드민턴장
* ☕ 카페

👉 종목 늘리는 건 **나중**

**성공 기준:**
- [ ] 현재 위치 정확히 획득
- [ ] 반경 2km 내 장소 검색
- [ ] 4개 카테고리 모두 지원

---

### 3️⃣ 지도에 핀 표시

**기능**

* 검색 결과를 지도 핀으로 표시
* 가장 가까운 장소 1개 강조

❌ 리스트 먼저 보여주기
⭕ 지도 먼저 보여주기

**성공 기준:**
- [ ] 검색 결과 핀 표시 (3-5개)
- [ ] 가장 가까운 핀 강조 (크기/색상)
- [ ] 지도 자동 줌 (모든 핀 보이도록)

---

### 4️⃣ 결과 카드 1개 (하단)

**표시 정보 (최소)**

* 장소 이름
* 거리
* 간단 상태 (예: 무료 / 실내 / 운영중)

👉 카드 **1개만**
👉 스크롤 리스트 ❌

**성공 기준:**
- [ ] 가장 가까운 장소만 카드 표시
- [ ] 필수 정보 3개 포함
- [ ] 카드 클릭 시 상세 정보 (P1)

---

### 5️⃣ 길찾기 연결

**기능**

* "여기로 가기" 버튼
* 또는 "여기로 가자" 음성

👉 외부 지도 연결도 OK (초기엔)

**성공 기준:**
- [ ] 버튼 클릭 시 길찾기 시작
- [ ] 음성 명령으로도 길찾기 가능
- [ ] 외부 지도 앱 연결 (선택)

---

## 🟡 P1 — 있으면 확 좋아지는 기능 (하지만 MVP는 아님)

### 6️⃣ 조건 검색 (간단)

* "지금 여는 헬스장"
* "무료 축구장"

👉 조건은 **1개만 허용**

**구현 범위:**
- 시간 조건: "지금 여는", "밤에도"
- 가격 조건: "무료", "저렴한"
- 기타: "실내", "야외"

---

### 7️⃣ 장소 상세 정보

* 운영 시간
* 주소
* 간단 후기 요약 (있으면)

👉 카드 클릭 시만 노출

**구현 범위:**
- 기본 정보: 주소, 운영시간
- 후기 요약: AI 생성 (간단)
- 길찾기 버튼

---

### 8️⃣ 음성 재검색

* "다른 데"
* "조금 더 가까운 곳"

👉 UX 부드러워짐

**구현 범위:**
- "다른 곳" → 같은 카테고리 다른 장소
- "더 가까운 곳" → 반경 축소
- "더 먼 곳" → 반경 확대

---

## ⚪ P2 — 지금은 절대 하면 안 되는 것

(이거 하면 1번 망가짐)

* ❌ 상태 추론 ("쉬는 중" 등)
* ❌ 자동 추천
* ❌ 이동 예측
* ❌ 팀/운동 기록
* ❌ AI 동행 멘트
* ❌ 리스트 무한 스크롤
* ❌ 복잡한 필터
* ❌ 비교 기능
* ❌ 즐겨찾기

👉 **전부 2번 이후**

---

# 🧪 1번 MVP 성공 판정 기준 (이게 전부)

이 4개만 YES 나오면 **1번 성공**이다.

1. ❓ 키보드 없이 **말만으로** 장소 찾았나?
2. ❓ 결과가 **바로 지도에** 나왔나?
3. ❓ 어디 갈지 **3초 안에 결정**했나?
4. ❓ "이거 편하다"라는 말이 나왔나?

**측정 지표:**
- DAU ❌
- MAU ❌
- **"편하다" 한마디 ⭕**
- **3초 내 결정률 ⭕**

---

# 🗺️ 지금 화면에 대입하면

지금 화면에서 MVP로 필요한 건:

* 상단: **"말로 주변 장소를 찾아보세요"**
* 중앙: 지도 + 핀
* 하단: 🎙️ **말로 장소 찾기**
* 결과: 핀 + 카드 1개

👉 이 이상 넣으면 **과설계**

---

# 🧠 핵심 원칙

> **1번 MVP는
> "똑똑함"을 증명하는 단계가 아니다.
>
> "편리함"을 증명하는 단계다.**

---

# 📊 기능 우선순위 요약

## P0 (반드시)
1. 음성 검색
2. 내 위치 기반 검색
3. 지도 핀 표시
4. 결과 카드 1개
5. 길찾기 연결

## P1 (있으면 좋음)
6. 조건 검색
7. 장소 상세 정보
8. 음성 재검색

## P2 (하지 않음)
- 상태 추론
- 자동 추천
- 이동 예측
- 팀/운동 기록
- AI 동행 멘트
- 리스트 무한 스크롤

---

# ✅ MVP 범위 확정 체크리스트

- [ ] P0 기능 5개 모두 포함
- [ ] P1 기능은 선택적
- [ ] P2 기능은 전부 제외
- [ ] 성공 기준 4개 명확
- [ ] 화면 구성 최소화
