```

전화번호만 입력하면
비밀번호나 이메일 없이 바로 시작하는 방식인데,
아직 초기라 많이 부족합니다.

혹시 써보시고
"여기서 막힌다 / 이건 별로다"
같은 피드백 주실 분 계실까요?

👉 링크: https://yourapp.com
```

```
이메일 인증, 비밀번호 설정 없이
번호만 입력하면 바로 시작할 수 있어요.

아직 초기 단계라 부족한 점이 많지만,
혹시 써보시고 피드백 주실 분 계실까요?

👉 https://yourapp.com
```

### 🅱️ "문제 공감형" (비개발자 커뮤니티)

**쓰는 곳:** 네이버 카페, 슬랙/디스코드 일반 커뮤니티

```
혹시 앱 가입할 때
이메일 인증 + 비번 만들기 귀찮아서
그냥 나간 적 있으신가요?

저도 그게 너무 싫어서
번호만 넣으면 바로 쓸 수 있는 걸 만들어봤어요.
설치도 없어요.

필요한 분만 한 번 써보세요.
👉 https://yourapp.com
```

**변형 버전:**

```
가입 절차 때문에 앱을 포기한 적 있으신가요?

이메일 인증하고, 비밀번호 만들고...
이미 지쳐서 포기한 적이 많았어요.

그래서 번호만 입력하면 바로 쓸 수 있는
서비스를 만들어봤습니다.

혹시 써보시고 어떠신지 알려주세요!
👉 https://yourapp.com
```

### 🅲 "제작 스토리형" (장기 유입)

**쓰는 곳:** 브런치, 미디엄, 노션 공개 페이지

```
서비스 만들면서 제일 힘들었던 게
기능이 아니라 '가입'이더라고요.

아무리 좋아도
가입에서 막히면 끝이라서,
아예 가입을 없애보자는 생각으로
번호 인증만 남겼습니다.

혹시 비슷한 문제 겪는 분 있으면
경험 공유해주셔도 좋고,
써보고 욕해주셔도 좋습니다.

👉 https://yourapp.com
```

**변형 버전 (더 긴 버전):**

```
# 회원가입 때문에 유저를 잃지 않기 위해

## 문제

서비스를 만들면서 가장 힘들었던 건 기능이 아니라 '가입'이었어요.

아무리 좋은 기능을 만들어도,
가입 단계에서 유저가 이탈하면 끝이더라고요.

이메일 인증하고, 비밀번호 만들고, 프로필 입력하고...
이미 지쳐서 포기하는 경우가 너무 많았습니다.

## 해결

그래서 아예 가입을 없애보자는 생각으로
전화번호 인증만 남긴 서비스를 만들어봤어요.

번호만 입력하면 바로 시작할 수 있어서,
회원가입 때문에 포기하는 일이 없어요.

## 현재 상태

아직 초기 단계라:
- 기능이 부족할 수 있어요
- UX가 완벽하지 않을 수 있어요
- 버그가 있을 수 있어요

하지만 핵심 기능은 동작하고,
지속적으로 개선하고 있습니다.

## 피드백 요청

비슷한 문제를 겪고 계신 분들,
혹시 써보시고 뭐가 불편한지 알려주시면
정말 감사하겠습니다!

👉 https://yourapp.com
```

### 💡 Why Section

```
왜 이렇게 만들었나요?

좋은 서비스도
가입이 복잡하면 시작조차 못 한다는 걸
너무 많이 봤기 때문입니다.

이메일 인증을 기다리고,
비밀번호를 만들고,
프로필을 입력하는 동안
이미 지쳐서 포기하는 경우가 많았어요.

그래서 전화번호만 입력하면
바로 시작할 수 있게 만들었습니다.
```

### ⚙️ How Section

```
어떻게 작동하나요?

✅ 실제 번호 인증 (보안 OK)
SMS 인증을 통해 실제 전화번호를 확인합니다.

🔒 필요한 정보만 최소 수집
인증에 필요한 정보만 수집하고, 불필요한 정보는 요청하지 않습니다.

⚡ 평균 가입 시간 1분 이내
복잡한 절차 없이 빠르게 시작할 수 있습니다.
```

---

### 🙋 FAQ Section

```
자주 묻는 질문

Q. 전화번호 안전한가요?
인증에만 사용하며, 다른 용도로 쓰지 않습니다.
인증 후에는 저장하지 않아요.

Q. 스팸 오지 않나요?
인증 외에는 메시지를 보내지 않습니다.
마케팅 메시지도 보내지 않아요.

Q. 무료인가요?
현재는 무료로 제공 중입니다.
향후 유료 기능이 추가될 수 있지만,
기본 기능은 계속 무료입니다.

Q. 왜 이메일 가입이 없나요?
더 빠르게 시작하는 게 목적이기 때문이에요.
이메일 인증을 기다리는 시간을 없애고,
바로 서비스를 사용할 수 있게 했습니다.

Q. 비밀번호를 잊어버리면?
비밀번호가 없어요.
전화번호 인증만으로 로그인하므로,
비밀번호를 기억할 필요가 없습니다.
```

1. **피드백 요청형** → 개발자/스타트업 커뮤니티
2. **문제 공감형** → 일반 사용자 커뮤니티
3. **제작 스토리형** → 블로그/장기 유입

1. **친구** → 캐주얼 버전
2. **지인** → 기본형
3. **지인2** → 정중한 버전

1. **Hero** → 첫 인상
2. **Why** → 신뢰 구축
3. **How** → 작동 방식
4. **FAQ** → 우려 해소
5. **CTA** → 행동 유도
