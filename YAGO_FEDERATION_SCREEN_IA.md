## 1️⃣ 홈 화면

### 경로: `/federations/nowon-football` (기본 탭: home)

### 화면 구성

```
┌─────────────────────────────────────────┐
│ 협회 헤더                                │
│ - 협회명                                 │
│ - 지역/종목                              │
│ - 관리자 버튼 (관리자만)                 │
├─────────────────────────────────────────┤
│ 탭 메뉴                                  │
│ 홈 | 협회소개 | 공지 | 대회/리그 | ...   │
├─────────────────────────────────────────┤
│ 메인 비주얼                              │
│ - 협회 슬로건                            │
│ - CTA 버튼 (대회 보기, 경기 일정)       │
├─────────────────────────────────────────┤
│ 요약 카드                                │
│ [진행중 리그] [참가 팀] [총 경기]        │
├─────────────────────────────────────────┤
│ 진행중 대회                              │
│ - 대회 카드 (대회명, 기간, 참가팀 수)   │
│ - [대회 보기] 버튼                       │
├─────────────────────────────────────────┤
│ 공지사항                                 │
│ - 중요 공지 (상단 고정)                  │
│ - 최근 공지 (5개)                        │
├─────────────────────────────────────────┤
│ 다가오는 경기                            │
│ - 오늘 경기                              │
│ - 이번주 경기                            │
│ - 경기 카드 (팀, 시간, 경기장)          │
├─────────────────────────────────────────┤
│ 최근 경기 결과                           │
│ - 최근 5경기                             │
│ - 결과 카드 (스코어, 날짜, 대회명)      │
├─────────────────────────────────────────┤
│ 순위 요약                                │
│ - 현재 순위 TOP 5                        │
│ - [전체 순위 보기] 버튼                  │
├─────────────────────────────────────────┤
│ 대표 참가팀/클럽                         │
│ - 클럽 카드 (로고, 이름, 지역)          │
│ - [전체 팀 보기] 버튼                    │
├─────────────────────────────────────────┤
│ 유소년 섹션                              │
│ - 유소년 프로그램 소개                   │
│ - 아카데미 목록                          │
├─────────────────────────────────────────┤
│ 후원사 배너                              │
│ - 공식 후원사 로고                       │
│ - 협력 업체                              │
└─────────────────────────────────────────┘
```

### 필터/검색

- 없음 (홈 화면은 요약 정보만 표시)

### 액션

- [대회 보기] → 대회/리그 탭으로 이동
- [경기 일정 보기] → 경기일정 탭으로 이동
- [참가 안내] → 규정/자료실 탭으로 이동
- [관리자 로그인] → 관리자 대시보드로 이동 (관리자만)
