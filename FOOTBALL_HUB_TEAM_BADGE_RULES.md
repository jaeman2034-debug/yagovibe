# 축구 허브 팀 카드 배지 표시 규칙 (Association Badge Rules)

**목적**: 축구 허브(`/sports/football`)의 팀 카드에서 협회 관계(산하/연계)를 배지로 표시하여 공식성을 자연스럽게 인지하게 함

---

## 1️⃣ 적용 위치

### 대상 화면
- **축구 허브 메인**: `/sports/football`
  - "팀 / FC 찾기" 카드 클릭 → 팀 검색/목록 페이지
- **팀 검색/목록 페이지**: `/teams/search?type=football`
  - 모든 팀 카드에 적용
- **팀 상세 페이지**: `/teams/{teamId}` (헤더/정보 영역)

### 비대상
- ❌ 관리자 대시보드 (내부 도구)
- ❌ 팀 관리 페이지 (소유자 전용)

---

## 2️⃣ 배지 표시 규칙

### 배지 조건
팀 데이터에 `associationRelation` 필드가 존재하고, `status`가 `'official'` 또는 `'related'`일 때만 표시

```typescript
// 팀 데이터 구조 (참고용)
{
  id: string;
  name: string;
  associationRelation?: {
    associationId: 'assoc-nowon-football',
    status: 'official' | 'related' | 'independent'
  }
}
```

### 배지 종류

| 상태 | 배지 색상 | 배지 텍스트 | 표시 위치 |
|------|----------|------------|----------|
| `official` | 🟢 초록 | 노원구 축구협회 산하 | 팀명 옆 또는 카드 상단 |
| `related` | 🟡 노랑 | 노원구 축구협회 연계 | 팀명 옆 또는 카드 상단 |
| `independent` | 없음 | - | 배지 없음 (일반 팀) |

---

## 3️⃣ UI 배치 규칙

### 팀 카드 내 배지 위치 (옵션 A - 추천)

```
┌─────────────────────────────────┐
│ [팀명] 🟢 노원구 축구협회 산하    │  ← 배지 (팀명 옆, 작은 크기)
│                                 │
│ [팀 정보...]                    │
│ 멤버 수, 활동 지역 등            │
└─────────────────────────────────┘
```

### 팀 카드 내 배지 위치 (옵션 B - 대안)

```
┌─────────────────────────────────┐
│ 🟢 노원구 축구협회 산하          │  ← 배지 (카드 상단, 독립 영역)
│                                 │
│ [팀명]                          │
│ [팀 정보...]                    │
└─────────────────────────────────┘
```

**선택 기준**: 
- 옵션 A (팀명 옆): 공간 효율적, 스캔 용이
- 옵션 B (상단 독립): 시각적 강조, 협회 관계가 주요 정보일 때

**추천**: 옵션 A (팀명 옆, 작은 배지)

---

## 4️⃣ 배지 스타일 규칙

### 배지 크기
- **텍스트 크기**: 작은 크기 (`text-xs` 또는 `text-sm`)
- **아이콘 크기**: 작은 이모지 또는 작은 아이콘
- **전체 배지**: 팀명보다 작게 (주요 정보가 팀명이므로)

### 배지 색상 (Tailwind 기준)
- **OFFICIAL (초록)**: 
  - 배경: `bg-green-100` 또는 `bg-green-50`
  - 텍스트: `text-green-700` 또는 `text-green-600`
- **RELATED (노랑)**: 
  - 배경: `bg-yellow-100` 또는 `bg-yellow-50`
  - 텍스트: `text-yellow-700` 또는 `text-yellow-600`

### 배지 형태
- 둥근 모서리 (`rounded-full` 또는 `rounded-md`)
- 작은 패딩 (`px-2 py-1` 또는 `px-1.5 py-0.5`)
- 인라인 배치 (팀명과 같은 줄)

---

## 5️⃣ 표시 조건 (명확화)

### 표시 조건
```
IF (team.associationRelation?.associationId === 'assoc-nowon-football' AND
    team.associationRelation?.status IN ('official', 'related'))
THEN
    배지 표시
ELSE
    배지 없음 (일반 팀으로 표시)
```

### 동일 협회만 표시
- 현재는 **노원구 축구협회(`assoc-nowon-football`)**만 표시
- 다른 협회는 Phase 4 이후 확장 가능 (현재는 노원구만)

## 7️⃣ 클릭 동작 (기존 유지)

### 배지 클릭
- **동작**: 배지 클릭 시 협회 공식 페이지로 이동 (`/association/assoc-nowon-football`)
- **의도**: "이 협회에 대해 더 알고 싶다" 자연스러운 탐색 경로

### 팀 카드 클릭 (배지 제외 영역)
- **동작**: 기존과 동일 (팀 상세 페이지로 이동)
- **변경 없음**

---

## 8️⃣ 정렬/필터링 영향

### 정렬 규칙
- **기존 정렬 유지** (이름순, 활동일순 등)
- **배지 유무로 정렬하지 않음** (협회 산하가 우선순위가 아니므로)

### 필터링 (Phase 4 이후 확장 가능)
- 현재는 필터링 없음
- Phase 4 이후 "협회 산하 팀만 보기" 필터 추가 가능

---

## 9️⃣ 모바일 반응형

### 모바일 화면
- 배지 크기: 더 작게 (`text-xs`)
- 배치: 팀명 옆 (옵션 A 유지)
- 터치 영역: 배지는 작지만 터치 가능 (협회 페이지 이동)

---

## 🔟 관리자 Edit Mode 영향

### 관리자 화면
- **배지 표시**: Public과 동일하게 표시 (읽기 전용)
- **배지 수정**: ❌ (팀 카드에서 협회 관계 수정 불가)
- **협회 관계 변경**: Phase 5 승인 시스템에서만 처리

## ✅ 성공 기준

1. **공식성 인지**: 허브에서 "노원구 축구협회 산하 팀"이 한눈에 구분되는가?
2. **자연스러움**: 배지가 정보를 방해하지 않고 자연스럽게 읽히는가?
3. **분쟁 차단**: "우리 팀은 산하냐?" 질문이 화면으로 즉시 답변 가능한가?
4. **확장성**: 다른 협회 배지 추가 시 구조 변경 최소화

## 🚫 절대 금지 사항

- ❌ 배지 클릭 시 팀 가입/신청 페이지로 이동

- ❌ 배지 텍스트 커스터마이징 (고정 문구만 사용)
- ❌ 배지 크기 과대 (팀명보다 작게)
