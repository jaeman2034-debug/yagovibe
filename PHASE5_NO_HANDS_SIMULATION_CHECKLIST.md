## STEP 2️⃣ 실패 지점 체크 (이게 핵심)

아래 질문에 **YES가 하나라도 나오면** 그게 다음 작업입니다.

### 체크 질문

1. **"여기서 사람 손 한번 들어갔네?"**
   - [ ] YES → 어떤 단계에서 사람 개입이 필요한지 기록
   - [ ] NO → 통과

2. **"이건 설명 없으면 헷갈리겠다"**
   - [ ] YES → UI/문구 개선 필요 지점 기록
   - [ ] NO → 통과

3. **"이건 아직 엑셀 생각난다"**
   - [ ] YES → 엑셀 대체 기능 부족 지점 기록
   - [ ] NO → 통과

**👉 기능 추가 ❌ / 화면 수정만 ⭕**

## 📋 시뮬레이션 실행 순서

1. **공지** → 예약 게시 → 자동 반영 확인
2. **대회** → 입력 → Public 노출 확인
3. **회비** → 미납자 → 자동 알림 확인
4. **정산** → 영수증 업로드 → 보고서 출력
5. **기부/후원** → 등록 → 영수증 발급

**각 단계마다 실패 지점 체크**
