# "협회 → 팀 → 개인" 전체 사용자 흐름 고정 선언

**목적**: 협회 페이지에서 팀이 어떻게 연결되는지, 팀 생성/승인 흐름이 어디까지 자동인지 명확히 정의

---

## 1️⃣ 핵심 질문에 대한 답변

### 질문
**"노원구 축구협회 공식 페이지가 열리면, 그 다음 사용자는 어디로 가야 하는가?"**

### 답변 (명확한 흐름)

```
협회 공식 페이지
  ↓
1. 정보 확인 (대회/대관/공지)
  ↓
2. 산하 팀 목록 확인 (읽기 전용)
  ↓
3. 선택지:
   - 팀 페이지로 이동 (팀 상세 정보)
   - 축구 허브로 이동 (다른 팀 찾기)
   - 팀 생성 (신규 팀 생성)
```

---

## 2️⃣ 최종 사용자 흐름 (공식 고정안)

### ① 축구 허브 진입 (모든 사용자 공통)

**흐름:**
```
사용자 → 축구 허브 (/sports/football)
  ↓
최상단에 '노원구 축구협회 공식 카드' 표시
  ↓
인식: "아, 여기가 기준이구나"
```

**핵심:**
- 모든 사용자가 처음 보는 것은 협회 공식 카드
- 협회가 최상위 기준점임을 즉시 인지

---

### ② 협회 공식 페이지 진입

**URL:** `/association/assoc-nowon-football`

**사용자 행동:**
- 공식 대회 일정 확인
- 대관 현황 확인
- 협회 공지/스토리 확인

**핵심:**
- 👉 **여기서 끝나는 사용자가 대부분이다**
- 이게 정상이고, 목표다
- 추가 CTA 없음 (정보 확인으로 충분)

---

### ③ 팀으로 흐르는 경우 (의도된 소수)

**사용자 이해:**
- "대회는 협회가 관리"
- "팀은 이 대회에 참여하는 주체"

**흐름:**
```
협회 페이지
  ↓
축구 허브로 돌아감
  ↓
[팀 / FC 찾기]로 이동
  ↓
팀 목록/검색
```

**핵심:**
- **협회 → (허브) → 팀**
- 직접 점프 ❌ / 간접 흐름 ⭕
- 협회 페이지에서 팀으로 직접 이동하는 CTA 없음

---

### ④ 개인 활동

**개인의 위치:**
- 개인은 어디에도 "주인공"으로 나오지 않음
- 개인은 항상:
  - 팀 소속이거나
  - 협회 정보 열람자

**핵심:**
- 👉 개인 중심 UI가 앞에 나오지 않는 이유
- 개인은 소비자/열람자 역할

---

## 3️⃣ 지금 단계에서 명확히 선 긋는 것

### ❌ 지금 하지 않는 것

| 기능 | 상태 | 이유 |
|------|------|------|
| 협회 페이지에서 팀 생성 | ❌ | Phase 4 이후 |
| 협회 페이지에서 가입 신청 | ❌ | Phase 4 이후 |
| 협회 페이지에서 문의 버튼 | ❌ | Phase 4 이후 |
| 협회 페이지에서 승인 플로우 | ❌ | Phase 5 이후 |

**이유:** 이건 전부 Phase 4 이후다.

**핵심:** 협회 페이지는 정보 확인 장소, 행동 유도 장소 아님

---

## 4️⃣ 팀 생성/승인 흐름 (지금 vs Phase 4)

### 현재 상태 (Phase 3-4-1) - 지금은 ❌

#### 팀 생성
- **위치**: 축구 허브 → "우리 팀 관리" 또는 팀 찾기 페이지
- **플로우**: 
  1. 사용자가 팀 생성
  2. 팀이 독립적으로 생성됨
  3. 협회와의 관계는 **자동 연결되지 않음**
- **협회 승인**: ❌ 없음 (수동 관계 설정 필요)

#### 협회 승인 워크플로우
- **신청**: ❌ 없음
- **승인**: ❌ 없음
- **관계 표시**: 수동 설정 또는 Phase 5 이후

---

### Phase 4 상태 - Phase 4에서 ⭕️

#### 팀 생성
- **위치**: 동일 (축구 허브 → 팀 생성)
- **플로우**: 
  1. 사용자가 팀 생성
  2. 팀이 독립적으로 생성됨
  3. 지역 기반으로 **협회 제안** (자동)
     - 예: 노원구 지역 → "노원구 축구협회 산하로 등록하시겠습니까?" 제안
- **협회 승인**: ⏸️ 제안만, 승인 워크플로우는 Phase 5

#### 협회 승인 워크플로우 (Phase 5 이후)
- **신청**: ⭕️ 팀이 협회 산하 신청
- **승인**: ⭕️ 협회 관리자 승인
- **관계 표시**: 자동으로 "🟢 노원구 축구협회 산하" 배지 표시

---

## 5️⃣ 전체 사용자 흐름도

### 흐름 1: 정보 확인 (Public)
```
협회 공식 페이지
  ↓ (스크롤)
대회/대관/공지 정보 확인
  ↓
산하 팀 목록 확인 (읽기 전용)
  ↓
[선택지]
  ├─ 팀 페이지로 이동 (팀 상세)
  ├─ 축구 허브로 이동 (다른 팀 찾기)
  └─ 팀 생성 (신규 팀)
```

### 흐름 2: 팀 생성 (회원)
```
축구 허브
  ↓
"우리 팀 관리" 또는 "팀 찾기"
  ↓
팀 생성
  ↓
[Phase 4] 협회 제안 (지역 기반)
  ↓
[Phase 5] 협회 산하 신청 (선택)
  ↓
[Phase 5] 협회 관리자 승인
  ↓
산하 팀으로 표시
```

### 흐름 3: 팀 찾기 (회원/비회원)
```
축구 허브
  ↓
"팀 / FC 찾기"
  ↓
팀 목록 (협회 산하 배지 표시)
  ↓
팀 선택
  ↓
팀 페이지로 이동
```

## 7️⃣ 권한 및 접근 제어

### 협회 페이지
- **Public 접근**: ⭕️ (모든 사용자)
- **산하 팀 목록**: ⭕️ (읽기 전용)
- **팀 정보 수정**: ❌ (팀 페이지에서)

### 팀 페이지
- **Public 접근**: ⭕️ (기본 정보)
- **팀 정보 수정**: ⭕️ (팀 관리자만)
- **협회 관계 수정**: ❌ (Phase 5 이후 협회 승인으로만)

### 축구 허브
- **Public 접근**: ⭕️
- **팀 생성**: ⭕️ (로그인 필요)
- **협회 관계 설정**: ❌ (Phase 5 이후)
