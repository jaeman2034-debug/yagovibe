# 대회 징계 기준 · 시스템 로그 문장 (최종본)

**개발자 로그 / 관리자 액션 로그 / 감사 로그**
**복붙해서 바로 쓰는 문장**
**규정·책자·시스템 문구와 100% 정합**

---

## 1️⃣ 선수 자격 관련 로그

### ▸ 선수 명단 제출 시

```
[선수자격검증] 선수 명단이 제출되었습니다.
본 대회는 대한축구협회 조인 KFA 시스템 등록 회원만 선수 등록이 가능합니다.
```

---

### ▸ 조인 KFA 미등록 의심 시

```
[선수자격경고] 일부 선수의 조인 KFA 등록 여부 확인이 필요합니다.
선수 자격 확인의 1차 책임은 해당 단위 클럽에 있습니다.
```

---

### ▸ 부정선수 확정 시

```
[부정선수확정] 조인 KFA 미등록 또는 허위 등록 선수가 확인되었습니다.
해당 경기는 대회 규정에 따라 몰수패 처리됩니다.
```

---

## 2️⃣ 참가 자격 (회비·참가비) 관련 로그

### ▸ 회비 미납 상태에서 참가 시도

```
[참가불가] 회비 미납 상태에서는 대회 참가 신청이 불가합니다.
대회 참가를 위해서는 참가 신청 이전까지 회비 납부가 완료되어야 합니다.
```

---

### ▸ 참가비 미결제 상태

```
[참가대기] 참가비 결제가 완료되지 않아 참가가 확정되지 않았습니다.
결제 완료 시 참가 상태가 자동으로 확정됩니다.
```

---

## 3️⃣ 대진표·결과 처리 로그

### ▸ 대진표 확정 시

```
[대진표확정] 대진표가 참가 자격 요건을 충족한 선수 기준으로 확정되었습니다.
시스템에 게시된 대진표가 공식 기준입니다.
```

---

### ▸ 몰수패 처리 시

```
[경기결과처리] 본 경기는 대회 징계 기준에 따라 몰수패로 처리되었습니다.
```

---

## 4️⃣ 징계 결정 관련 로그

### ▸ 협회 징계 확정

```
[징계결정] 부정선수 출전 및 규정 위반 사항에 대해
협회 차원의 징계가 협의·결정되었습니다.
```

---

### ▸ 출전 정지 / 차기 대회 제한

```
[출전제한] 규정 위반으로 인해 해당 선수(또는 팀)는
추가 출전이 제한됩니다.
```

---

## 5️⃣ 이의 제기 관련 로그

### ▸ 이의 제기 접수

```
[이의접수] 이의 제기가 접수되었습니다.
규정에 따라 경기 종료 후 3일 이내 접수 건만 검토됩니다.
```

---

### ▸ 이의 기각

```
[이의기각] 접수 기한 경과 또는 규정 미충족으로
이의 제기는 처리되지 않습니다.
```

---

## 6️⃣ 최종 고정 로그 (가장 중요)

이 문장은 모든 분쟁의 종결 로그입니다.

```
[공식기준] 본 대회의 참가 자격, 선수 자격 및 징계 여부는
대회 운영 시스템 및 대회 징계 기준을 공식 기준으로 합니다.
```
