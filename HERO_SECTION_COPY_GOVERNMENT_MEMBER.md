# Hero Section 문구 확정 (구청용 / 회원용)

**목적**: 협회 Hero Section 문구를 구청용과 회원용으로 각각 두 줄로 확정

---

## 1️⃣ Hero Section의 역할 재확인

### 역할
- **행정 선언문** (홍보용 아님)
- "여기가 공식 채널이다"를 3초 안에 증명
- 구청/회원/비회원 모두에게 같은 메시지를 전달

---

## 2️⃣ 문구 최종 확정 (구청용 / 회원용)

### 상단 타이틀 (공통 · 고정)
```
노원구 축구협회
```

---

### 🅰️ 구청 · 체육회 담당자용 (1줄)

```
노원구 내 축구 대회·대관·행정을 공식적으로 운영합니다.
```

**특징:**
- "공식" 명시
- 행정 범위 명확 (대회·대관·행정)
- 보고/열람용으로 바로 사용 가능

---

### 🅱️ 회원 · 현장 사용자용 (1줄)

```
노원구 축구의 모든 일정과 운영 기준이 이곳에서 관리됩니다.
```

**특징:**
- "기준"이라는 단어로 권위 전달
- 회원/비회원 모두 이해 가능
- 행동 유도 ❌, 안내 중심 ⭕

---

## 3️⃣ 적용 규칙

### 기본 적용 (Public)
- **회원/현장용 문구 사용** (🅱️)
- `/association/assoc-nowon-football` 접근 시 기본 표시

### 구청용 적용 (선택적)
- **구청 · 체육회 담당자용 문구** (🅰️)
- 구청 열람 링크 전용
- 별도 URL 파라미터 또는 쿼리스트링으로 전환
- 예: `/association/assoc-nowon-football?view=government`

### Admin 모드 영향
- **Admin 모드 (`?mode=admin`)에서도 회원/현장용 문구 유지** (🅱️)
- 구청용 문구는 구청 전용 링크에서만 사용

---

## 4️⃣ 문구 선택 기준

### 회원/일반용 사용
- ✅ 일반 Public 접근
- ✅ 회원 팀 접근
- ✅ 비회원 팀 접근
- ✅ Admin 모드 (`?mode=admin`)

### 구청용 사용
- ✅ 구청 담당자 열람 링크
- ✅ 체육회 관계자 열람 링크
- ❌ 일반 사용자 접근

---

## 6️⃣ Hero Section에 넣지 않는 것

### 금지 사항
- ❌ 버튼
- ❌ CTA 문구
- ❌ 신청/문의 링크
- ❌ 장황한 설명
- ❌ Quick Links / 스크롤 앵커

### 이유
**Hero는 설명하는 곳이 아니라 '정의하는 곳'이다.**

## 7️⃣ 구현 고려사항

### 문구 전환 방식
