# PHASE 4-1 와이어프레임

**텍스트 기반 와이어프레임 (디자인/색상 논의 ❌, 구조만 ⭕)**

---

## 화면 1️⃣ 정산 보드 메인 화면 (수납 상태 한눈에)

### 구조

```
[HEADER]
노원구 축구협회 · 정산 보드
(관리자 전용)

[SUMMARY STRIP]
- 총 선지급액: ₩________
- 수납완료: ₩________
- 미수: ₩________
- 환불완료: ₩________

[FILTER BAR]
[기간 선택] [대회 선택(옵션)] [상태 필터: 전체/미수/완료/환불]

[TABLE]
┌─────────────────────────────────────────────┐
│ 날짜 │ 구장 │ 팀/대상 │ 금액 │ 상태 │ 액션 │
├─────────────────────────────────────────────┤
│ 8/31 │ 마들 │ A팀     │ 100k │ 미수 │ 상태변경 │
│ 9/01 │ 육사 │ 비회원B │ 120k │ 완료 │ 상태변경 │
│ 9/02 │ 초안 │ C팀     │ 100k │ 환불 │ 상태변경 │
└─────────────────────────────────────────────┘

[FOOTER ACTIONS]
[영수증 업로드]   [보고서 생성]
```

### 규칙
- 상태변경은 미수 ↔ 완료 ↔ 환불만
- 결제/송금 버튼 ❌
- 관리자만 접근 ⭕️

### 기능 상세
- **HEADER**: 페이지 타이틀 + 관리자 전용 표시
- **SUMMARY STRIP**: 요약 정보 (총액, 상태별 금액)
- **FILTER BAR**: 기간/대회/상태 필터
- **TABLE**: 수납 목록 (날짜, 구장, 팀/대상, 금액, 상태, 액션)
- **액션 컬럼**: 상태 변경 버튼 (드롭다운 또는 버튼 그룹)
- **FOOTER ACTIONS**: 영수증 업로드, 보고서 생성 버튼

## 화면 2️⃣ 영수증 업로드 플로우 (모바일 우선)

### 구조

```
[HEADER]
영수증 업로드

[STEP 1]
[사진 촬영/업로드]
- 카메라 아이콘 또는 파일 업로드 영역
- 드래그 앤 드롭 지원

[STEP 2]
자동 분류(선택)
( ) 시설 사용료
( ) 심판 수당
( ) 운영비
- 라디오 버튼 또는 체크박스

[STEP 3]
입력 필드
- 날짜 (자동/수정 가능) [YYYY-MM-DD]
- 금액 [숫자 입력]
- 대회명 (옵션) [텍스트 입력]
- 메모 (옵션) [텍스트 입력]

[STEP 4]
[저장]
- 저장 버튼
- 취소 버튼
```

### 규칙
- OCR 정확도 요구 ❌
- "사진 + 분류 + 날짜"면 성공
- 업로드 즉시 목록에 반영

### 기능 상세
- **STEP 1**: 파일 업로드 (모바일: 카메라, 데스크탑: 파일 선택/드래그 앤 드롭)
- **STEP 2**: 자동 분류 (OCR 또는 사용자 선택)
- **STEP 3**: 필수/선택 필드 입력
- **STEP 4**: 저장 후 영수증 목록으로 이동

### 모바일 최적화
- 단계별 스텝 표시 (진행 표시)
- 큰 터치 영역
- 키보드 최적화 (날짜/금액 입력)

## 화면 3️⃣ 보고서 생성 화면 (기간 선택 → 출력)

### 구조

```
[HEADER]
정산 보고서 생성 (읽기 전용)

[INPUT]
기간 선택: [시작일 YYYY-MM-DD] ~ [종료일 YYYY-MM-DD]
- 달력 선택 또는 날짜 입력
- 기본값: 이번 달 (선택적)

[PREVIEW]
- 수입 합계: ₩________
- 지출 합계: ₩________
- 잔액: ₩________
- 미리보기 영역 (수정 불가, 읽기 전용)

[ACTIONS]
[PDF 다운로드]   [Excel 다운로드]
- 버튼 클릭 시 파일 다운로드 시작
- 생성 완료 메시지 표시
```

### 규칙
- 미리보기 수정 ❌
- 양식 변경 ❌
- 관리자 전용 ⭕️

### 기능 상세
- **INPUT**: 기간 선택 (시작일/종료일)
- **PREVIEW**: 기간 내 수입/지출/잔액 미리보기 (실시간 계산)
- **ACTIONS**: PDF/Excel 생성 및 다운로드

### 데이터 집계 기준
- **수입**: 수납완료 상태의 대관료 합계
- **지출**: 영수증 디지털화된 지출 합계 (시설/심판/운영비)
- **잔액**: 수입 - 지출

## 화면 4️⃣ 권한 / 접근 가드 표기 (관리자 전용)

### 구조

```
[ACCESS GUARD]
이 페이지는 노원구 축구협회 관리자만 접근할 수 있습니다.
- 메시지 표시
- 로그인 유도 버튼 (비로그인 시)
- 뒤로가기 버튼
```

### 규칙
- public/회원 접근 시 즉시 차단
- URL 직접 접근도 차단

### 구현 방식
- `useIsAssociationAdmin` Hook 사용
- 권한 없는 경우 즉시 리다이렉트 또는 접근 차단 메시지 표시
- 모든 Phase 4-1 화면에 적용

## 화면 간 네비게이션

### 네비게이션 흐름

```
정산 보드 메인 화면
  ↓ (영수증 업로드 클릭)
영수증 업로드 화면
  ↓ (저장)
정산 보드 메인 화면 (목록에 반영)

정산 보드 메인 화면
  ↓ (보고서 생성 클릭)
보고서 생성 화면
  ↓ (PDF/Excel 다운로드)
정산 보드 메인 화면
```

### 라우팅 구조
- `/association/:associationId/accounting` - 정산 보드 메인 화면
- `/association/:associationId/accounting/receipts/upload` - 영수증 업로드
- `/association/:associationId/accounting/reports` - 보고서 생성

## ✅ 완료 체크 (개발/검증 기준)

### 기능 체크
1. ✅ 한 화면에서 미수/완료/환불 상태가 보인다
2. ✅ 영수증을 폰으로 찍어 바로 저장한다
3. ✅ 보고서를 버튼 한 번으로 뽑는다
4. ✅ 결제/송금/신청 기능은 없다

### 권한 체크
- ✅ 관리자만 접근 가능
- ✅ public/회원 접근 차단
