# ⚡ 빠른 테스트 체크리스트

> 5분 안에 확인

---

## 1️⃣ 앱 상태 체크 (1분)

- [ ] 앱 실행됨
- [ ] 위치 권한 팝업 표시
- [ ] 마이크 권한 팝업 표시

---

## 2️⃣ 기본 흐름 테스트 (2분)

### 테스트 문장

1. "근처 축구장"
2. "지금 여는 헬스장"
3. "근처 카페"

### 체크

- [ ] 버튼 누르고 말하기
- [ ] 3초 안에 반응
- [ ] 지도에 핀 표시

---

## 3️⃣ 합격 기준

- [ ] 말 끝나고 3초 안에 반응
- [ ] 결과가 지도에 바로 찍힘
- [ ] "어? 된다" 느낌

---

## ✅ 통과하면

→ 외부 테스트 진행 가능

## ❌ 문제 발생 시

### "음성 안 됨"

→ 서버 로그 확인
→ Whisper STT 확인

### "결과 안 나옴"

→ 키워드 매핑 확인
→ Mock 데이터 확인
