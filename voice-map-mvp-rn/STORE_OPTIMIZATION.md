## 📱 앱 이름 / 부제 (권장)

### 앱 이름

```
YAGO
```

### 부제

```

```

- ❌ AI / 스마트 / 추천 같은 단어 쓰지 마라

- ⭕ **명확하고 간단하게**

> 그대로 써도 된다.

```
YAGO는 말로 주변 장소를 빠르게 찾을 수 있는 지도 앱입니다.

"근처 축구장"
"지금 여는 헬스장"
처럼 말하면
지도에서 바로 위치를 확인할 수 있습니다.
```

## 🖼️ 스크린샷 문구 (4장만 있으면 충분)

### ① 첫 화면

**스크린샷**
- 지도 화면
- 하단 음성 버튼

**문구**

```
말로 주변 장소를 찾아보세요
```

---

### ② 음성 입력

**스크린샷**
- 음성 버튼 누르는 화면
- "듣고 있어요..." 표시

**문구**

```
"근처 축구장"
이라고 말해보세요
```

---

### ③ 결과 화면

**스크린샷**
- 지도에 핀 표시
- 결과 카드

**문구**

```
말하자마자
지도에 바로 표시
```

---

### ④ 길찾기

**스크린샷**
- 길찾기 버튼
- 경로 표시

**문구**

```
바로 이동
```
