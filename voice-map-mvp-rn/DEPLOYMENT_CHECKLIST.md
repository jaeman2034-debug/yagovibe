# 1️⃣ 🍎 iOS — TestFlight 체크리스트

## ✅ 1-1. Apple Developer 기본

- [ ] Apple Developer 계정 ($99/년)
- [ ] App ID 생성
- [ ] Bundle ID 고정 (`com.yago.voicemapmvp`)
  👉 **중간에 바꾸면 지옥**

### Bundle ID 설정 (`app.json`)

```json
{
  "expo": {
    "ios": {
      "bundleIdentifier": "com.yago.voicemapmvp"
    }
  }
}
```

---

## ✅ 1-2. iOS 필수 권한 문구 (리젝 1순위)

### `app.json` 확인

```json
{
  "expo": {
    "ios": {
      "infoPlist": {
        "NSMicrophoneUsageDescription": "주변 장소를 음성으로 검색하기 위해 마이크 접근이 필요합니다.",
        "NSLocationWhenInUseUsageDescription": "내 주변 장소를 찾기 위해 위치 정보가 필요합니다."
      }
    }
  }
}
```

### 📌 체크 포인트

- [ ] "왜 필요한지" **명확히** 설명
- [ ] "수집" / "기록" 같은 단어 ❌
- [ ] 사용자 행동 후 권한 요청 (앱 시작 시 자동 팝업 ❌)

---

## ✅ 1-3. TestFlight 빌드

### 방법 1: EAS Build (권장)

```bash
# EAS CLI 설치
npm install -g eas-cli

# 로그인
eas login

# 빌드
eas build --platform ios
```

### 방법 2: Expo Build (레거시)

```bash
npx expo build:ios
```

### 빌드 전 체크

- [ ] `app.json` 설정 확인
- [ ] Bundle ID 확인
- [ ] 권한 문구 확인

---

## ✅ 1-4. App Store Connect 설정

### 필수 항목

- [ ] 앱 이름: "Voice Map" (또는 원하는 이름)
- [ ] 부제: "음성으로 주변 장소 찾기"
- [ ] 설명 (아래 예시)

#### 설명 예시 (리젝 안 당하는 톤)

```
이 앱은 음성을 이용해 주변의 운동장, 헬스장, 카페 등을
빠르게 찾을 수 있는 지도 서비스입니다.

주요 기능:
- 음성으로 장소 검색
- 지도에서 바로 확인
- 내 위치 기반 검색
```

### 📌 주의사항

- ❌ "AI가 알아서" 같은 과장 표현
- ❌ "최고의" 같은 광고 문구
- ⭕ **명확하고 간단한 설명**

---

## ✅ 1-5. TestFlight 심사 통과 체크

- [ ] 앱 실행 시 크래시 ❌
- [ ] 권한 요청이 **행동 후** 발생 (자동 팝업 ❌)
- [ ] 음성 버튼 누르기 전 마이크 접근 ❌
- [ ] 위치 권한 거부해도 앱 정상 작동 ⭕

👉 TestFlight는 **보통 1~2일 내 승인**

# 2️⃣ 🤖 Android — Play Store 체크리스트

## ✅ 2-1. Google Play Developer 계정

- [ ] Google Play Developer 계정 ($25, 1회)
- [ ] 앱 생성

---

## ✅ 2-2. Android 권한 설정

### `app.json` 확인

```json
{
  "expo": {
    "android": {
      "permissions": [
        "RECORD_AUDIO",
        "ACCESS_FINE_LOCATION"
      ]
    }
  }
}
```

### 📌 주의사항

- [ ] 위치 권한 **BACKGROUND ❌**
- [ ] `ACCESS_FINE_LOCATION`만 사용
- [ ] 불필요한 권한 추가 ❌

---

## ✅ 2-3. AAB 빌드 (필수)

### EAS Build (권장)

```bash
eas build --platform android
```

### 빌드 전 체크

- [ ] `app.json` 설정 확인
- [ ] Package name 확인 (`com.yago.voicemapmvp`)
- [ ] 권한 설정 확인

### 📌 중요

- ❌ APK 업로드 불가
- ⭕ **AAB만 허용**

---

## ✅ 2-4. Play Console 필수 서류 (중요)

### 개인정보 처리 질문 (정답)

- 위치 수집? → **YES**
- 음성 수집? → **YES**
- 서버 전송? → **YES**
- 제3자 공유? → **NO**

### 📌 주의사항

- ⭕ **정직하게** 써야 통과됨
- ❌ 거짓 답변 시 리젝

---

## ✅ 2-5. 스토어 설명 (Android)

### 앱 설명 예시

```
말로 주변 장소를 검색하고
지도에서 바로 확인할 수 있는 서비스입니다.

주요 기능:
- 음성으로 장소 검색
- 지도에서 바로 확인
- 내 위치 기반 검색
```

### 📌 주의사항

- ❌ "AI" 과장 표현
- ❌ "최고의" 같은 광고 문구
- ⭕ **명확하고 간단한 설명**
