## 🍎 iOS App Store

### 앱 이름
```
Voice Map
```

### 부제
```
음성으로 주변 장소 찾기
```

### 설명

```
이 앱은 음성을 이용해 주변의 운동장, 헬스장, 카페 등을
빠르게 찾을 수 있는 지도 서비스입니다.

주요 기능:
- 음성으로 장소 검색
- 지도에서 바로 확인
- 내 위치 기반 검색

사용 방법:
1. 음성 버튼을 누르고 말하세요
2. "근처 축구장" 같은 장소를 말하세요
3. 지도에서 바로 확인하세요
```

### 키워드 (선택)
```
음성검색, 지도, 장소찾기, 운동장, 헬스장
```
