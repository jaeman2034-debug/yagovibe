## 3️⃣ 권한 요청 타이밍 리젝

### ❌ 잘못된 예시

```javascript
// 앱 시작 시 바로 권한 요청
useEffect(() => {
  Location.requestForegroundPermissionsAsync();
  Audio.requestPermissionsAsync();
}, []);
```

### ⭕ 올바른 예시

```javascript
// 사용자 행동 후 권한 요청
const startRecording = async () => {
  const { status } = await Audio.requestPermissionsAsync();
  if (status !== "granted") {
    Alert.alert("마이크 권한 필요", "...");
    return;
  }
  // 녹음 시작
};
```

### 📌 체크리스트

- [ ] 앱 시작 시 자동 권한 요청 ❌
- [ ] 사용자 행동 후 권한 요청 ⭕
- [ ] 권한 거부 시 앱 정상 작동 ⭕

## 4️⃣ 크래시 리젝

### ❌ 잘못된 예시

```javascript
// 에러 처리 없음
const text = await sttRes.json();
searchPlace(text);
```

### ⭕ 올바른 예시

```javascript
// 에러 처리 포함
try {
  const sttRes = await fetch(`${API_BASE_URL}/stt`, {...});
  if (!sttRes.ok) {
    throw new Error("STT 처리 실패");
  }
  const { text } = await sttRes.json();
  await searchPlace(text);
} catch (error) {
  Alert.alert("오류", "음성 인식 중 오류가 발생했어요.");
}
```

### 📌 체크리스트

- [ ] 모든 네트워크 요청 try-catch
- [ ] 권한 거부 시 에러 처리
- [ ] 서버 오류 시 에러 처리
- [ ] 실기기에서 크래시 테스트
