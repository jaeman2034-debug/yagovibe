# 실무자(사무국) 전용 안심 멘트 & 사용 시나리오

## 🎯 사용 가이드

**사무국/총무 등 실제 운영 담당자를 위한 안심 메시지**

**원칙:** 기술 설명 ❌ / 업무 편의 ⭕ / 책임 경감 ⭕

---

## 💬 안심 멘트 세트

### 멘트 1: "전화보다 쉽습니다"

```
전화 받는 것보다 쉽습니다.

신청 목록 보고

승인 / 반려 버튼만 누르시면 됩니다.
```

**설명:**
- 복잡한 메뉴나 설정 없음
- 버튼 2개만 누르면 됨
- 전화 받고 확인하고 엑셀에 입력하는 것보다 단순

---

### 멘트 2: "작업량이 줄어듭니다"

```
지금은 전화 받고, 카톡 확인하고, 엑셀 입력하고...

이 모든 작업을 

'승인 버튼 하나'로 대체합니다.
```

**설명:**
- 전화 문의 80% 감소
- 엑셀 관리 불필요
- 승인 기록 자동 저장

---

### 멘트 3: "책임이 명확해집니다"

```
지금은 개인 판단으로 처리하시는데

시스템에서는

모든 승인 기록이 남습니다.

누가, 언제, 어떤 이유로 승인했는지

전부 기록되니까

나중에 문제 생겨도 추적 가능합니다.
```

**설명:**
- 책임 소재 명확
- 감사 대응 용이
- 개인 판단의 부담 감소

---

### 멘트 4: "민원이 줄어듭니다"

```
지금 민원이 생기는 이유는

'왜 안 되냐'는 질문 때문인데

시스템에서는 기준이 명확하니까

'왜 안 되냐'고 물어볼 이유가 없어요.

화면에 기준이 그대로 나와 있거든요.
```

**설명:**
- 기준이 명확해서 설명 불필요
- "왜 안 되냐"는 질문 자체가 줄어듦
- 전화 설명하는 시간 감소

---

## 📋 실제 사용 시나리오

### 시나리오 1: 전화 문의 감소

**실무자 걱정:** "전화 많이 오면 어쩌죠?"

**안심 멘트:**
```
오히려 전화 문의가 줄어듭니다.

지금 전화 문의가 생기는 이유는

'우리 팀은 회원인가요?', '예약 어떻게 하나요?' 같은

확인 질문 때문인데

시스템에서는 팀이 직접 확인할 수 있어요.

자신의 팀 상태, 예약 가능 여부를

화면에서 바로 볼 수 있거든요.

그래서 전화 문의가 80% 이상 줄어들 겁니다.
```

---

### 시나리오 2: 엑셀 관리 불필요

**실무자 걱정:** "엑셀로 관리하는 게 더 편한데요?"

**안심 멘트:**
```
엑셀 관리의 단점이 뭔가요?

데이터 입력하고, 정렬하고, 백업하고...

이 모든 작업을 시스템이 알아서 합니다.

승인만 하시면

모든 기록이 자동으로 저장되고,

검색도 되고, 리포트도 자동으로 만들어집니다.

엑셀보다 훨씬 편해요.
```

---

### 시나리오 3: 책임 부담 경감

**실무자 걱정:** "승인 잘못하면 제 책임 아닌가요?"

**안심 멘트:**
```
오히려 책임이 명확해집니다.

지금은 개인 판단으로 처리하시는데

시스템에서는 모든 승인 기록이 남습니다.

누가, 언제, 어떤 이유로 승인했는지

전부 기록되니까

나중에 문제 생겨도

'개인 판단이 아니라 협회 결정'으로

명확하게 보호받을 수 있어요.
```

---

### 시나리오 4: 컴퓨터 사용 걱정

**실무자 걱정:** "컴퓨터 잘 못하는데요?"

**안심 멘트:**
```
전화 받는 것보다 쉽습니다.

복잡한 기능이 아니라

신청 목록 보고

승인 또는 반려 버튼만 누르시면 됩니다.

2개 버튼만 있으니까

어렵지 않으실 거예요.

처음에 10분 정도 설명해 드리면

바로 사용 가능하실 겁니다.
```

---

## 🎯 핵심 안심 메시지 (4가지)

### 1. "작업량이 줄어듭니다"
- 전화 문의 80% 감소
- 엑셀 관리 불필요
- 승인 버튼 하나로 대체

### 2. "책임이 명확해집니다"
- 모든 기록 자동 저장
- 개인 판단 → 협회 결정
- 감사 대응 용이

### 3. "민원이 줄어듭니다"
- 기준이 명확해서 설명 불필요
- "왜 안 되냐"는 질문 감소
- 전화 설명 시간 감소

### 4. "사용이 쉽습니다"
- 전화 받는 것보다 쉬움
- 버튼 2개만 누르면 됨
- 10분 설명으로 충분
