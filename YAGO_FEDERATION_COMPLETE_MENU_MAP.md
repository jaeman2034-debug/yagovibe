## 3️⃣ 화면별 상세 구조

### 3-1. 홈 탭

```
협회 헤더
  - 협회명
  - 지역/종목
  - 관리자 버튼 (관리자만)

메인 비주얼
  - 협회 슬로건
  - CTA 버튼 (대회 보기, 경기 일정, 참가 안내)

진행중 대회
  - 대회 카드 (대회명, 기간, 참가팀 수, 상태)
  - [대회 보기] 버튼

공지사항
  - 중요 공지 (상단 고정)
  - 최근 공지 (5개)

다가오는 경기
  - 오늘 경기
  - 이번주 경기
  - 경기 카드 (팀, 시간, 경기장)

최근 경기 결과
  - 최근 5경기
  - 결과 카드 (스코어, 날짜, 대회명)

순위 요약
  - 현재 순위 TOP 5
  - [전체 순위 보기] 버튼

대표 참가팀/클럽
  - 클럽 카드 (로고, 이름, 지역)
  - [전체 팀 보기] 버튼

유소년 섹션
  - 유소년 프로그램 소개
  - 아카데미 목록

후원사 배너
  - 공식 후원사 로고
  - 협력 업체

AI 챗봇 (우하단)
  - "노원구 축구협회 AI 안내"
  - 빠른 질문 메뉴
```

## 4️⃣ AI 에이전트 배치

```
노원구 축구협회 AI 안내

빠른 질문 메뉴:
- 대회 안내
- 일정/결과
- 참가 신청
- 규정/서류
- 협회 문의
- 후원 문의

기능:
- 홈페이지 전체 검색
- 대회/공지/일정/규정 빠른 응답
- 해당 페이지로 즉시 이동
- 민원/문의 접수
```

```
/federations/nowon-football/admin/ai

AI 대시보드:

- 결과 미입력 경기 요약
- 공지 초안 생성
- 대진표 생성 도우미
- 문의 자동 분류
- 규정 위반 의심 체크

에이전트별 패널:
- 대회 안내 에이전트
- 경기 운영 에이전트
- 팀/선수 등록 에이전트
```
