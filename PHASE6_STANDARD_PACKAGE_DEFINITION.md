# Phase 6: 표준 패키지 정의

**목표: "이 시스템을 처음 보는 협회도 사무국 설명 없이 바로 쓰게 만들기"**

---

## STEP 1️⃣ 표준 패키지 정의

### 🔹 협회 기본 패키지 v1

아래를 고정 구성으로 묶습니다.

#### 포함 기능 (고정)

1. **공지 관리**
   - 작성/수정/예약 게시
   - Public 자동 반영

2. **대회 / 일정**
   - 대회 입력
   - Public 일정 자동 노출
   - TournamentSection 자동 연동

3. **대관 상태**
   - available / blocked / event 토글
   - Public 읽기 전용

4. **정산**
   - 수입/지출 입력
   - 영수증 업로드
   - 자동 계산
   - 보고서 출력

5. **회비**
   - 회비 설정
   - 회원별 납부 상태 자동 분류
   - 자동 알림

6. **기부/후원**
   - 기부/후원 등록
   - 영수증 발급
   - 대회 연결

7. **보고서 출력**
   - 대회별 정산 보고서
   - 연간 결산서
   - 구청 제출용 포맷

#### 제외 (의도적)

- ❌ 커스터마이징
- ❌ 예외 로직
- ❌ 협회별 특수 규칙

**👉 "있는 그대로 쓰는 게 표준"**

---

## STEP 2️⃣ 도입 기준 문장 만들기 (중요)

이 문장 하나로 영업·설명 끝납니다.

### 핵심 문장

**"사무국 엑셀·전화·단톡을 없애는 시스템입니다."**

### 금지 사항

- ❌ 부연 설명
- ❌ 기능 나열
- ❌ 기술 설명

---

## STEP 3️⃣ 도입 체크리스트 1장

협회에 던질 질문은 이것뿐입니다.

### 도입 체크리스트

1. **사무국이 엑셀 씁니까?**
   - [ ] YES
   - [ ] NO

2. **회비 미납 전화합니까?**
   - [ ] YES
   - [ ] NO

3. **공지 때문에 단톡 씁니까?**
   - [ ] YES
   - [ ] NO

4. **정산 때 영수증 모읍니까?**
   - [ ] YES
   - [ ] NO

**👉 3개 이상 YES면 도입 대상**

---

## STEP 4️⃣ 가격 전략 (아주 단순하게)

### 권장 구조

**협회 단위 월 구독**

- 회원 수 무관
- 기능 차등 ❌

### 예시

- **월 정액**: 월 50,000원
- **연 결제 할인**: 연 500,000원 (월 41,667원, 17% 할인)

### 논리

**"사무국 인건비 1/10"**

- 사무국 인건비: 월 500,000원 가정
- 시스템 비용: 월 50,000원
- 절감 효과: 월 450,000원

---

## STEP 5️⃣ 파일 하나 만들기 (필수)

### 〈사무국 인수인계 문서〉 (A4 1장)

**내용:**

1. **사무국이 하는 일 4가지**
   - 승인
   - 예외 처리
   - 정책 결정
   - 긴급 대응

2. **하지 않는 일 목록**
   - 입력 ❌
   - 계산 ❌
   - 전달 ❌
   - 독촉 ❌
   - 보고서 작성 ❌

3. **문제 생기면 어디서 확인하는지**
   - 시스템 오류: 기술 지원팀
   - 정책 문의: 협회장
   - 예외 처리: 사무국장

**이 문서 =**
- ✅ 내부 안정성
- ✅ 외부 배포 가능성
