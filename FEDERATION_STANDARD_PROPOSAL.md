# 협회 운영 표준(안)

**상급단체 제출용 표준 선언서**

---

## (A) 표준 선언

### 협회 운영 표준(안)

- 공지·일정·회비·정산은 시스템 기준
- 단톡/엑셀은 공식 효력 없음
- 협회별 브랜드·운영은 자율

---

## (B) 운영 효과

**숫자 말고 현상만:**

- 전화 0건
- 엑셀 0개
- 문의 "이미 봤어요"로 종료

---

## (C) 화면 3컷

- 공지
- 정산 요약
- 회비 상태

**캡션 1줄:**
```
"회원이 보는 실제 화면"
```

---

## 상급단체용 결정 문장

```
"각 협회가 선택해 쓰는 시스템이 아니라,
연합회 운영 표준으로 권장한다."
```

**※ '의무'가 아니라 '권장'으로 시작**
**→ 반발 0, 채택률 최대**
