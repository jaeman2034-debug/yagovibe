# 🚀 모임온 (MOIMON) MVP 런칭 & 3개월 성장 전략

## 📱 MVP 화면 구조 (최종 확정)

### 하단 탭 5개 (절대 변경 불가)

```
[홈] [채팅] [일정] [랭킹] [프로필]
```

**이유:**
- 확장성 최적
- 직관성 최고
- 유지보수 용이

---

## 1️⃣ 홈 화면 (앱의 심장)

### 목적
앱을 열었을 때 **"활동 중"** 느낌을 줘야 한다.

### 상단 영역 (공통)

```
🔥 오늘 우리 모임

- 오늘 출석 12명
- 새 메시지 84개
- 예정 일정 1개
- 현재 1위 홍길동 🔥
```

### 타입별 강조 영역

#### 🎓 동아리 (club)
- 이번 달 출석률 %
- MVP 후보 3명
- 최근 공지

#### 📚 스터디 (study)
- 오늘 학습 목표
- 다음 일정 D-1
- 주간 목표 진행률

#### 🎨 취미 (hobby)
- 이번 주 이벤트
- 인기 사진 TOP 3
- 참여 멤버 수

**핵심:** 홈은 정보 요약 + 동기부여 화면

---

## 2️⃣ 채팅 화면 (활동 중심)

### 상단
- 팀 이름
- 멤버 수
- 📌 고정 공지 (club만)

### 메시지 타입
- 텍스트

- 출석 카드

**핵심:** 메시지는 단순 대화가 아니라 **활동 허브**

## 3️⃣ 일정 탭

## 📈 3개월 런칭 & 성장 전략

### 🎯 타겟 전략

#### 1단계: 대학 동아리 공략 (1개월)

**타겟:**
- 대학 동아리 리더 (회장, 부회장)
- 신입생 모집 시즌 (3월, 9월)

**공략 방법:**
1. **캠퍼스 직접 방문**
   - 동아리 연합회 방문
   - 리더 미팅 주선
   - 데모 시연

2. **온라인 공략**
   - 대학 커뮤니티 (에브리타임, 대학생 SNS)
   - "동아리 출석 관리 앱" 키워드
   - 리더 후기 영상

3. **바이럴 설계**
   - 첫 팀 생성 시 리워드 (스타벅스 쿠폰)
   - 추천인 코드 시스템
   - 팀 초대 시 양쪽 모두 리워드

**목표:**
- 1개월 내 50개 동아리 가입
- 월간 활성 사용자 500명

---

#### 2단계: 스터디 모임 확장 (2개월)

**타겟:**
- 스터디 카페 운영자
- 온라인 스터디 플랫폼 (스터디큐브, 스터디그룹)
- 대학 도서관 스터디룸 이용자

**공략 방법:**
1. **스터디 카페 제휴**
   - 스터디 카페에 포스터 부착
   - 카페 이용자에게 앱 소개
   - 카페 운영자에게 수수료 제공

2. **온라인 스터디 플랫폼 연동**
   - 스터디큐브, 스터디그룹과 제휴
   - "출석 체크 앱"으로 포지셔닝
   - 스터디 리더에게 직접 연락

3. **대학 도서관 공략**
   - 도서관 스터디룸 예약 시스템과 연동
   - 도서관 내 포스터 부착
   - 도서관 사서에게 앱 소개

**목표:**
- 2개월 내 100개 스터디 그룹 가입
- 월간 활성 사용자 1,500명

---

#### 3단계: 취미모임 자연 유입 (3개월)

**타겟:**
- 취미모임 카페 운영자
- 온라인 취미 커뮤니티 (네이버 카페, 디스코드)
- 지역 모임 (동호회)

**공략 방법:**
1. **취미모임 카페 제휴**
   - 취미모임 카페에 포스터 부착
   - 카페 운영자에게 수수료 제공
   - 카페 이용자에게 앱 소개

2. **온라인 커뮤니티 공략**
   - 네이버 카페에 앱 소개 글 작성
   - 디스코드 서버에 봇 연동
   - 취미 관련 인플루언서 협업

3. **지역 모임 공략**
   - 지역 동호회 연합회 방문
   - 동호회 리더 미팅 주선
   - 지역 축제/행사 부스 운영

**목표:**
- 3개월 내 200개 취미모임 가입
- 월간 활성 사용자 3,000명

## 🔥 바이럴 설계

### 추천인 시스템

```
A가 B를 초대
→ A: 500원 상당 리워드
→ B: 500원 상당 리워드
→ 팀 생성 시 추가 1,000원 리워드
```

### 팀 초대 보너스

```
팀 멤버 5명 이상 가입
→ 팀 리더: 2,000원 상당 리워드
→ 팀 멤버 각자: 500원 상당 리워드
```

### 이벤트 공유 보너스

```
이벤트를 SNS에 공유
→ 공유자: 200원 상당 리워드
→ 공유를 통해 가입한 사용자: 500원 상당 리워드
```
