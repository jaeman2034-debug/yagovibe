# 🔧 핫 리로드 문제 해결 (화면 안 바뀔 때)

## ✅ 체크리스트 (순서대로)

### 1. Expo 개발 서버 실행 확인

터미널에서:

```bash
cd mobileApp
npm start
```

**확인:**
- [ ] 서버가 실행 중
- [ ] QR 코드 또는 연결 URL 표시
- [ ] 에러 메시지 없음

---

### 2. 폰과 PC 같은 네트워크 확인

**PC:**
```bash
ipconfig
# 또는 (Mac)
ifconfig
```

**폰:**
- 설정 → Wi-Fi → 연결된 네트워크 확인
- PC와 같은 Wi-Fi인지 확인

**확인:**
- [ ] 같은 Wi-Fi 네트워크
- [ ] 같은 IP 대역 (예: 192.168.0.x)

---

### 3. Expo Go 앱 연결 확인

**Expo Go 앱에서:**
- [ ] 스캔으로 연결했는지
- [ ] "Connected" 상태인지
- [ ] 에러 메시지 없는지

---

### 4. 파일 저장 확인

**확인:**
- [ ] `index.tsx` 파일이 실제로 저장됨 (Ctrl+S)
- [ ] 파일 경로가 정확함 (`mobileApp/app/(tabs)/index.tsx`)
- [ ] 에디터에 저장 표시 (점이 사라짐)

---

### 5. 수동 새로고침

**Expo Go 앱에서:**
- 흔들기 (Shake) → "Reload" 선택
- 또는 `r` 키 누르기 (개발 서버 터미널에서)

---

### 6. Expo 서버 재시작

**터미널에서:**
```bash
# Ctrl+C로 서버 중지
# 다시 시작
npm start
```

**확인:**
- [ ] 서버 재시작 완료
- [ ] QR 코드 새로 생성됨

---

## ❌ 그래도 안 되면

### 다른 방법으로 확인

**임시로 더 눈에 띄는 변경:**

```tsx
<Text style={{ padding: 20, fontSize: 24, fontWeight: 'bold', color: 'red' }}>
  🔥 테스트 🔥
</Text>
```

```tsx
<View style={{ flex: 1, backgroundColor: 'yellow' }}>
```
