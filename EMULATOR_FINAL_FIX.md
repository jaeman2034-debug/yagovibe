# 🔧 에뮬레이터 최종 해결 방법

## 🔍 발견된 문제

로그 파일에서 확인된 문제:
- ❌ **포트 8080 충돌**: Firestore 에뮬레이터 시작 실패
- ❌ **포트 9199 충돌**: Storage 에뮬레이터 시작 실패
- ⚠️ **포트 5000 충돌**: Hosting 에뮬레이터가 5002로 변경됨

## ✅ 해결 방법

### 1단계: 모든 프로세스 종료

```powershell
taskkill /PID 4888 /F /T
taskkill /PID 6944 /F /T
taskkill /PID 12228 /F /T
```

### 2단계: 포트 해제 확인

```powershell
Start-Sleep -Seconds 3
netstat -ano | findstr "8080\|9199\|9099\|5003"
```

모든 포트가 해제되었는지 확인 (아무것도 나오지 않아야 함)

### 3단계: 에뮬레이터 재시작

```bash
# 프로젝트 루트에서
firebase emulators:start
```
