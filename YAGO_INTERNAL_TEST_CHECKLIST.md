## 9️⃣ Firestore 데이터 확인

### 테스트 시나리오
1. Firebase Console 접속
2. Firestore Database → Data 탭
3. 각 컬렉션 확인

### 확인 컬렉션

#### users
- [ ] 테스트 사용자 문서 생성 확인
- [ ] 필드: `uid`, `displayName`, `email`, `createdAt` 등

#### teams
- [ ] 생성한 팀 문서 확인
- [ ] 필드: `name`, `sport`, `region`, `ownerUid` 등

#### activities
- [ ] Activity 문서 생성 확인
- [ ] 필드: `type`, `refType`, `sport`, `authorId`, `createdAt` 등
- [ ] `visibility: "public"` 확인

#### marketPosts
- [ ] 거래 글 문서 생성 확인
- [ ] 필드: `title`, `price`, `authorId`, `sport`, `status` 등

#### recruitPosts
- [ ] 모집 글 문서 생성 확인
- [ ] 필드: `teamId`, `position`, `slots`, `authorId` 등

### 예상 결과
- 각 컬렉션에 테스트로 생성한 문서들이 정상적으로 저장됨
- 필수 필드가 모두 포함되어 있음
